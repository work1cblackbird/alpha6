План видов расчета "Виды периодов аренды"

Назначение


Предназначен для хранения списка периодов аренды. Содержит два предопределенных элемента «Использование» и «Отмена». Первое используется для расчета периодов аренды, второе — используется в изменениях условий договора для вытеснения интервалов использования автомобилей.


Относится к подсистеме «Аренда автомобилей»


Форма вида расчета содержит признак «Аренда», который разделяет интервалы использования автомобиля от других вспомогательных видов периода. Также форма содержит список видов интервалов, которые могут вытеснять текущий вид периода при расчете аренды.
