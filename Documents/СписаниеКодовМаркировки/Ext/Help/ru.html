Документ "Списание кодов маркировки"

Назначение

Документ «Списание кодов маркировки» предназначен для формирования факта списания кодов маркировки и отправки документа в государственную информационную систему мониторинга за оборотом товаров (ГИС МТ). Данный документ заполняется кодами маркировки товаров, которые были испорчены/утеряны/уничтожены, для вывода их с оборота. Если код маркировки был введен в оборот, то не обходимо сделать перемаркировку данного товара соответствующим документом.

ВАЖНО! Перед использованием документа необходимо убедится в наличии личного кабинета в ГИС МТ для данной организации. Необходимо указать в настройках способ обмена с ГИС МТ через файловый обмен или API ГИС МТ.

Относится к подсистеме Маркировка товаров раздела Снабжение.

Текущее состояние кодов маркировки определяется хозяйственной операцией документа:

  • Списание КМ, нанесенных на товар. Отражает списание кодов маркировки товаров, которые введены в оборот и привязан к товару.
  • Списание КМ, не нанесенных на товар. Используется в том случае, когда были новые коды маркировки получены по заказу кодов маркировки, но не привязаны к товару (не введены в оборот, но эмитированы).

Реквизиты

  • Статус. Реквизит является обязательным для заполнения. Ссылается на перечисление «Статусы документов маркировки». Отображает текущее состояние документа.
  • Товарная группа. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы маркировки». Определение товарной группы маркируемого товара.
  • Идентификатор документа. Тип значения: «Строка». Заполняется автоматически при отправке документа в ГИС МТ при использовании обмена через API.
  • Причина. Реквизит является обязательным для заполнения. Ссылается на перечисление «Причины списания кодов маркировки». Причина списания кодов маркировки.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ. На основании ИНН организации формируется запрос в ГИС МТ.  
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ. 
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Адрес места выбытия. Тип значения: «Строка». Адрес места выбытия товаров с кодами маркировки. Указывается только для товарных групп «Табачная продукция».
  • Вид первичного документа. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды первичных документов». Вид документа, на основании которого производится списание. Указывается только для товарных групп «Табачная продукция».
  • Наименование первичного документа. Реквизит является обязательным для заполнения при указании вида первичного документа «Прочее». Тип значения: «Строка». Наименование документа, на основании которого производится списание. Указывается только для товарных групп «Табачная продукция».
  • Дата первичного документа. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата документа, на основании которого производится списание. Указывается только для товарных групп «Табачная продукция».
  • Номер первичного документа. Реквизит является обязательным для заполнения. Тип значения: «Строка». Номер документа, на основании которого производится списание. Указывается только для товарных групп «Табачная продукция».

Табличная часть «Товары»

  • Номенклатура. Реквизит обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Характеристика. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п. 
  • Код маркировки. Реквизит является уникальным и обязательным для заполнения. Тип значения: «Строка». Код маркировки товара, которому изменяется состояние.
  • Причина. Ссылается на перечисление «Причины списания кодов маркировки». Причина списания кода маркировки.

Кнопка Выгрузить используется для формирования документа «Списание КМ, нанесенных на товар» или «Списание КМ, не нанесенных на товар» и его отправки (при настройке обмена через API) в ГИС МТ. Для отображения факта отправки документа в учетную систему необходимо перевести документ в состояние «Отправлен». После успешной загрузки документа необходимо установить статус «Выполнен» и для перевода указанных кодов маркировки в новый статус.
