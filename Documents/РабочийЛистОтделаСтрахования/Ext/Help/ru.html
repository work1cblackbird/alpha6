Документ «Рабочий лист отдела страхования»

Назначение

Используется для фиксирования параметров потенциальной сделки по оказанию клиенту услуг страхования. Страховые услуги могут быть связаны со сделкой по продаже автомобиля, а так же могут быть независимы от нее.

Параметры

Длина номера: 10.

Периодичность номера: Непериодический.

Тип номера: Строка.

Вводится на основании: «Рабочий лист».

Объекты, вводимые на основании: «Страховой полис», «Событие».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник "Пользователи". Автор документа - пользователь, породивший документ.
  • Валюта документа. Ссылается на справочник "Валюты". Валюта документа.
  • Вид оплаты. Реквизит является обязательным для заполнения. Ссылается на перечисление "Виды оплаты". Вид оплаты.
  • Документ основание. Документ, на основании которого был введен рабочий лист отдела страхования.
  • Комментарий. Тип значения: Строка. Произвольный комментарий к документу.
  • Контактный телефон. Реквизит является обязательным для заполнения. Тип значения: Строка. Контактный телефон страховщика.
  • Курс валюты. Тип значения: Число. Курс валюты управленческого учета документа.
  • Курс документа. Тип значения: Число. Курс валюты документа.
  • Менеджер. Реквизит является обязательным для заполнения. Ссылается на справочник "Сотрудники". Менеджер отдела страхования.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник "Организации". Идентификатор юридического лица, от которого выписан документ.
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник "Подразделения компании". Подразделение компании, в котором был выписан документ.
  • Причина отказа. Ссылается на справочник "Возражения и причины отказов". Причина отказа.
  • Статус. Реквизит является обязательным для заполнения. Ссылается на справочник "Статусы рабочего листа". Статус рабочего листа отдела страхования.
  • Страхователь. Ссылается на справочник "Контрагенты". Страхователь.
  • Тип цен. Ссылается на справочник "Типы цен". Тип цен документа.
  • Операция. Ссылается на справочник "Хоз. операции". Выполняемая документом операция.

    Табличная часть «Страхование»

  • Объект страхования. Ссылается на справочник "Автомобили". Объект страхования. 
  • Вид страхования. Ссылается на перечисление "Виды страхования". Вид страхования.
  • Программа страхования. Ссылается на справочник "Программы страхования". Программа страхования.
  • Страховщик. Ссылается на справочник "Контрагенты". Страховая компания.
  • Страховая сумма. Тип значения: "Число". Сумма страховки.
  • Тариф. Тип значения: "Число". Тариф страхования. 
  • Сумма брутто. Тип значения: "Число". Сумма, которую страхователь оплачивает агенту, без учета скидки. 
  • Сумма нетто. Тип значения: "Число". Сумма, которую агент оплачивает страховщику.
  • Процент скидки. Тип значения: "Число". Процент скидки. 
  • Сумма скидки. Тип значения: "Число". Сумма скидки. 
  • Процент комиссии. Тип значения: "Число". Процент комиссионного вознаграждения. 
  • Сумма комиссии. Тип значения: "Число". Сумма комиссионного вознаграждения. 
  • Сумма премии. Тип значения: "Число".  Сумма, которую страхователь оплачивает агенту, с учетом скидки. 
  • Пролонгация. Тип значения: "Булево". Признак продления страховки. 
  • Примечание. Тип значения: "Строка". Примечание.  

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Взаимоотношения с клиентами --> Рабочий лист отдела страхования".
