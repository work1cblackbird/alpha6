Документ "Инвентаризация товаров"

Назначение

Документ Инвентаризация товаров предназначен для отражения в учете факта проведения инвентаризации товаров по выбранному складу или у комиссионера. Относится к подсистеме Инвентаризация товаров раздела Склад.

При проведении документ формирует хозяйственные операции:

  • Инвентаризация товаров. Хозяйственная операция в зависимости от реальных остатков приходует или списывает материальные активы, хранимые на оптовых складах компании, в виде ТМЦ  с собственным признаком партии, на регистры Остатки товаров компании и Партии товаров компании, отражает факт продажи товаров комитента, если они в процессе инвентаризации были утеряны или испорчены, на регистре Реализованные товары. А так же регистрирует розничные цены, которые были изменены в процессе инвентаризации, на регистре сведений Цены.
  • Инвентаризация товаров, отданных на комиссию. Хозяйственная операция аналогична операции "Инвентаризации товаров" за тем исключением, что приходуются или списывается товары, отданные комиссионеру на регистре  Партии товаров отданные, а также начисляются взаиморасчеты на сумму недостачи или излишка на регистре Взаиморасчеты компании.

В табличной части документа не могут фигурировать услуги, работы и т.п.

При проведении документа в случае, если фактическое количество товара превышает учетное, на склад будет оприходован излишек (разница между фактическим и учетным количеством), при этом будет образована партия с признаком «собственный товар». Если же фактическое количество товарно-материальных ценностей меньше учетного, при проведении документа со склада по выбранной статье расхода будет списана недостача. Если при проведении инвентаризации не зафиксировано отклонений фактического количества товара от учетного в какую-либо сторону, то при проведении никаких движений по учетным блокам сформировано не будет.

Документ Инвентаризация товаров не применяется для инвентаризации ордерных складов. Для инвентаризации ордерных складов используется документ Инвентаризация ордерного склада.

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения при инвентаризации товаров у комиссионера. Ссылается на справочник Контрагенты и контакты. Комиссионер, у которого производится инвентаризация.
  • Договор. Реквизит является обязательным для заполнения при инвентаризации товаров у комиссионера. Ссылается на справочник Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Склад компании. Реквизит является обязательным для заполнения при инвентаризации собственных товаров. Ссылается на справочник Склады компании. Склад компании, на котором проводится инвентаризация. Если вид документа Инвентаризация товаров отданных на комиссию, то поле Склад компании отсутствует.

Табличная часть «Товары»

На вкладке Товары содержится таблица инвентаризируемых товаров.

Добавочные табличные графы:

  • Кол-во книжное, Кол-во факт. Расчетное (книжное) количество товара, и фактическое количество.
  • Количество. Количество расчетное минус количество фактическое.
  • Цена. Цена закупки на момент инвентаризации.
  • Сумма книжная, Сумма факт. Сумма расчетная и сумма фактическая. Рассчитываются от цены закупки.
  • Сумма. Сумма фактическая минус сумма расчетная.
  • Цена розничная Розничная цена номенклатурной позиции.
  • Сумма книжная(розничная), Сумма факт.(розничная). Сумма расчетная розничная и сумма фактическая розничная.
  • Сумма розничная Сумма фактическая розничная минус сумма расчетная розничная. По этой сумме будут производиться движения.
  • ГТД излишков. Номер грузовой таможенной декларации (для товаров, поступивших по импорту).
  • Партия. Партия, с которой будет списан товар. Появляется при установленном праве Выборочное списание партий.
  • № листа. Номер листа в инвентаризационной ведомости.
  • Ячейка. Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов.

Меню кнопки Заполнение содержит пункты меню:

  • Пересчитать книжное количество и сумму. Производит пересчет показателей Книжное количество и Себестоимость по данным учета.
  • Заполнить по договору комиссии. Формирует таблицу при помощи всех товаров, переданных на комиссию, за которые комиссионер еще не отчитался.
  • Заполнить складскими остатками. Заполняет таблицу товарными остатками указанного склада.
  • Заполнить складскими остатками номенклатурой с ячейкой по умолчанию. Заполняет таблицу товарными остатками указанного склада с выбранной ячейкой по умолчанию.
  • Заполнить по группе номенклатуры. В номенклатурном справочнике выбирается группа элементов. Таблица заполняется всеми элементами выбранной группы.
  • Заполнить складскими остатками по группе. Будет открыто окно, в котором можно будет выбрать группу элементов справочника Номенклатура. В табличную часть будут выведены все имеющиеся в наличии номенклатурные позиции, относящиеся к выбранной группе.
  • Загрузить из ТСД. Таблица заполняется товарами, информация о которых находится в терминале сбора данных, подключенном к Типовому решению.
  • Заполнить всей номенклатурой. Таблица заполняется всем содержимым справочника Номенклатура.
  • Свернуть одинаковые строки. Несколько табличных строк, соответствующих одному и тому же номенклатурному элементу, сворачиваются в одну строку.
  • Обнулить. фактическое количество Обнуляет графу Кол-во факт.
  • Заполнить фактическое кол-во расчетным. Числа в графе Кол-во факт. становятся равными числам в графе Кол-во книжное.

 

На вкладке Параметры располагаются четыре поля. Два из этих полей ссылаются на справочник Статьи доходов и расходов: Статья списания обнаруженной недостачи и Статья оприходования обнаруженных излишков. Движения по доходам и расходам будут произведены в разрезе этих статей. Значения этих реквизитов не используются, если выбрана хозяйственная операция Инвентаризация товаров, отданных на комиссию. Поле Основание для проведения инвентаризации текстовое. Режим корректировки количества перед проведением определяет возможность пользователя исправить книжное количество товара в документе, что определяется правом Корректировать книжное количество в инвентаризации.

Ввод на основании

Документ Инвентаризация товаров вводится на основании документа Инвентаризация ордерного склада.

На основании документа Инвентаризация товаров могут быть введены документы:

  • Приходный кассовый ордер,
  • Инвентаризация ордерного склада,
  • Корректировка движений,
  • Переоценка,
  • Возврат от покупателя.

Печатные формы

  • Инвентаризация товаров на комиссии. Простая печатная форма, содержит список инвентаризируемых товаров.
  • Инвентаризационная опись (ИНВ-3). Унифицированная печатная форма №ИНВ-3, утвержденная постановлением Госкомстата России.
  • Сличительная ведомость (ИНВ-19). Унифицированная печатная форма №ИНВ-19, утвержденная постановлением Госкомстата России.
  • Пустографка. Простая печатная форма, содержит список инвентаризируемых товаров без их фактического количества.
  • Итоги по листам. Печатная форма, содержит список инвентаризируемых товаров с указанием сумм закупки и сумм продажи.
