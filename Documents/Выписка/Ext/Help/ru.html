Документ "Банковская выписка"

Назначение

Документ Банковская выписка предназначен для отражения в учете движения безналичных денежных средств по расчетным счетам, вводится по реальной банковской выписке. Суммы по строкам выписки будут отнесены на взаиморасчеты с контрагентами. Документ относится к подсистеме Банк раздела Финансы.

При проведении документ формирует хозяйственные операции:

  • Строка банковской выписки . Хозяйственная операция в зависимости от типа статьи (приход или расход) начисляет дебиторскую задолженность контрагента перед компанией или списывает кредиторскую задолженность компании перед контрагентом соответственно, на регистр  Взаиморасчеты компании, приходует или списывает денежный актив предприятия, хранимый на расчетных счетах, с учетом типа статьи на регистр Денежные средства компании, отражает суммовую разницу, если таковая была на регистре Доходы и расходы компании.
  • Поступление в операционную кассу. Отражает факт поступления денег безналичным путем в операционную кассу банка.
  • Расход из операционной кассы. Операция, обратная операции Поступление в операционную кассу.
  • Банковская выписка. Эта хозяйственная операция аналогична хозяйственной операции Строка банковской выписки. Создание документов с операцией Банковская выписка заблокировано начиная с релиза 6.0.28.12. Это связано с проблемами перегрузки документов с данной хозяйственной операцией в программный продукт "1C:Бухгалтерия 8".

Реквизиты

  • Счет. Реквизит является обязательным для заполнения. Ссылается на справочник Банковские счета. Банковский счет организации, по которому прошла выписка.

  • Платежное поручение. Ссылка на документ Платежное поручение. Платежное поручение-основание.

Табличная часть «Платеж»

  • Статья ДДС. Реквизит является обязательным для заполнения. Ссылается на справочник Статьи ДДС. Статья движения денежных средств, по которой производится платеж. При выборе статьи "Выдача заработной платы" становятся видимыми реквизиты поиска зарплатной ведомости. Данные реквизиты используются для поиска ведомостей при перегрузке Банковской выписки в Бухгалтерию предприятия 3.0.
  • Контрагент. Реквизит является обязательным для заполнения, кроме операций перемещения между счетами, в кассу, из кассы . Ссылается на справочник Контрагенты и контакты. Плательщик или получатель денежных средств.
  • Договор. Реквизит является обязательным для заполнения, кроме операций перемещения между счетами, в кассу, из кассы. Ссылается на справочник Договоры взаиморасчетов. Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Счет получатель/счет отправитель. Реквизит обязательный для статей ДДС, в которых выбраны виды операций - "Перевод на другой счет организации", "Перевод с другого счета организации".
  • Касса. Реквизит обязательный для статей ДДС, в которых выбраны операции - "Взнос наличными в кассу", "Снятие наличных в кассу".
  • Сумма приход. Сумма поступления денежных средств на банковский счет компании. Поле доступно, если выбрана приходная статья движения денежных средств.
  • Сумма расход. Сумма расхода денежных средств банковского счета. Поле доступно, если выбрана расходная статья движения денежных средств.
  • Сумма услуг. Сумма банковских услуг, которые связаны с выполнением операций банковского счета.
  • % НДС. Ставка НДС по строке.
  • НДС. Сумма НДС по строке.
  • Назначение платежа. Назначение платежа.
  • Фискальные реквизиты. Эта группа реквизитов используется в том случае, если оплата осуществляется через фискальный регистратор (фискальный номер чека, сквозной номер документа ФР, номер смены ФР и дата пробития чека).

Табличная часть «Состав»

  • Курс. Курс валюты, по которой ведутся взаиморасчеты с текущим контрагентом.
  • Сделка. ПВХ Типы сделок. Документ, по которому были уплачены (получены) денежные средства.
  • Документ планирования. Ссылается на документ Заявка на Расход ДС, План поступления ДС. Документ планирования поступления или расхода ДС.
  • Сумма приход. Сумма поступления денежных средств на банковский счет компании. Поле доступно, если выбрана приходная статья движения денежных средств.
  • Сумма расход. Сумма расхода денежных средств банковского счета. Поле доступно, если выбрана расходная статья движения денежных средств.
  • % комиссии. Процент комиссии банка за перевод денежных средств банковского счета.
  • Сумма услуг. Сумма банковских услуг, которые связаны с выполнением операций банковского счета.
  • Сумма разница. Сумма поступления/расхода денежных средств на банковский счет без суммы банковских услуг.
  • НДС. Сумма НДС по строке.
  • Авто закрытие долгов. Если флажок установлен, то при проведении документа все незакрытые сделки будут автоматически закрываться (в порядке ФИФО).
  • Корр. счет. Ссылается на план счетов Регламентированный. Корреспондирующий счет.

Таблица вкладки Состав может быть автоматически заполнена при помощи пунктов меню кнопки Заполнение:

  • По платежным поручениям,
  • По планам поступления ДС,
  • По заявкам на расход ДС,
  • Заполнить по текущей задолженности сотрудникам

Создание перемещений денежных средств в банковской выписке

С помощью банковской выписки можно делать операции перемещения денежных средств. Перемещение денежных средств - двойная операция, необходимо сделать документ расхода и обратный документ прихода. Ниже перечислены возможные для банковской выписки перемещения.

Операция указанная в Статье ДДС Вид движения
Перевод на другой счет организации Расход
Перевод с другого счета организации Приход
Взнос наличными из кассы Приход
Снятие наличных в кассу Расход

Ввод на основании

Документ Строка банковской выписки вводится на основании документов:

  • Заявка на расход ДС,
  • План поступления ДС,
  • Счет на оплату,
  • Счет на оплату за автомобили,
  • Инкассация,
  • Заявка на хранение шин.

На основании Строки банковской выписки могут вводиться документы:

  • Корректировка движений,
  • Счет-фактура выданный,
  • Счет-фактура полученный,
  • Авансовый отчет,
  • Чек коррекции,
  • Событие.
