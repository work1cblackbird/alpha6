Документ "Закрытие кассовой смены"

Назначение

Документ Закрытие кассовой смены предназначен для ввода обобщенной информации о продажах, совершенных в течение кассовой смены. При формировании документа табличные части заполняется товарами, реализованными за смену, предоставленными скидками и их суммами и списком контрагентов, осуществившие покупки. При проведении документа все свернутые чеки удаляются.

При проведении документ формирует хозяйственные операции:

  • Закрытие кассовой смены. Хозяйственная операция сворачивает чеки в пределах кассовой смены.
  • Акт о реализации. Хозяйственная операция отражает розничную продажу конечному покупателю.

Документ формируется в конце каждой кассовой смены. Формирование документа автоматически осуществляется регламентной обработкой Закрытие кассовой смены. В этом случае документу автоматически присваивается хозяйственная операция Закрытие кассовой смены. При этом ручное редактирование документа не допускается.

Документ также может быть введен вручную. При этом документу автоматически присваивается хозяйственная операция Акт о реализации. Редактирование документа возможно только при этом условии. Документ тогда выступает в качестве акта о реализации за смену, что может использоваться для работы, например, с неавтоматизированными торговыми точками.

Реквизиты

  • Касса ККМ. Реквизит является обязательным для заполнения. Ссылается на справочник Кассы ККМ. Касса ККМ, для которой была произведена свертка чеков.
  • Склад компании. Ссылается на справочник Склады компании. Отображается в случае, если документ был введен вручную. 

Табличная часть «Товары»

Табличная часть документа Товары содержит список всех товаров, проданных за смену через указанную кассу.

Кнопка Заполнение, находящаяся над табличной частью документа, позволяет выполнить автозаполнение табличной части по следующим схемам:

  • Очистить табличную часть. Из табличной части будут удалены все позиции.
  • Загрузить из ТСД. Вызов обработки загрузки данных с терминала сбора данных.
  • Загрузить из файла. Вызов обработки Загрузка данных из файла.

Кнопка Подбор позволяет выполнить подбор номенклатуры в обычном режиме. С помощью кнопки Подбор по списку заказанного можно осуществить подбор номенклатуры по списку товаров, заказанных покупателями.

При автоматическом формировании документа Закрытие кассовой смены кнопки Заполнение и Подбор отключаются.

В случае, если документ сформирован с помощью обработки Закрытие кассовой смены, в табличной части появляются дополнительные вкладки.

Табличная часть «Возвраты»

Табличная часть документа Возвраты содержит список всех товаров, возвращенных клиентами за смену через указанную кассу.

  • Дата реализации. Тип значения: «Дата». Указывается дата проведения документа, по которому был реализован исходный товар.
  • Номер чека.  Тип значения: «Строка». Номер чека по которому выданы деньги покупателю.

    • Табличная часть «Скидки»

    Вкладка Скидки содержит сводку всех скидок в свернутых чеках.

    • Тип скидки. Ссылается на справочник Типы скидок и наценок. Тип скидки, примененный при продаже. Если ячейка таблицы в графе Тип скидки не заполнена, то это означает, что соответствующие продажи были произведены без скидок.
    • Сумма. Сумма продажи без скидки.
    • Скидка. Сумма скидки.
    • Всего. Сумма продажи с учетом скидки.
    • Чеков. Количество чеков, в которых была применена данная скидка.

    Табличная часть «Контрагенты»

    Вкладка Контрагенты содержит список покупателей - контрагентов и владельцев дисконтных карточек, по которым происходил расчет за проданные товары. 

    Табличная часть «Оплаты»

    Вкладка Оплаты содержит информацию о формах оплаты, которые осуществлялись за смену (например, наличными, платежной картой и т. п.). В графе Тип оплаты указывается тип оплаты. Остальные графы уточняют описание оплаты.

    Табличная часть «Возврат оплаты»

    Вкладка Оплаты содержит информацию о формах возврата оплаты, которые осуществлялись за смену (например, наличными, платежной картой и т. п.). В графе Тип оплаты указывается тип оплаты. Остальные графы уточняют описание оплаты.

    Табличная часть «Свойства»

    На вкладке Свойства описываются свойства документа Чек, если они заданы. Свойства задаются в плане видов характеристик Свойства объектов. Свойства чека могут использоваться, например, для маркетинговых исследований.

    Ввод на основании

    Документы, вводимые на основании:

    • Возврат от покупателя,
    • Корректировка движений,
    • Счет-фактура выданный.

    Печатные формы

    • Закрытие смены.  Простая печатная форма, содержит список реализованных товаров.
    • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников реализованных товаров.
