Документ "Бюджет движения денежных средств"

Назначение

Документ Бюджет движения денежных средств предназначен для планирования движения денежных средств подразделений в разрезе статей ДДС. Запланированные документом данные относятся к выбранному периоду планирования, который  соответствует периодичности сценария планирования (поддерживаются следующие периоды планирования: месяц, квартал, год). Планируемый период не может быть меньше даты документа.

Предусмотрена возможность отражения факта отсрочки платежа - смещение оплаты относительно планируемого периода. Для этого в документе заполняются не только оборотные суммы по каждой статье, но и процентное распределение этих сумм по периодам - график погашения по периодам. График может быть общим для всех статей, либо индивидуальным для каждой статьи.

Документ относится к подсистеме Бюджетирование раздела Предприятие.

При проведении документ формирует хозяйственные операции:

  • Бюджет движения денежных средств. Формируются движения по регистру накопления: Бюджет ДДС, в котором производится накопление запланированных данных. 

Реквизиты

  • Сценарий планирования.  Реквизит является обязательным для заполнения. Ссылается на справочник Сценарии планирования. Определяет периодичность планирования.
  • Использовать раздельные графики погашения. Если этот флажок установлен, то для каждой статьи используются раздельные графики распределения платежей; иначе для всех статей используется один график.
  • Период. Реквизит является обязательным для заполнения. Начальная дата периода, на который происходит планирование. Зависит от реквизита Периодичность выбранного сценария планирования.

Табличная часть «База распределения ДДС»

Основная табличная часть документа, предназначенная для хранения запланированных данных.

  • Статья ДДС. Ссылается на справочник Статьи ДДС.
  • Базовая сумма (упр.). Прогнозируемый оборот по статье.
  • Базовый бюджет. Ссылается на документы вида: Бюджет продаж, Бюджет закупок. Определяет бюджет, на основании которого заполнена сумма предполагаемого движения по статье.

Табличная часть «График погашения по периодам»

Подчиненная табличная часть документа, предназначенная для хранения данных о распределении платежей по периодам.

  • Смещение. Смещение периода относительно базового.
  • Период. Смещение периода относительно прогнозируемого. Может принимать отрицательные значения (предоплата), Если равно нулю, то период равен планируемому периоду. 
  • %. Процент платежа в текущем периоде от общей суммы по статье.
  • Сумма погашения (упр.). Сумма платежа в текущем периоде в валюте управленческого учета.

Ввод на основании

Может вводиться на основании документа Бюджет ДДС.

Объекты, вводимые на основании:

  • Бюджет ДДС,
  • Корректировка движений.

Печатные формы

  • Бюджет ДДС без графиков. Простая печатная форма бюджета ДДС без графиков.
  • Бюджет ДДС с графиками. Простая печатная форма бюджета ДДС с графиками.
