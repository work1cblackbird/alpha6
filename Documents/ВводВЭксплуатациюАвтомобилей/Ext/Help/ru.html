Документ "Ввод в эксплуатацию автомобилей"

Назначение

Документ предназначен для отражения в учете факта ввода в эксплуатацию автомобилей с указанием склада, по которому происходит списание и указанием подразделения компании, которое принимает автомобили на баланс.

При проведении документ формирует хозяйственные операции:

  • Ввод в эксплуатацию автомобилей. Хозяйственная операция уменьшает автомобили на складах предприятия, хранимые в регистре «Остатки автомобилей», с указанием партии списания, отражает сумму списанных автомобилей на регистре «Доходы и расходы», приходует автомобили в подразделение-получатель, хранимые на регистре «Прочие активы в эксплуатации». Разницу суммы списания и балансовой стоимости введенных в эксплуатацию автомобилей отражает на регистре «Доходы и расходы».
  • Перевод автомобиля в тест-драйв. Хозяйственная операция аналогична операции Ввод в эксплуатацию автомобилей, с той разницей, что дает возможность использовать автомобиль для тест драйва, оприходовав его на регистр «Автомобили для тест-драйва».

Параметры

Длина номера: 10

Тип номера: Строка

Периодичность номера: Год

Вводится на основании: справочники «Прочие активы», «Автомобили», документы «Поступление автомобилей», «Перемещение автомобилей».

Объекты, вводимые на основании: «Расходный складской ордер на автомобили», «Событие», «Вывод из эксплуатации автомобилей», «Перемещение активов», «Списание активов».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Подразделение-получатель. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, принимающее на баланс нетоварные активы.
  • Материально ответственное лицо. Реквизит является обязательным для заполнения. Ссылается на справочник «Сотрудники». Материально ответственное за учёт нетоварных активов лицо.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором происходит списание.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Документ основание. Ссылается на документы «Перемещение товаров», «Поступление товаров». Документ, на основании которого введен данный.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").

Табличная часть «Активы»

  • Актив. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Прочие активы». Нетоварный актив.
  • Тип эксплуатации. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы эксплуатации». Тип эксплуатации.
  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль.
  • Балансовая стоимость. Реквизит является обязательным для заполнения. Тип значения: «Число». Балансовая стоимость. По умолчанию рассчитывается по сумме списываемой номенклатуры по партиям.
  • Максимальная продолжительность эксплуатации. Тип значения "Число". Максимальная продолжительность эксплуатации автомобиля на тест-драйве. Доступно только для хозяйственной операции "Перевод автомобиля в тест-драйв".
  • Максимальный пробег. Тип значения "Число". Максимальный пробег автомобиля на тест-драйве. Доступно только для хозяйственной операции "Перевод автомобиля в тест-драйв".

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Ввод в эксплуатацию. Простая печатная форма, содержит список введенных в эксплуатацию нетоварных активов.
  • ОС-1 (Акт о приеме-передаче объекта ОС). Унифицированная печатная форма №ОС-1, утвержденная постановлением Госкомстата России.
  • М-11 (Требование-накладная). Унифицированная печатная форма №М-11, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Склад автосалона --> Производство --> Ввод в эксплуатацию автомобилей" и "Аренда автомобилей --> Управление автомобилями аренды --> Ввод в эксплуатацию автомобилей".

Журналы, отображающие документ: «Журнал документов учета нетоварных активов».

Движения

  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Остатки автомобилей».
  • Остаточный регистр «Прочие активы в эксплуатации».

Последовательности, изменяемые документом: «автомобили».
