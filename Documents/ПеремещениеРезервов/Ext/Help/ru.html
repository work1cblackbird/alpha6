Документ "Перемещение резервов"

Назначение

Документ Перемещение резервов предназначен для оформления операций по резервированию товаров под заказ за счет снятия резервов с других заказов. В табличной части документа не должны фигурировать услуги, работы и т.п. Относится к подсистеме Резервирование товаров раздела Продажи.

Основным режимом ввода является ввод на основании документов Заказ внутренний или Заказ покупателя, однако, при необходимости, документ можно заполнить вручную.

При проведении документ формирует хозяйственные операции:

  • Перемещение резерва. Хозяйственная операция списывает резервы товаров с одного заказа и резервирует этот товар под другой заказ на регистре Заказы покупателей. Также снимает распределение заказа-основания с регистра Заказы распределение, если товар, который будет зарезервирован, уже распределен по заказам поставщику.

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочники Контрагенты и контакты, Подразделения компании. Покупатель или подразделение компании, чьи заказы снимаются с резерва.
  • Заказ. Реквизит является обязательным для заполнения. Ссылается на документы: Заказ внутренний, Заказ покупателя. Заказ, на который будет перемещен резерв. Также у заказа автоматически спишется избыточное распределение по заказам поставщику, т.е. сумма резерва и распределения не может превышать количество заказа.
  • Корректировка заказа. Если данный флажок установлен, то при снятии резерва по заказу будет также на эту величину уменьшено количество заказанного товара.

Табличная часть «Товары»

Добавочные графы:

  • Резерв. Реквизит является обязательным для заполнения. Количество единиц измерения номенклатуры, резервируемых на складе.
  • Заказ. Реквизит является уникальным и обязательным для заполнения. Ссылается на документы: Заказ внутренний, Заказ покупателя. Заказ покупателя, с которого будет снят резерв. 
  • Место размещения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Склады компании. Склад компании, с которого будет сниматься резерв покупателя или подразделения.

Ввод на основании

Документ Перемещение резервов вводится на основании документов:

  • Заказ внутренний,
  • Заказ покупателя.

Документы, вводимые на основании:

  • Корректировка движений.

Печатные формы

  • Перемещение резервов. Простая печатная форма, содержит список товаров, для которых производится снятие резервов.
