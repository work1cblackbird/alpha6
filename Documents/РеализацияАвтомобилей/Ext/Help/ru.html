Документ "Реализация автомобилей"

Назначение

Документ "Реализация автомобилей" предназначен для фиксации факта реализации (отгрузки) автомобилей. Отгрузка и отпуск автомобилей на комиссию (передачи автомобилей на реализацию) в конфигурации фиксируется одним и тем же документом. Вид документа определяется хозяйственной операцией.

При проведении документ формирует хозяйственные операции:

  • Реализация автомобилей. Хозяйственная операция списывает кредиторскую задолженность покупателя по другим сделкам, если таковые были и начисляет дебиторскую задолженность покупателя перед компанией на регистре «Взаиморасчетов компании», уменьшает количество автомобилей, на складах предприятия в виде автомобилей с собственным или комиссионным признаком партии, хранимые на регистре «Остатки автомобилей» и приходует сумму и себестоимость проданных автомобилей на регистре «Продажи автомобилей». А так же регистрирует сумму продажи на регистре сведений «Накопления по карточкам контрагентов».
  • Передача автомобилей на комиссию. Хозяйственная операция аналогична операции "Реализация автомобилей" за тем исключение, что дебиторская задолженность комиссионеру не начисляется, а начисляется товарная задолженность на регистре «Автомобили отданные», так же не приходуется и не регистрируется сумма проданных автомобилей на соответствующих регистрах.

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Счет на оплату за автомобили», «Поступление автомобилей», «Перемещение автомобилей», «Заказ на автомобиль», «Ввод остатков автомобилей».

Объекты, вводимые на основании: «Возврат автомобилей от покупателя», «Жалоба клиента», «Заявка на расход ДС», «Корректировка реализации», «Корректировка реализации автомобилей», «Приходный кассовый ордер», «Отчет комитенту за автомобили», «Отчет комиссионера за автомобили», «Счет на оплату за автомобили», «Счет фактура выданный», «Расходный ордер на склад», «Событие», «Чек на оплату», «Уведомление о перемещении прослеживаемых товаров».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Менеджер. Сотрудник, ответственный за реализацию автомобиля. Ссылается на справочник «Сотрудники».
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты».  Покупатель или комиссионер, с которым ведутся взаиморасчеты за автомобиль.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с покупателем или комиссионером.
  • Заказчик. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Покупатель или комиссионер, которому отгружается товар.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого продаем автомобили.
  • Документ основание. Ссылается на документы «Счет на оплату за автомобили», «Поступление автомобилей», «Перемещение автомобилей», «Заказ на автомобиль». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, в которых отгружаются автомобили.
  • Скидка/наценка. Ссылается на справочник «Типы скидок и наценок». Скидка/наценка документа.
  • Сумма. Сумма документа.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сбросить сведения о цене. Тип значения: «Булево». При установленном флаге в регистр сведений «Цены автомобилей» вносится запись с нулевой ценой для типа цен, указанного в документе.
  • Включать в стоимость автомобиля стоимость оборудования. Тип значения: «Булево». Флаг позволяет учитывать (включать) стоимость оборудования автомобиля в стоимость самого автомобиля, при формировании счета-фактуры. При установленном флаге, стоимость оборудования включается в стоимость автомобиля, при снятом флаге - не включается.
    Так же данный флаг влияет на формирование ЭДО и печатной формы УПД. Данные для заполнения берутся из счета-фактуры. Если счет-фактура отсутствует, то заполнение печатных форм производится с учетом флага "Включать в стоимость автомобиля стоимость оборудования".
    По умолчанию флаг установлен в значение «Истина».

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Отгружаемый автомобиль.
  • Количество. Реквизит является обязательным для заполнения. Количество автомобилей. Всегда равно 1.
  • Цена. Цена автомобиля.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС автомобиля.
  • Сумма НДС. Сумма НДС автомобиля.
  • Процент скидки. Процент скидки на отгружаемый автомобиль.
  • Сумма скидки. Сумма скидки по строке документа.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.
  • Автомобиль без заказа. Признак того, что автомобиль отгружается покупателю без предварительного заказа.
  • Заказ на автомобиль. Ссылается на документ «Заказ на автомобиль». Заказ на автомобиль,  предварительный заказа которого, оформляется  данным  документом. 

Табличная часть «Товары» (Оборудование автомобиля)

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Реализуемый или отданный на комиссию автомобиль.
  • Номенклатура.  Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (оборудование).
  • Количество. Реквизит является обязательным для заполнения. Количество оборудования установленного в автомобиль.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения оборудования.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Цена оборудования.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС оборудования.
  • Сумма НДС. Сумма НДС оборудования.
  • Процент скидки. Процент скидки.
  • Сумма скидки. Сумма скидки по строке документа.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.

Табличная часть «Зачет авансов»

Вкладка Зачет авансов содержит способ зачета оплаты/авансов, а так же список подобранных вручную документов оплаты и суммы оплаты для погашения задолженности. Так же в список можно добавить документы с помощью кнопки Подбор авансов. Данный список документов оплат доступен только при выборе способа зачета оплаты/авансов "По документу".
  • Документ аванса. Реквизит является уникальным и обязательным для заполнения. Документ, на основании которого будет погашена задолженность.
  • Сумма. Реквизит обязательным для заполнения. Сумма для зачета по документу аванса.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Реализация автомобилей. Простая печатная форма, содержит список отгружаемых автомобилей покупателю или комиссионеру.
  • Торг-12. Унифицированная печатная форма №Торг-12, утвержденная постановлением Госкомстата России.
  • 1-Т (Транспортная накладная). Унифицированная печатная форма №1-T, утвержденная постановлением Госкомстата России.
  • Приложение №4 (Транспортная накладная). Приложение №4 к унифицированной печатной форме №1-Т.
  • Акт приема-передачи ТС. Унифицированная печатная форма акта приема-передачи транспортного средства.
  • Универсальный передаточный документ. Универсальный передаточный документ, утвержденный письмом ФНС России.
  • Договор купли-продажи автомобиля. Унифицированная печатная форма договора купли-продажи автомобиля.

Способ вызова

Вызывается из меню "Автомобили с пробегом --> Продажи --> Реализации автомобилей". Также документ можно вызвать из меню "Продажа автомобилей --> Отгрузка --> Реализации автомобилей".
