Документ "Акт разногласий"

Назначение

Документ Акт разногласий предназначен для ввода корректировок в закрытый заказ-наряд. Относится к подсистеме Автосервис раздела Автосервис.

При вводе на основании документа Заказ-наряд табличная часть документа Акт разногласий заполняется выполненными работами и использованными при этом деталями в соответствии с документом-основанием. При помощи документа Акт разногласий возможно изменение количества, цен, скидок, нормочасов с перерасчетом суммы заказ-наряда. Документ может быть введен только
на основании закрытого заказ-наряда с платным видом ремонта.

Документ Акт разногласий используется в случае, если существуют какие-либо разногласия относительно стоимости ремонта. Все эти разногласия отражаются в данном документе.

Список хозяйственных операций:

  • Акт разногласий. Хозяйственная операция начисляет дебиторскую задолженность плательщика по заказ-наряду или сторнирует его задолженность на регистре Взаиморасчеты компании. Так же осуществляется увеличение проданных деталей/работ по заказ-наряду или сторнирование проданных деталей/работ по заказ-наряду на регистре Продажи.

Реквизиты

  • Заказчик. Реквизит является обязательным для заполнения. Ссылается на справочник Контрагенты и контакты. Заказчик выполнения работ по заказ-наряду.
  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник Автомобили. Ремонтируемый автомобиль. Справа находится гиперссылка, по которой открывается отчет История автомобилей, сформированный для данного автомобиля.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник Контрагенты и контакты. Контрагент-плательщик по заказ-наряду.
  • Договор взаиморасчетов. Реквизит является обязательным для заполнения. Ссылается на справочник Договоры взаиморасчетов. Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Документ-основание. Сумма работ по заказ-наряду. Документ Заказ-наряд, на основании которого был введен данный документ Акт разногласий.

Табличная часть «Автотовары»

На вкладке Автоработы перечисляются работы для выполнения заказ-наряда.

Табличная часть «Товары»

На вкладке Товары перечисляются — запчастей для выполнения работ по заказ-наряду.

 

Ввод на основании

Документ Акт разногласий может вводиться на основании документа Заказ-наряд.

Документы, вводимые на основании:

  • Счет на оплату,
  • Корректировка движений,
  • Счет-фактура выданный.

Печатные формы

  • Заказ-наряд;
  • ТОРГ-12 (Товарная накладная);
  • Акт об оказании услуг.
