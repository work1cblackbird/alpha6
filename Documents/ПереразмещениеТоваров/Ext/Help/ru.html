Документ "Переразмещение товаров"

Назначение

Документ Переразмещение товаров предназначен для автоматизации процесса перемещения товаров внутри ячеистого склада. Относится к подсистеме Ордерный склад раздела Склад.

При проведении документ формирует хозяйственные операции:

  • Переразмещение товаров. Хозяйственная операция списывает товары из одной ячейки (Старая ячейка) склада по регистру Остатки товаров на ордерном складе и приходует в другую (Новая ячейка).
  • Переразмещение шин. Хозяйственная операция аналогична операции Переразмещение товаров за тем исключением, что в регистре Остатки товаров на ордерном складе заполняется дополнительное измерение - "Заявка на хранение шин".

Реквизиты

  • Склад компании. Ссылается на справочник Склады компании. Склад компании, на котором происходит переразмещение товаров.
  • Склад-получатель. Ссылается на справочник Склады компании. Склад компании, на который приходуется товар. 

 

Меню кнопки Заполнение содержит добавочные пункты, которые служат для автоматического заполнения таблицы документа:

  • Заполнить складскими остатками.
  • Заполнить складскими остатками по ячейке.
  • Заполнить остатками по группе номенклатуры.
  • Погашение минусов: из ячеек с положительным остатком товары записываются в ячейку с отрицательным остатком так, чтобы в новой ячейке количество стало нулевым.
  • Оптимизация: собираются одинаковые товары из разных ячеек в одну — в ту, количество товара в которой было максимальным.
  • Заполнить ячейки у выбранных товаров.

Ввод на основании

Вводится на основании:

  • Приходный складской ордер,
  • Переразмещение товаров.

Объекты, вводимые на основании:

  • Расходный складской ордер,
  • Корректировка движений,
  • Переразмещение товаров.

Печатные формы

  • Переразмещение товаров. Простая печатная форма документа.

Журналы, отображающие документ

  • Складские ордера.
