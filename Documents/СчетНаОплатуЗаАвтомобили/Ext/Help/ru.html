Документ "Счет на оплату за автомобили"

Назначение

Документ служит для фиксации предварительной договоренности о приобретении контрагентом автомобилей, и может являться документом, на основании которого производится отгрузка автомобилей. При вводе контрагента из его реквизитов автоматически подставляются в документ договор, текущая скидка. Эти значения потом можно поменять.

Документ содержит хозяйственную операцию:

  • Счет на оплату за автомобили. Не создает движений.
  • Счет на предоплату за автомобили. Не создает движений. 

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Заказ на автомобиль», «Реализация автомобилей».

Объекты, вводимые на основании: «Реализация автомобилей».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Комментарий. Произвольный комментарий к документу.
  • Покупатель. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Покупатель, которому выписываем счет.
  • Договор с покупателем. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с покупателем.
  • Заказчик. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Заказчик, которому выписываем счет.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, в которых выписывается счет.
  • Сумма. Сумма документа.
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Документ основание. Ссылается на документы: «Заказ на автомобиль», «Реализация автомобилей». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).  
  • Скидка/наценка. Ссылается на справочник «Типы скидок и наценок». Скидка/наценка, сделанная на все автомобили, указанные в документе.
  • Расчетный счет организации. Реквизит является обязательным для заполнения. Ссылается на справочник «Банковские счета». Номер расчетного счета нашей организации.
  • Действителен до. Тип значения: «Дата». Срок действия счета на оплату за автомобили.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили» или «Модели автомобилей». Автомобиль или модель автомобиля, счет на который выписывается.
  • Вариант комплектации автомобиля. Ссылается на справочник «Варианты комплектации». Вариант комплектации модели автомобиля.
  • Количество. Реквизит является обязательным для заполнения. Количество автомобилей. Всегда равно 1.
  • Цена продажная. Цена автомобиля.
  • Сумма продажи. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС автомобиля.
  • Сумма НДС. Сумма НДС автомобиля.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.
  • Сумма скидки. Сумма скидки по строке документа.
  • Процент скидки. Процент скидки по строке документа.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Счет на оплату. Печатная форма, выставляемого счета покупателю.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Отгрузка автомобилей --> Счет на оплату за автомобили".
