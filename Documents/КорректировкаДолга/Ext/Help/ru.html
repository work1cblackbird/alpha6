Документ "Корректировка долга"

Назначение

Документ Корректировка долга предназначен для ручной корректировки долга контрагента. В документе указывается контрагент, договор и сделка, по которой необходимо произвести корректировку долга. Относится к подсистеме Взаиморасчеты раздела Финансы.

При проведении документ формирует хозяйственную операцию:

  • Корректировка долга. Хозяйственная операция начисляет или удерживает сумму задолженности контрагента перед компанией или перед другим контрагентам на регистре «Взаиморасчеты компании», приходует или списывает откорректированную разницу долга на регистр «Доходы и расходы» компании.

В табличной части документа может быть введено несколько строк по различным договорам и сделкам, оформленным с контрагентом. Суммы вводятся в валюте взаиморасчетов по договору с контрагентом.

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, долг которому мы корректируем.
  • Статья дохода, Статья расхода. Реквизиты являются обязательными для заполнения. Ссылаются на справочник «Статьи доходов и расходов». Две статьи, в разрезе которых будет изменен баланс доходов и расходов.
  • Долг контрагента составляет. Гиперссылка отражает текущее состояние расчетов с контрагентом по договору. Производится переход к отчету Взаиморасчеты с контрагентами, сформированному для указанного контрагента.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.

Табличная часть «Состав»

  • Договор. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, взаиморасчеты по которому подлежат корректировке.
  • Курс взаиморасчетов. Курс валюты, по которой ведутся взаиморасчеты.
  • Сделка / Документ расчетов. Сделка, взаиморасчеты по которой подлежат корректировке.
  • Заказ. Заказ, по которому произошли расчеты с контрагентом.
  • Тип расчета. Признак, по которому определяется вид оплаты: аванс или по документу.
  • Способ зачета авансов. Доступен только для договоров с признаком учета взаиморасчетов "По сделкам". Если установлено в значение "Автоматически", то при проведении документа все незакрытые сделки будут автоматически закрываться (в порядке ФИФО). В значении "Не зачитывать" - незакрытые сделки не будут погашены.
  • Долг +. Сумма, на которую следует увеличить долг контрагента.
  • Долг -. Сумма, на которую следует уменьшить долг контрагента.

Ввод на основании

Документ Корректировка долга может быть создан на основании следующих документов: Приходный кассовый ордер, Расходный кассовый ордер.

На основании документа Корректировка долга может быть введен документ Корректировка движений.

Печатные формы

  • Корректировка долга. Простая печатная форма, содержит корректировку долга в разрезе договоров и сделок.
