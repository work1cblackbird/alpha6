Документ "Договор аренды"

Назначение


Документ отражает заключение с клиентом сделки по аренде автомобилей и является основанием для выставления клиенту счетов и актов, выдачи автомобилей клиенту, печати договора и доверенности на управление транспортными средствами.


Относится к подсистеме «Аренда автомобилей»


Вид аренды - значение классификация аренды из соответствующего справочника. Обязательное для заполнения поле. От выбранного вида аренды зависит внешний вид формы и состав доступных полей.


Период аренды — даты начала и окончания аренды. Для удобства редактирования данных добавлено изменение времени в отдельных полях. Также присутствует команда для выбора периода аренды в отдельном окне.


Основное


Заказчик — клиент, являющийся конечным пользователем аренды автомобилей.


Контрагент — плательщик по договору, при платном виде аренды поле обязательно для заполнения, при бесплатном — поле недоступно. На указанного контрагента выставляются счета на оплату, акты, контрагент является Арендатором по договору.


Договор взаиморасчетов — договор контрагента с признаком «Аренда». На указанный договор выставляются счета на оплату услуг аренды и актов об оказании услуг.


Договор депозита — договор с признаком «Депозит аренды». Для платного вида аренды владельцем договора является контрагент, для бесплатного — заказчик. Поле доступно если в виде аренды установлен признак использования депозита. На указанный договор выставляется счет на оплату депозита.


Состояние — текущее состояние договора. Состояния изменяются вручную пользователем, историю состояний можно посмотреть в меню дополнительных функций формы.


Источник информации — возможность вести аналитику по эффективности рекламных кампаний, маркетинговых мероприятий по продвижению услуг аренды.


Подписант от организации — данные ответственного лица организации, использующиеся при печати договора и ряда приложений к договору.


Подписант от контрагента — данные ответственного лица клиента, использующиеся при печати договора и ряда приложений к договору. В случае платного вида аренды подписант относится к контрагенту, в случае бесплатного — к заказчику.


Действующее соглашение — группа полей заполняется в том случае, если по договору введены изменения условий. Предназначены для быстрого перехода к последнему введенному изменению. Также присутствует кнопка ввода нового изменения договора аренды.


Водители


Список допущенных к управлению автомобилями лиц. Печатная форма доверенности формируется для указанных в списке водителей. Водители выбираются из справочника «Контрагенты». Также для каждого водителя возможно указания паспорта и водительского удостоверения. В случае, когда заказчиком является физическое лицо, в командной панели доступна команда добавления заказчика в список водителей. Водителей можно выбирать из списка контактных лиц заказчика, для этого добавлена кнопка в командной панели таблицы водителей.


Автомобили


Список передаваемых по договору аренды автомобилей. Для платных видов аренды выводятся стоимость аренды, сумма НДС, процент и сумма скидки. Если по виду аренды предусмотрен депозит, то отображается сумма депозита автомобиля, если по виду аренды контролируется приемка и выдача, то в таблице отображается текущий статус автомобиля, а в командной панели таблицы доступны документы «Выдать автомобиль» и «Принять автомобиль». Доступен ввод автомобилей из соответствующего справочника и подбор с использованием планировщика аренды.


Дополнительные услуги


Список услуг, включенных в договор аренды. Отображается только для платного вида аренды. Состав полей таблицы дополнительных услуг соответствует составу документов по реализации услуг. Дополнительные услуги включаются в счета на оплату услуг аренды а также в акты об оказании услуг. Присутствует возможность подбора услуг из справочника «Номенклатура» и быстрый подбор услуг из вида аренды.


Тарификация


Таблица предназначена для визуализации расчета стоимости аренды и содержит информацию по стоимости периодов аренды. Отображается только для платного вида аренды. Состав отображаемых периодов зависит от настроек вида аренды.


Дополнительные условия


Место выдачи автомобилей — блок параметров, определяющий адрес по которому будет производиться выдача автомобилей. Доступен выбор одного из трех вариантов — фактический адрес указанного подразделения компании, фактический адрес указанного контрагента, произвольный адрес. По умолчанию устанавливается адрес подразделения документа.


Место приемки автомобилей — блок параметров, определяющий адрес по которому будет производиться приемка автомобилей. Доступен выбор одного из трех вариантов — фактический адрес указанного подразделения компании, фактический адрес указанного контрагента, произвольный адрес. По умолчанию устанавливается адрес подразделения документа.


Условия использования — произвольный текст для регистрации особых договоренностей с клиентом, который выводится при печати договора в соответствующем разделе.


Печатные формы


Для документа предусмотрены печатные формы:


Договор аренды — представлен в двух вариантах — для физического лица с выводом паспортных данных заказчика и для юридического с выводом реквизитов контрагента. В случае платной аренды договор заключается с контрагентом (плательщиком)


Договор безвозмездного использования — представлен в двух вариантах — для физического лица с выводом паспортных данных заказчика и для юридического с выводом реквизитов заказчика. В случае использования депозита по бесплатному виду аренды в договор будет включен пункт про необходимость произвести обеспечительный платеж.


Акт приема-передачи транспортного средства — используется для оформления факта передачи автомобиля клиенту. Для каждого автомобиля формируется отдельная печатная форма.


Акт возврата транспортного средства — используется для оформления факта получения автомобиля от клиента. Для каждого автомобиля формируется отдельная печатная форма.


Доверенность — доверенность на управление транспортным средством. Формируется одна доверенность на все автомобили с указанием всех водителей.


Тарификация аренды — выводятся стоимости аренды автомобиля для предусмотренных видом аренды периодов. Также выводится список дополнительных услуг из вида аренды с указанием цен, согласно ценообразованию документа.
