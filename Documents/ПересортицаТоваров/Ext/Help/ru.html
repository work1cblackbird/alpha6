Документ "Пересортица товаров"

Назначение

Документ Пересортица товаров предназначен для устранения возникающей пересортицы (ошибок списания товара при продаже). С помощью этого документа можно внести в учетные данные
исправления, устраняющие пересортицу как на оптовом, так и на розничном складе. В таблице документа не могут фигурировать услуги, работы и т. п.

Относится к подсистеме Инвентаризация товаров раздела Склад.

При проведении документ формирует хозяйственную операцию:

  • Пересортица товаров. Хозяйственная операция списывает недостающий товар и приходует имеющийся на выбранный склад, в том случае, если была обнаружена пересортица, на регистрах Остатки товаров компании и Партии товаров компании, приходуем или списываем суммовую разницу, выявленную при пересортице, на регистр Доходы и расходы, а так же установка розничных цен, в случае изменения цен на розничном складе на регистре Цены.

Реквизиты

  • Склад компании. Реквизит является обязательным для заполнения. Ссылается на справочник Склады компании. Склад компании, на котором произошла пересортица.
  • Причина. Тип значения: «Строка». Предназначен для указания причины пересортицы.

Табличная часть «Товары»

  • Номенклатура расход. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Номенклатура. Списываемая номенклатура.
  • Характеристика расход. Реквизит является уникальным. Ссылается на справочник Характеристики номенклатуры. Характеристика списываемой номенклатуры.
  • Партия. Реквизит является обязательным для заполнения, если у пользователя установлено право Разрешить выборочное списание партий. Ссылается на ПВХ Типы партий. Партия, с которой будет списан товар.
  • Ед., Коэф. Единица измерения номенклатуры и коэффициент пересчета.
  • Остаток расход. Количество расходуемого товара на складе.
  • Количество расход. Реквизит является обязательным для заполнения. Количество списываемого товара.
  • Понижение сортности. Если флажок установлен, то партия товара сохраняется, но переводится на другую номенклатуру. Если флажок снят, то номенклатура сохраняется, но выбирается другая партия или характеристика.
  • Ячейка расход. Ссылается на справочник Ячейки хранения. Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов.
  • Цена розничная расход. Цена розничная списываемого товара. Поле появляется только при задании розничного склада.
  • Сумма розничная расход. Сумма розничная списываемого товара. Поле появляется только при задании розничного склада.
  • Номенклатура приход. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Номенклатура. Приходуемая номенклатура.
  • Количество приход. Реквизит является обязательным для заполнения. Количество приходуемого товара.
  • Цена розничная приход. Цена розничная приходуемого товара. Поле появляется только при задании розничного склада.
  • Сумма розничная приход. Сумма розничная приходуемого товара. Поле появляется только при задании розничного склада.
  • Сумма переоценки. Общая сумма переоценки по строке (отрицательная сумма выводится красным шрифтом).
  • Цена. Цена приходуемого товара.
  • Сумма. Сумма приходуемого товара.
  • ГТД. Ссылается на справочник ГТД. Номер грузовой таможенной декларации (для товаров, поступивших по импорту).
  • Ячейка. Ссылается на справочник Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов. 

Ввод на основании

Документы, вводимые на основании:

  • Корректировка движений,
  • Приходный складской ордер,
  • Расходный складской ордер,
  • Переоценка товаров в рознице.

Печатные формы 

  • Переоценка оприходованных товаров. Простая печатная форма, содержит список переоцененных товаров.
  • Пересортица. Простая печатная форма, содержит список приходуемых и расходуемых при пересортице товаров.
  • Сличительная ведомость (ИНВ-19). Унифицированная печатная форма №ИНВ-19, утвержденная постановлением Госкомстата России.
