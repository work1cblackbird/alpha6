Документ "Заявка на аренду"

Назначение


Документ предназначен для регистрации намерения клиента получить автомобиль в аренду, калькуляции стоимости аренды автомобиля и дополнительных услуг, регистрации контактной информации клиента и параметров выбора автомобиля.


Относится к подсистеме «Аренда автомобилей»


Вид аренды - значение классификация аренды из соответствующего справочника. Обязательное для заполнения поле. От выбранного вида аренды зависит внешний вид формы и состав доступных полей.


Период аренды — даты начала и окончания аренды. Для удобства редактирования данных добавлено изменение времени в отдельных полях. Также присутствует команда для выбора периода аренды в отдельном окне.


Основное


Клиент - группа полей, содержащие данные заказчика и плательщика аренды. Заказчика можно выбрать из справочника «Контрагенты» либо указать обращение к заказчику, также фиксируется контактные данные клиента. Для договора взаиморасчетов контрагента должен быть установлен признак «Аренда». Заполнение контрагента и договора взаиморасчетов доступно для платного вида аренды.


Параметры подбора автомобиля — набор характеристик, по которым производится подбор автомобиля.


Условия использования — описание ограничений по аренде автомобиля в произвольной форме.


Источник информации — возможность вести аналитику по эффективности рекламных кампаний, маркетинговых мероприятий по продвижению услуг аренды.


Причина отказа от обслуживания — заполняется в случае установки статуса «Отклонено» для заявки на аренду для дальнейшего анализа причин потери клиентов.


Автомобили


Список передаваемых по договору аренды автомобилей. Для платных видов аренды выводятся стоимость аренды, сумма НДС, процент и сумма скидки. Если по виду аренды предусмотрен депозит, то отображается сумма депозита автомобиля.


Дополнительные услуги


Список услуг, включенных в договор аренды. Отображается только для платного вида аренды. Состав полей таблицы дополнительных услуг соответствует составу документов по реализации услуг. Дополнительные услуги включаются в счета на оплату услуг аренды а также в акты об оказании услуг. Присутствует возможность подбора услуг из справочника «Номенклатура» и быстрый подбор услуг из вида аренды.


Тарификация


Таблица предназначена для визуализации расчета стоимости аренды и содержит информацию по стоимости периодов аренды. Отображается только для платного вида аренды. Состав отображаемых периодов зависит от настроек вида аренды.


Печатные формы


Для документа предусмотрены печатные формы:


Заявка на аренду — печать основной информации документа по автомобилям с указанием цен, о параметрах подбора, данных клиента. Предназначена для передачи клиенту в виде коммерческого предложения либо для внутренних целей, например для подготовки автомобиля к передаче клиенту.
