Документ "Тест-драйв"

Назначение

Используется для отражения в системе информации о проведение тест-драйвов автомобилей. Документ формирует записи в регистре "График работы ресурсов" о загруженности автомобилей и сотрудников.

На форме документа в разделе "Результаты анкетирования" присутствуют гиперссылки для быстрого перехода к предварительному и итоговому анкетному опросу. Ввод нового предварительного и итогового опроса доступен для автомобилей, для которых в регистре «Настройки тест-драйва» указаны соответствующие анкеты. Если опросы уже введены, гиперссылки позволяют открыть формы соответствующих документов.

 

Относится к подсистеме Взаимодействие раздела Заказы автосалона.  

Реквизиты

  • Дата начала. Реквизит является обязательным для заполнения. Дата начала тест-драйва.
  • Дата окончания. Реквизит является обязательным для заполнения. Дата окончания тест-драйва.
  • Состояние. Реквизит является обязательным для заполнения. Ссылается на перечисление «СостоянияСобытий». Текущее состояние выполнения тест-драйва.
  • Маршрут. Реквизит является обязательным для заполнения. Ссылается на справочник «Маршруты тест-драйвов». Маршрут тест-драйва.
  • Пробег на начало. Пробег на одометре автомобиля на момент начала тест-драйва.
  • Пробег на конец. Пробег на одометре автомобиля на момент окончания тест-драйва.
  • Результат тест-драйва. Ссылается на справочник «Результаты контакта». Результат проведения тест-драйва.

Формы

  • Форма ввода дат документа. Форма используется для ввода дат начала и окончания тест-драйва. Содержит раздельный ввод даты и времени соответствующих реквизитов.

Дополнительная информация

Документ может находится в различных состояниях:

1. В момент начала тест-драйва документ переводится в состояние "Выполняется". Перевод в состояние "Выполняется" допустим только после заполнения предварительной анкеты, если такая задана для данной модели автомобиля.

2. После окончания тест-драйва документ переводится в состояние "Завершено". Перевод в состояние "Завершено" допустим только после заполнения итоговой анкеты, если такая задана для данной модели автомобиля.

3. Документ переводится в состояние "Отменено", если тест-драйв по каким-либо причинам был отменен. Если клиент приехал не вовремя (раньше или позже), то текущий документ переводится в состояние "Отменено" и вводится новый документ.
