Документ "Корректировка статусов кодов маркировки"

Назначение

Документ «Корректировка статусов кодов маркировки» предназначен для актуализации состояний кодов маркировки товаров с состояниями из государственной информационной системой мониторинга за оборотом товаров (ГИС МТ).

Относится к подсистеме Маркировка товаров раздела Снабжение.

Реквизиты

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ, а так же собственник данного товара.  
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ. 
  • Документ основание. Ссылается на документ «Ввод остатков товаров». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов. 
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 

Табличная часть «Товары»

  • Номенклатура. Реквизит обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Характеристика. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п. 
  • Код маркировки. Реквизит является уникальным и обязательным для заполнения. Тип значения: «Строка». Код маркировки товара, которому изменяется состояние.
  • Статус кода маркировки. Реквизит обязательным для заполнения. Ссылается на перечисление «Состояния кодов маркировки». Текущее состояние кода маркировки в ГИС МТ.

В подменю Заполнение командной панели табличной части документа добавлена дополнительная команда Заполнить статусы кодов маркировки. Заполняет выбранным статусом строки табличной части.

Ввод на основании

Документ Корректировка статусов кодов маркировки может вводиться на основании следующих документов:

  • Ввод остатков товаров.
