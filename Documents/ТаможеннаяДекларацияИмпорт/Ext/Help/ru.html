Документ "Таможенная декларация (импорт)"

Назначение

Документ Таможенная декларация (импорт) предназначен для регистрации таможенных сборов, пошлин и штрафов. Таможенная декларация на импорт может быть оформлена по одному или нескольким документам поставки. Предусмотрена также возможность оформления нескольких таможенных деклараций по одному документу поставки. Для корректного поведения документа необходимо вводить документы поступления без учета НДС. При проведении документ формирует хозяйственные операции: 

  • Таможенная декларация (импорт). Хозяйственная операция начисляет кредиторскую задолженность компании перед контрагентом на регистре Взаиморасчеты компании, приходует товары на склад предприятия, с тем признаком партии, с которым товар был оприходован, по регистрам Остатки товаров компании, Партии товаров компании и ГТД партий товаров компании.
  • Таможенная декларация (импорт автомобилей). Хозяйственная операция аналогично операции Таможенная декларация (импорт) начисляет кредиторскую задолженность компании перед контрагентом на регистре Взаиморасчеты компании, приходует автомобили на склад предприятия, с тем признаком партии, с которым они были оприходованы, по регистру Остатки автомобилей. 

Реквизиты

  • Таможня/Брокер. Реквизит является обязательным для заполнения. Ссылается на справочник Контрагенты и контакты. Контрагент, оказывающий услуги.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник Договоры взаиморасчетов. Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Таможенный сбор. Сумма таможенного сбора.
  • Таможенный штраф. Сумма таможенного штрафа. 
  • Статья. Ссылается на справочник Статьи доходов и расходов. Статья доходов и расходов для штрафа и сбора.
  • Код таможни. Код таможни, на которую поступают товары.
  • Номер декларации. Номер таможенной декларации.
  • Склад. Склад компании, на который поступают товары.
  • Способ зачета авансов. Если установлено в значение "Автоматически", то при проведении документа все незакрытые сделки будут автоматически закрываться (в порядке ФИФО). В значении "Не зачитывать" - незакрытые сделки не будут погашены. Для договоров с признаком учета взаиморасчетов "По расчетным документам" доступен вариант "По документу" с возможностью выбора авансовых документов для зачета оплаты.

Табличная часть «Товары»

Верхняя табличная часть

  • Раздел. Реквизит является уникальным и обязательным для заполнения. Номер раздела декларации, в который группируются товары декларации.
  • Таможенная стоимость. Декларированная (таможенная) стоимость номенклатурной позиции.
  • Ставка пошлины. Ставка таможенной пошлины, в % от таможенной стоимости облагаемых товаров.
  • Сумма пошлины. Сумма таможенной пошлины облагаемых товаров.
  • Страна. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Страны мира. Страна для автоподстановки в товары раздела, соответствует происхождения, указанной в листе таможенной декларации.

Нижняя табличная часть

  • Раздел. Реквизит является уникальным и обязательным для заполнения. Номер раздела документа.
  • Номенклатура. Ссылается на справочник Номенклатура. Номенклатура (услуги).

  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Таможенная стоимость. Декларированная (таможенная) стоимость номенклатурной позиции.
  • Сумма пошлины. Размер пошлины номенклатурной позиции.
  • НДС. Сумма НДС по номенклатурной позиции.
  • Партия. Ссылается на документ Поступление товаров. Партия номенклатурной позиции.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник Склады  компании. Склад компании, на который поступают товары. 
  • Страна. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Страны мира. Страна происхождения номенклатурной позиции.
  • Номер декларации для с/ф. Реквизит является уникальным и обязательным для заполнения. Номер ГТД / РНПТ для номенклатурной позиции.

Табличная часть «Автомобили»

  • Номер раздела. Реквизит является уникальным и обязательным для заполнения. Тип значения: «Число». Номер раздела документа.

  • Страна происхождения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Страны мира. Страна происхождения номенклатурной позиции..
  • Номер для СФ. Реквизит является уникальным и обязательным для заполнения. Тип значения: «Строка».  Номер ГТД / РНПТ для номенклатурной позиции.
  • Таможенная стоимость. Тип значения: «Число». Стоимость номенклатурной позиции.

Табличная часть «Зачет авансов»

Вкладка Зачет авансов содержит способ зачета оплаты/авансов, а так же список подобранных вручную документов оплаты и суммы оплаты для погашения задолженности. Так же в список можно добавить документы с помощью кнопки Подбор авансов. Данный список документов оплат доступен только при выборе способа зачета оплаты/авансов "По документу".
  • Документ аванса. Реквизит является уникальным и обязательным для заполнения. Документ, на основании которого будет погашена задолженность.
  • Сумма. Реквизит обязательным для заполнения. Сумма для зачета по документу аванса.

Ввод на основании

Вводится на основании документа Поступление товаров, Поступление автомобилей.

Документы, вводимые на основании:

  • Перемещение товаров,
  • Перемещение автомобилей,
  • Поступление дополнительных расходов».
