Документ "Переразмещение автомобилей"

Назначение

Документ Переразмещение автомобилей предназначен для автоматизации процесса перемещения автомобилей внутри ячеистого склада. 

При проведении документ формирует хозяйственные операции:

  • Переразмещение автомобилей. Хозяйственная операция списывает товары из одной ячейки (Старая ячейка) склада по регистру «Остатки автомобилей на ордерном складе» и приходует в другую (Новая ячейка).

Параметры

Длина номера: 11.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: Переразмещение автомобилей, «Приходный складской ордер на автомобили», «Заказ-наряд».

Объекты, вводимые на основании: Переразмещение автомобилей, «Расходный складской ордер на автомобили», «Событие».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, внутри которого происходит перемещение автомобилей.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Юридическое лицо, от которого выписан документ.
  •  Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, создавший документ.
  • Документ основание. Ссылается на документы Переразмещение автомобилей, «Приходный складской ордер на автомобили», «Заказ-наряд». Документ, на основании которого введен данный.
  • Операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.  

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль, переразмещение которого производится.
  • Ячейка старая. Реквизит является обязательным для заполнения. Ссылается на справочник «Ячейки хранения». Старое место хранения ячеистого склада.
  • Ячейка новая. Реквизит является обязательным для заполнения. Ссылается на справочник «Ячейки хранения». Новое место хранения ячеистого склада.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Переразмещение автомобилей. Простая печатная форма документа.

Способ вызова

Вызывается из меню "Склад автосалона --> Ордерный склад --> Переразмещение автомобилей".

Движения

  • «Остатки автомобилей на ордерном складе».
