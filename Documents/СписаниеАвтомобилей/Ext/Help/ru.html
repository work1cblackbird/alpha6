Документ "Списание автомобилей"

Назначение

Документ Списание автомобилей предназначен для отражения в учете факта списания автомобилей с указанием склада, по которому происходит списание, по выбранной статье.

Относится к подсистеме Складской учет раздела Склад автосалона.

При проведении документ формирует хозяйственную операцию:

  • Списание автомобилей. Хозяйственная операция списывает автомобили по статье расходов, указанной в реквизите Статья списания.
  • Списание автомобилей отданных на комиссию. Хозяйственная операция аналогична операции Списание автомобилей с той разницей, что уменьшает партии товаров, отданных на реализацию, и за которые комиссионер должен отчитаться. В табличной части документа с данной хозяйственной операцией будет отображаться дополнительная графа Документ передачи — документ, по которому автомобиль был передан на комиссию.

Реквизиты

  • Контрагент. Ссылается на справочник Контрагенты и контакты. Комиссионер, которому были переданы автомобили на комиссию.
  • Договор. Ссылается на справочник Договоры взаиморасчетов. Договор, по которому ведутся взаиморасчеты с комиссионером.
  • Подразделение. Реквизит является обязательным для заполнения при списании автомобилей со склада. Ссылается на справочник Подразделения компании. Подразделение компании, на которое списываются затраты. 
  • Статья списания. Реквизит является обязательным для заполнения. Ссылается на справочник Статьи доходов и расходов. Статья списания товарно-материальных ценностей.

 

По кнопке Заполнение табличная часть документа может быть заполнена складскими остатками, а также при списании автомобилей, отданных на комиссию, — по договору комиссии.

Ввод на основании

На основании документа Списание автомобилей могут быть введены следующие документы:

Расходный складской ордер на автомобили,

Корректировка движений.

Печатные формы

  • Акт списания автомобилей. Печатная форма, содержащая список автомобилей списанных со склада, с указанием ответственных лиц.
  • ТОРГ-16. Унифицированная печатная форма №Торг-16, утвержденная постановлением Госкомстата России.
