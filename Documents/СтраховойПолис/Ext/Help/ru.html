Документ «Страховой полис»

Назначение

Используется для оформления страхового полиса. Вводится на основании рабочего листа отдела страхования.

Параметры

Длина номера: 10.

Периодичность номера: Непериодический.

Тип номера: Строка.

Вводится на основании: «Рабочий лист отдела страхования».

Объекты, вводимые на основании: отсутствует.

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник "Пользователи". Автор документа - пользователь, породивший документ.
  • Валюта документа. Ссылается на справочник "Валюты". Валюта документа.
  • Вид страхования. Ссылается на перечисление "Виды страхования". Вид страхования.
  • Дата выдачи. Реквизит является обязательным для заполнения. Тип значения: Дата. Дата выдачи полиса. 
  • Дата начала страхования. Тип значения: Дата. Дата начала страхования.
  • Дата окончания страхования. Тип значения: Дата. Дата окончания страхования.
  • Дата начала использования ТС. Тип значения: Дата. Дата начала использования транспортного средства.
  • Дата окончания использования ТС. Тип значения: Дата. Дата окончания использования транспортного средства.
  • Документ основание. Документ, на основании которого был введен рабочий лист отдела страхования.
  • Комментарий. Тип значения: Строка. Произвольный комментарий к документу.
  • Курс валюты. Тип значения: Число . Курс управленческой валюты.
  • Курс документа. Тип значения: Число. Курс валюты документа.
  • Номер квитанции. Реквизит является обязательным для заполнения. Тип значения: Строка. Номер квитанции.
  • Номер полиса. Реквизит является обязательным для заполнения. Тип значения: Строка. Номер полиса.
  • Объект страхования. Ссылается на справочник "Автомобили". Объект страхования.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник "Организации". Идентификатор юридического лица, от которого выписан документ.
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник "Подразделения компании". Подразделение компании, в котором был выписан документ.
  • Программа страхования. Реквизит является обязательным для заполнения. Ссылается на справочник "Программы страхования".  Программа страхования.
  • ПТС. Ссылается на справочник "Подтверждающие документы". Тех. паспорт на транспортное средство.
  • Серия квитанции. Тип значения: Строка. Серия квитанции полиса.
  • Серия полиса. Тип значения: Строка. Серия страхового полиса.
  • Собственник ТС. Ссылается на справочник "Контрагенты". Собственник транспортного средства.
  • Срок страхования. Тип значения: Число. Срок страхования.
  • Страхователь. Реквизит является обязательным для заполнения. Ссылается на справочник "Контрагенты". Клиент.
  • Страховщик. Реквизит является обязательным для заполнения. Ссылается на справочник "Контрагенты". Страховщик.
  • Сумма премии. Тип значения: Число. Сумма премии.
  • Транспортное средство. Ссылается на справочник "Модели". Транспортное средство.
  • Операция. Ссылается на справочник "Хоз. операции". Выполняемая документом операция.

Табличная часть «Лица, допущенные к управлению ТС»

  • Лицо допущенное.Ссылается на справочник "Контрагенты". Лицо, допущенное к управлению транспортным средством
  • Водительское удостоверение.Ссылается на справочник "Подтверждающие документы". Серия и номер водительского удостоверения.
  • Год начала водительского стажа.Тип значения: Дата. Год, с которого начался водительский стаж допущенного к управлению ТС лица.
  • ВодительскийСтаж.Тип значения: Число. Водительский стаж (лет).

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Взаимоотношения с клиентами --> Страховой полис".
