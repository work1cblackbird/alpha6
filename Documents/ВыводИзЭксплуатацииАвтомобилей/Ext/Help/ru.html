Документ "Вывод из эксплуатации автомобилей"

Назначение

Документ предназначен для отражения в учете факта вывода нетоварных активов подразделением компании.

При проведении документ формирует хозяйственные операции:

  • Вывод из эксплуатации автомобилей.  Хозяйственная операция списывает с баланса все позиции нетоварных активов, хранимые на регистре «Прочие активы в эксплуатации, отражает остаточную стоимость списанных активов на регистре «Доходы и расходы». Разность суммы реализации и остаточной стоимости активов записывается по соответствующей статье в регистр «Доходы и расходы». Приходует сумму и себестоимость проданных товаров на регистре «Остатки автомобилей» и «Комплектация автомобилей».

Параметры

Длина номера: 10.

Тип номера: Строка.

Периодичность номера: Год.

Вводится на основании: «Ввод в эксплуатацию автомобилей», «Перемещение активов».

Реквизиты

  • Подразделение отгрузки. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение, с которого списываются активы.
  • Склад компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады компании». Склад, на который производится оприходование автомобиля.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Менеджер. Ссылается на справочник «Сотрудники». Менеджер - лицо, заключившее сделку.  
  • Документ основание. Ссылается на документы «Ввод в эксплуатацию автомобилей», «Перемещение активов». Документ, на основании которого введен данный.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.

Табличная часть «Активы»

  • Актив. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Прочие активы». Нетоварный актив.
  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль.
  • Инвентарный номер. Тип значения: «Строка». Инвентарный номер актива.
  • Серийный номер актива. Тип значения: «Строка». Серийный номер актива.
  • Балансовая стоимость. Тип значения: «Число». Балансовая стоимость актива.
  • Начисленная амортизация. Тип значения: «Число». Начисленная стоимость актива.
  • Себестоимость. Тип значения: «Число». Себестоимость оприходованного актива.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Вывод из эксплуатации автомобилей. Простая печатная форма, содержащая список выводимых нетоварных активов.

Способ вызова

Вызывается из меню "Склад автосалона --> Производство --> Вывод из эксплуатации автомобилей" и "Аренда автомобилей --> Управление автомобилями аренды --> Вывод из эксплуатации автомобилей".

Движения

  • Остаточный регистр «Доходы и расходы».
  • Оборотный регистр «Остатки автомобилей».
  • Оборотный регистр «Комплектация автомобилей».
  • Остаточный регистр «Прочие активы в эксплуатации».

Последовательности, изменяемые документом: "Автомобили", "Комплектация автомобилей".
