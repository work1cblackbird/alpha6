Документ «Рабочий лист кредитного отдела»

Назначение

Используется для фиксирования параметров кредита для потенциального покупателя автомобиля.

Параметры

Длина номера: 10.

Периодичность номера: Непериодический.

Тип номера: Строка.

Вводится на основании: «Событие», «Рабочий лист».

Объекты, вводимые на основании: «Событие».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник "Пользователи". Автор документа - пользователь, породивший документ.
  • Банк. Реквизит является обязательным для заполнения. Ссылается на справочник "Банки". Банк, через который оформляется кредит. 
  • Валюта заявленного дохода. Реквизит является обязательным для заполнения. Ссылается на справочник "Валюты". Валюта заявленного дохода.
  • Валюта документа. Реквизит является обязательным для заполнения. Ссылается на справочник "Валюты". Валюта документа.
  • Вид оплаты. Ссылается на перечисление "Виды оплаты". Вид оплаты документа.
  • Дата отправки анкеты. Тип значения: Дата.  Дата отправления анкеты клиента в банк.
  • Дата получения анкеты банком. Тип значения: Дата. Дата получения анкеты клиента банком.
  • Дата принятия решения. Тип значения: Дата. Дата принятия решения о кредитовании.
  • Дата сделки. Тип значения: Дата. Дата сделки.
  • Документ основание. Документ, на основании которого был введен рабочий лист кредитного отдела.
  • Заявленный доход. Реквизит является обязательным для заполнения. Тип значения: Число. Заявленный доход клиента.
  • Клиент. Ссылается на справочник "Контрагенты". Клиент.
  • Комментарий. Тип значения: Строка. Произвольный комментарий к документу.
  • Контактный телефонКлиента. Реквизит является обязательным для заполнения. Тип значения: Строка. Контактный телефон клиента.
  • Кредитная программа. Ссылается на справочник "Кредитные программы". Кредитная программа.
  • Курс валюты. Тип значения: Число. Курс валюты управленческого учета документа.
  • Курс документа. Тип значения: Число. Курс валюты документа.
  • Менеджер. Реквизит является обязательным для заполнения. Ссылается на справочник "Сотрудники". Менеджер кредитного отдела.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник "Организации". Идентификатор юридического лица, от которого выписан документ.
  • Первоначальный взнос. Реквизит является обязательным для заполнения. Тип значения: Число. Первоначальный взнос.
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник "Подразделения компании". Подразделение компании, в котором был выписан документ.
  • Причина отказа. Ссылается на справочник "Возражения и причины отказов".. Причина отказа клиенту.
  • Процентная ставка. Реквизит является обязательным для заполнения. Тип значения: Число. Процентная ставка кредита.
  • Срок кредита. Реквизит является обязательным для заполнения. Тип значения: Число. Срок кредита.
  • Статус. Реквизит является обязательным для заполнения. Ссылается на справочник "Статусы рабочего листа". Статус рабочего листа кредитного отдела.
  • Стоимость автомобиля. Реквизит является обязательным для заполнения. Тип значения: Число. Стоимость автомобиля.
  • Сумма кредита. Реквизит является обязательным для заполнения. Тип значения: Число. Сумма кредита.
  • Требование к подтверждению дохода. Реквизит является обязательным для заполнения. Ссылается на перечисление "Требования к подтверждению дохода". Требование к подтверждению дохода.
  • Операция. Ссылается на справочник "Хоз. операции". Выполняемая документом операция.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Взаимоотношения с клиентами --> Рабочий лист кредитного отдела".

 

Движения

  • Регистр сведений «Журнал состояний».
