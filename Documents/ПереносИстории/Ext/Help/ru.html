Документ "Перенос истории"

Назначение

Документ Исходный документ предназначен для хранения исторических данных из предыдущей учетной системы.

Относится к подсистеме Ввод начальных остатков раздела Предприятие.


При проведении документ формирует хозяйственные операции: 

  • История документа. Хозяйственная операция для указания документа из предыдущей учетной системы.
  • История продаж.  Хозяйственная операция для фиксирования истории продаж.

Реквизиты

  • Вид документа. Реквизит является обязательным для заполнения. Ссылается на справочник Хозяйственные операции. Вид исходного документа, которому соответствует данная операция.
  • Контрагент. Ссылается на справочник Контрагенты и контакты. Является обязательным для заполнения для некоторых видов документов. Для заказ-наряда заполняется заказчиком.
  • Автомобиль. Ссылается на справочник Автомобили. Является обязательным для заполнения для некоторых видов документов. Автомобиль по данным документа из предыдущей базы. 
  • Сумма авторабот. Тип значения: Число. Общая сумма по автоработам документа. Заполняется для вида документа Заказ-наряд.
  • Сумма запчастей. Тип значения: Число. Общая сумма по запчастям документа. Заполняется для вида документа Заказ-наряд.
  • Описание. Тип значения: Строка. Дополнительное описание документа в произвольной форме.
  • Краткое описание. Тип значения: Строка. Укороченное описание документа. Формируется на основании описания документа для вывода в отчетах.

Табличная часть «Продажи»

Табличная часть Продажи содержит информацию об истории продаж товаров из предыдущей учетной системы.
