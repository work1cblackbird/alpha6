Документ "Заявка на компенсацию по маркетинговой программе"

Назначение

Документ Заявка на компенсацию по маркетинговой программе предназначен для отображения взаиморасчетов с дистрибьютором по маркетинговым программам. Документ относится к подсистеме Маркетинг автомобилей раздела Продажа автомобилей.

Реквизиты

  • Контрагент. Ссылается на справочник Контрагенты и контакты. Дистрибьютор.
  • Договор. Ссылается на справочник Договоры взаиморасчетов. Договор, по которому ведутся взаиморасчеты.
  • Состояние. Ссылается на перечисление Статусы компенсации по маркетинговым программам. Отражает состояние заявки.
  • Основание. Ссылается на документы Заказ на автомобиль, Заказ поставщику на автомобиль, Заявка на компенсацию по маркетинговой программе. Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу-основанию. С помощью этого реквизита затем строится дерево подчиненных документов (по кнопке ).

Табличная часть «Состав компенсации»

Основная табличная часть документа.

  • Источник. Документ в котором была получена выгода по маркетинговой программе.
  • Автомобиль. Ссылается на справочник «Автомобили». Автомобиль, на который оформляется скидка.
  • Маркетинговая программа. Ссылается на справочник «Скидки и маркетинговые программы». Маркетинговая программа.
  • Сумма. Тип значения: «Число». Сумма компенсации по маркетинговой программе.

 

Ввод на основании

Документ Заявка на компенсацию по маркетинговой программе вводится на основании документов:

  • Заказ на автомобиль,
  • Заказ поставщику на автомобиль
  • Заявка на компенсацию по маркетинговой программе.


Документы, вводимые на основании:

  • Корректировка компенсации по маркетинговой программе,
  • Заявка на компенсацию по маркетинговой программе,
  • Событие.
