Документ "Корректировка опций автомобиля"

Назначение

С помощью этого документа можно провести корректировку опций автомобиля. Документ всегда вводится на основании документов  «Заказ на автомобиль», «Заказ поставщику на автомобиль», «Корректировка опций автомобиля» и справочник «Автомобили». Спецификация документа заполняет документ актуальными опциями автомобиля из регистра сведений «Опции автомобилей». При проведении документ формирует хозяйственную операцию:

  • Корректировка опций автомобиля. Хозяйственная операция формирует записи в регистр «Опции автомобилей» актуальными опциями для автомобиля.

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании:  «Заказ на автомобиль», «Заказ поставщику на автомобиль», «Корректировка опций автомобиля», справочник «Автомобили».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.  
  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль, для которого производится корректировка опций. 

Табличная часть «Опции»

  • Опция. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Опции автомобилей». Опции, которые будут установлены на автомобиль поставщиком.
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество устанавливаемых опций.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Корректировка опций автомобиля. Простая печатная форма, содержит список опций автомобиля, корректировка которых происходит.
