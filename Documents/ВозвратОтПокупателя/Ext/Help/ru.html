Документ "Возврат от покупателя"

Назначение

Документ Возврат товаров от покупателя предназначен для отражения в оперативном учете операции по возврату от покупателей или комиссионеров ранее отгруженных им товаров. В табличной части документа не могут фигурировать услуги, работы и т.п. При проведении документ пытается сторнировать отгрузку товаров по тем партиям, по которым они были получены, либо, если это невозможно, например, при отсутствии документа поступления, образует новую партию собственного товара.

Относится к подсистеме Отгрузка раздела Продажи. 

При проведении документ формирует хозяйственные операции:

  • Возврат товаров от покупателя. Хозяйственная операция списывает дебиторскую задолженность покупателя по другим сделкам, если таковые были, и начисляет кредиторскую задолженность компании перед покупателем или комиссионером на регистре Взаиморасчеты компании, сторнирует материальные активы, на оптовых складах предприятия с собственным или комиссионным признаком партии, хранимые на регистрах Остатки товаров компании и Партии товаров компании, сторнирует товарную задолженность перед комитентом по принятым на комиссию и проданным товарам на регистре Реализованные товары, сторнирует сумму и себестоимость проданных товаров на регистре Продажи.
  • Возврат товаров, переданных на комиссию. Хозяйственная операция аналогична Возврату товаров от покупателя за тем исключением, что начисление кредиторской задолженности не происходит, а сторнируется товарная задолженность комиссионера перед нашей компанией с регистра Партии товаров отданные.

Реквизиты

  • Склад компании. Реквизит является обязательным для заполнения. Ссылается на справочник Склады компании. Склад компании, на который будет оприходован возвращенный товар.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник Контрагенты и контакты. Покупатель или комиссионер, возвращающий товар.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник Договоры взаиморасчетов. Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Документ отгрузки. Документ, согласно которому были отгружены возвращаемые товары. Если документ не указан, то для каждого товара в таблице требуется задать документ отгрузки.
  • Ввести счет- фактуру. Если гиперссылка содержит текст «Ввести счет-фактуру», то при переходе по гиперссылке на основании данного документа будет создан счет-фактура выданный. Если этот документ был создан ранее, то гиперссылка позволяет открыть соответствующий документ.
  • По договору долг контрагента составляет. Гиперссылка отражает текущее состояние расчетов с контрагентом по договору. Производится переход к отчету Взаиморасчеты с контрагентами, сформированному для указанного контрагента.

Табличная часть «Товары»

Дополнительные графы:

  • Документ продажи. Документ, согласно которому был продан товар.
  • Процент скидки. Процент скидки на сумму с налогами. Это значение может быть заполнено автоматически и/или введено вручную. Это зависит от права пользователя на назначение скидки.
  • Всего. Сумма по строке в валюте документа с учетом скидки.
  • Партия. Партия товара, в которую возвращается данный товар.
  • Себестоимость. Себестоимость возвращенного товара.

Кнопка Заполнение, находящаяся над табличной частью документа, позволяет выполнить автозаполнение табличной части по следующим схемам:

  • Очистить табличную часть. Из табличной части будут удалены все позиции.
  • Установить % наценки для всей таблицы — позволяет задать единый процент наценки (вариант возможен, если выбран розничный склад).
  • Заполнить по документу отгрузки. Табличная часть перезаполнится по документу-основанию, по которому осуществлялась продажа.
  • Загрузить из ТСД. Вызов обработки загрузки данных с терминала сбора данных.
  • Загрузить из файла. Вызывает обработку Загрузка данных из файла.

Ввод на основании

Документ Возврат товаров от покупателя вводится на основании следующих документов:

  • Реализация товаров,
  • Закрытие смены,
  • Инвентаризация товаров,
  • Ввод остатков товаров.

Документы, вводимые на основании:

  • Возврат поставщику,
  • Перемещение товаров,
  • Расходный кассовый ордер,
  • Корректировка движений,
  • Счет-фактура полученный,
  • Приходный складской ордер,
  • Платежное поручение.

Печатные формы

  • Возврат от покупателя. Простая печатная форма, содержит список возвращаемых товаров от покупателя или комиссионера.

Журналы, отображающие документ

  • Операции с покупателями,
  • Складские документы.
