Документ "Изменение договора аренды"

Назначение

Документ используется для изменения ряда параметров по договору аренды, таких как сроки аренды, списка водителей, стоимости аренды, например, скидок по договору, списка дополнительных услуг, дополнительных условий (места выдачи и приемки автомобилей). Для бесплатного вида аренды доступно только изменение сроков, списка водителей и дополнительных условий договора. Изменения в договоре начинают действовать с указанной на закладке «Основное» даты.


Относится к подсистеме «Аренда автомобилей»


Вид аренды - значение классификация аренды из соответствующего справочника. Поле заполняется автоматически и недоступно для редактирования. От вида аренды зависит внешний вид формы и состав доступных полей.


Период аренды — даты начала и окончания аренды. Доступно изменение даты и времени окончания аренды при установке признака изменения сроков аренды. Для удобства редактирования данных добавлено изменение времени в отдельных полях.


Договор аренды — договор для которого вводится изменение.


Основное


Изменения — признаки корректировки разделов аренды. При изменении признака запрашивается подтверждение действия пользователя. При снятии признака данные возвращаются к данным предыдущему соглашению.


Виды периодов — виды интервалов аренды, которые используются при продлении и сокращении аренды. При проведении документ производится проверка могут ли указанные виды интервалов вытеснить уже используемые за период виды интервалов, в случае сообщения о невозможности вытеснения необходимо выбрать из списка подходящие виды интервалов или добавить их.


Подписант от организации — данные ответственного лица организации, использующиеся при печати договора и ряда приложений к договору.


Подписант от контрагента — данные ответственного лица клиента, использующиеся при печати договора и ряда приложений к договору. В случае платного вида аренды подписант относится к контрагенту, в случае бесплатного — к заказчику.


Действующее соглашение — группа полей заполняется в том случае, если по договору введены изменения условий. Предназначены для быстрого перехода к последнему введенному изменению. Также присутствует кнопка ввода нового изменения договора аренды.


Водители


Список допущенных к управлению автомобилями лиц. Печатная форма доверенности формируется для указанных в списке водителей. Водители выбираются из справочника «Контрагенты». Также для каждого водителя возможно указания паспорта и водительского удостоверения. В случае, когда заказчиком является физическое лицо, в командной панели доступна команда добавления заказчика в список водителей. Водителей можно выбирать из списка контактных лиц заказчика, для этого добавлена кнопка в командной панели таблицы водителей. Редактирование списка доступно при установке признака изменения водителей.


Автомобили


Список передаваемых по договору аренды автомобилей. Для платных видов аренды выводятся стоимость аренды, сумма НДС, процент и сумма скидки. Если по виду аренды предусмотрен депозит, то отображается сумма депозита автомобиля, если по виду аренды контролируется приемка и выдача, то в таблице отображается текущий статус автомобиля, а в командной панели таблицы доступны документы «Выдать автомобиль» и «Принять автомобиль». Доступен ввод автомобилей из соответствующего справочника и подбор с использованием планировщика аренды. Редактирование списка доступно при установке признака изменения стоимости.


Дополнительные услуги


Список услуг, включенных в договор аренды. Отображается только для платного вида аренды. Состав полей таблицы дополнительных услуг соответствует составу документов по реализации услуг. Дополнительные услуги включаются в счета на оплату услуг аренды а также в акты об оказании услуг. Присутствует возможность подбора услуг из справочника «Номенклатура» и быстрый подбор услуг из вида аренды. Редактирование списка доступно при установке признака изменения дополнительных услуг.


Тарификация


Таблица предназначена для визуализации расчета стоимости аренды и содержит информацию по стоимости периодов аренды. Отображается только для платного вида аренды. Состав отображаемых периодов зависит от настроек вида аренды.


Дополнительные условия


Редактирование данных доступно при установке признака изменения дополнительных условий.


Место выдачи автомобилей — блок параметров, определяющий адрес по которому будет производиться выдача автомобилей. Доступен выбор одного из трех вариантов — фактический адрес указанного подразделения компании, фактический адрес указанного контрагента, произвольный адрес. По умолчанию устанавливается адрес подразделения документа.


Место приемки автомобилей — блок параметров, определяющий адрес по которому будет производиться приемка автомобилей. Доступен выбор одного из трех вариантов — фактический адрес указанного подразделения компании, фактический адрес указанного контрагента, произвольный адрес. По умолчанию устанавливается адрес подразделения документа.


Условия использования — произвольный текст для регистрации особых договоренностей с клиентом, который выводится при печати договора в соответствующем разделе.


Печатные формы


Для документа предусмотрены печатные формы:


Доверенность — доверенность на управление транспортным средством. Формируется одна доверенность на все автомобили с указанием всех водителей.
