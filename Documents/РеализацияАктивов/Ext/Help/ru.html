Документ "Реализация активов"

Назначение

Документ Реализация активов предназначен для отражения в учете факта реализации нетоварных активов подразделением компании. Относится к подсистеме Прочие активы раздела Предприятие.

При проведении документ формирует хозяйственные операции:

  • Реализация активов.  Хозяйственная операция списывает с баланса все позиции нетоварных активов, хранимые на регистре Прочие активы в эксплуатации, отражает остаточную стоимость списанных активов на регистре Доходы и расходы. Разность суммы реализации и остаточной стоимости активов записывается по соответствующей статье в регистр Доходы и расходы. Приходует сумму и себестоимость проданных товаров на регистре Продажи. Начисляет дебиторскую задолженность покупателя перед компанией на регистре Взаиморасчетов компании.

Табличная часть «Активы»

  • № инвентарный. Инвентарный номер актива.
  • № серийный. Серийный номер актива.
    •
  • Актив. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Прочие активы. Нетоварный актив.
  • Количество. Количество единиц актива.
  • Сумма. Стоимость реализации актива.
  • Процент НДС. Ссылается на справочник Ставки НДС. Ставка НДС по позиции нетоварного актива.
  • НДС. Сумма НДС по позиции нетоварного актива.
  • Балансовая стоимость. Балансовая стоимость актива на дату документа.
  • Начисленная амортизация. Сумма начисленной амортизации на дату документа.
  • Остаточная стоимость. Остаточная стоимость актива на дату документа (балансовая стоимость за вычетом начисленной амортизации).
  • Сумма списания. Сумма списания балансовой стоимости актива.

Поддерживается автоматическая обработка табличной части документа. Для этого нужно нажать на кнопку Заполнение, находящуюся над табличной частью документа, и выбрать одну из возможных функций:

  • Очистить табличную часть. Из табличной части буду удалены все элементы.
  • Загрузить из файла. Вызов обработки Загрузка данных из файла.
  • Заполнить активами подразделения. В табличную часть будут занесены все нетоварные активы, числящиеся на балансе подразделения, от имени которого составляется документ.

Табличная часть «Зачет авансов»

Вкладка Зачет авансов содержит способ зачета оплаты/авансов, а так же список подобранных вручную документов оплаты и суммы оплаты для погашения задолженности. Так же в список можно добавить документы с помощью кнопки Подбор авансов. Данный список документов оплат доступен только при выборе способа зачета оплаты/авансов "По документу".
  • Документ аванса. Реквизит является уникальным и обязательным для заполнения. Документ, на основании которого будет погашена задолженность.
  • Сумма. Реквизит обязательным для заполнения. Сумма для зачета по документу аванса.

Ввод на основании

Вводится на основании документов:

  • Ввод в эксплуатацию,
  • Перемещение активов,
  • Ввод остатков прочих активов.

Объекты, вводимые на основании:

  • Приходный кассовый ордер,

  • Корректировка движений.

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Реализация активов. Простая печатная форма, содержащая список реализуемых нетоварных активов.
  • М-15 (Накладная на отпуск материалов на сторону). Унифицированная печатная форма № М-15, утвержденная постановлением Госкомстата России.
  • Универсальный передаточный документ.
