Документ "Амортизация"

Назначение

Документ Амортизация предназначен для ежемесячного расчета и отражения в учете начисления амортизации нетоварных активов. Амортизационные начисления по нетоварным активам начинаются с первого числа месяца, следующего за месяцем ввода в эксплуатацию, и начисляются до полного погашения стоимости актива, либо списания или реализации актива.

Документ относится к подсистеме Прочие активы раздела Предприятие.

При проведении документ формирует хозяйственные операции:

  • Амортизация. Хозяйственная операция для всех нетоварных активов в табличной части увеличивает накопленную сумму амортизации, хранимую на регистре Прочие активы в эксплуатации. 

Табличная часть «Активы»

  • № инвентарный. Инвентарный номер актива.
  • № серийный. Серийный номер актива.
  • Актив. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Прочие активы. Наименование актива. 
  • Тип эксплуатации. Реквизит является обязательным для заполнения. Ссылается на справочник Типы эксплуатации. Тип эксплуатации актива.
  • Балансовая стоимость. Балансовая стоимость актива.
  • Накопленная амортизация. Сумма накопленной амортизации по активу.
  • Выработка. Выработка актива (доступен только для оборудования) за текущий месяц. 
  • Единица. Единица выработки актива.
  • % амортизации. Процент амортизации. Рассчитывается автоматически в зависимости от типа эксплуатации.
  • Сумма амортизации. Реквизит является обязательным для заполнения. Рассчитывается по проценту.
  • Остаточная стоимость. Остаточная стоимость актива с учетом текущего начисления амортизации.
  • Статья расходов по амортизации. Ссылается на справочник Статьи доходов и расходов. Статья расходов по амортизации актива.

Ввод на основании

На основании документа Амортизация  можно ввести документ Корректировка движений.

Печатные формы

  • Амортизация. Простая печатная форма, содержит список амортизируемых нетоварных активов.
