Документ "Приходный складской ордер на автомобили"

Назначение

Поступление автомобилей на ордерный или ячеистый склад.

При проведении документ формирует хозяйственные операции:

  • Приходный складской ордер на автомобили. Хозяйственная операция приходует на склады/стоянки предприятия, имеющие вид "Ордерный" или "Ордерный и ячеистый", автомобили, хранимые в регистре «Остатки автомобилей на ордерном складе».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Переразмещение автомобилей».

Объекты, вводимые на основании: «Переразмещение автомобилей», «Расходный складской ордер на автомобили».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад/стоянка компании, на который поступают автомобили.
  • Владелец товара. Ссылается на справочники «Контрагенты и контакты». Контрагент, от которого мы получили автомобиль.
  • Документ основание. Ссылается на документы «Переразмещение автомобилей». Документ, на основании которого введен данный.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Юридическое лицо, от которого выписан документ.
  •  Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, создавший документ.
  • Операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Приходуемый автомобиль.
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество приходуемых автомобилей. Всегда = 1. 
  • Ячейка. Ссылается на справочник «Ячейки хранения». Место хранения ячеистого склада.  

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Приходный ордер.  Простая печатная форма, содержит список приходуемых на склад/стоянку автомобилей. Если склад  ячеистый, содержит колонку, отображающую ячейки, в которые помещаются приходуемые автомобили.
  • Форма М-4. Унифицированная печатная форма № М-4, утвержденная постановлением Госкомстата России.
  • Карта склада. Выводит графическое представление склада с отображением маршрута движения для размещения автомобилей по указанным ячейкам. Маршрут начинается в точке входа склада и заканчивается в точке выхода. Вывод карты склада возможен только для ячеистого склада.

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Приходный складской ордер на автомобили"

Журналы отображающие документ: «Складские ордера».

Движения

  • Остаточный регистр «Остатки автомобилей на ордерном складе».
