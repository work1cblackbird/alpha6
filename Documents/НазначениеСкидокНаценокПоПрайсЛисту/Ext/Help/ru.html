Документ "Назначение скидок и наценок по прайс-листу"

Назначение

Документ предназначен для назначения скидок/наценок на цены номенклатуры по прайс-листу контрагентов. Относится к подсистеме Скидки и наценки раздела Маркетинг.

При проведении документ формирует хозяйственные операции:

  • Установка наценок прайс-листа. Хозяйственная операция регистрирует назначение наценки в регистре сведений "Прайс-листы контрагентов. Наценки".
  • Установка скидок прайс-листа. Хозяйственная операция регистрирует назначение скидки в регистре сведений "Прайс-листы контрагентов. Скидки".

Реквизиты

  • Действует с. Реквизит является обязательным для заполнения. Дата начала действия скидки/наценки.
  • Контрагент.  Ссылается на справочник Контрагенты и контакты. Контрагент, прайс-листы которого используются для регистрации скидок/наценок.
  • Подразделение получатель. Реквизит является обязательным для заполнения. Ссылается на справочник Подразделения. Подразделение компании, для которого регистрируются скидки/наценки.
  • Базовый тип цен. Тип цен, относительно которого производится расчет.

Табличная часть «Прайс-листы»

  • ПрайсЛист. Реквизит является обязательным для заполнения. Ссылается на справочник Типы скидок и наценок. Определяет тип назначаемой скидки.
  • Скидка. Значение скидки.
  • Тег позиции. Тег позиции в прайс-листе.
  • Производитель. Производитель номенклатурной позиции прайс-листа.
  • Наценка. Значение наценки.
  • Округлять до. Величина округления рассчитываемых скидок/наценок.
  • АлгоритмРасчетаЦены. Расчет по указанному алгоритму.
