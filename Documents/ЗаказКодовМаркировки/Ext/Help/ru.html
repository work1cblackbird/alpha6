Документ "Заказ кодов маркировки"

Назначение

Документ «Заказ кодов маркировки» предназначен для формирования заказа кодов маркировки и отправки в государственную информационную систему мониторинга за оборотом товаров (ГИС МТ), а также фиксации полученных кодов и их печати. 

Для маркировки остатков товаров доступна команда отправки запроса на получение GTIN при упрощенном заполнении карточки. Данный способ заказа доступен для товарной группы "Шины и автопокрышки". Срок завершения маркировки остатков товаров можно узнать на сайте Честного знака.

ВАЖНО! Перед использованием документа необходимо убедится в наличии личного кабинета в ГИС МТ для данной организации, настройки работы со станцией управления заказами (СУЗ), баланс счета для получения кодов маркировки. Необходимо указать в настройках способ обмена с ГИС МТ через файловый обмен или API СУЗ и срок хранения кодов маркировки для печати.

Относится к подсистеме Маркировка товаров раздела Снабжение.

Реквизиты

  • Состояние. Реквизит является обязательным для заполнения. Ссылается на перечисление «Статусы документов маркировки». Отображает текущее состояние документа заказа.
  • Идентификатор документа. Тип значения: «Строка». Заполняется автоматически при отправке запроса кодов маркировки в ГИС МТ при использовании обмена через API СУЗ. 
  • Товарная группа. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы маркировки». Определение товарной группы маркируемого товара. Используется для формирования запроса кодов маркировки в ГИС МТ.
  • Способ выпуска в оборот. Реквизит является обязательным для заполнения. Ссылается на перечисление «Способы выпуска в оборот». Способ выпуска в оборот определяет способ выпуска продукции. Используется для формирования запроса кодов маркировки в ГИС МТ.
  • Идентификатор заказа. Тип значения: «Строка». Идентификатор производственного заказа. Используется для формирования запроса кодов маркировки в ГИС МТ.
  • Ответственный. Реквизит является обязательным для заполнения. Ссылается на справочник «Сотрудники». Сотрудник, который является ответственным за заказ. Используется для формирования запроса кодов маркировки в ГИС МТ.
  • Завезен в РФ после обязательной маркировки. Тип значения: «Булево». Признак того, что товар был завезен в РФ после установленного срока начала обязательной маркировки. Используется при заказе кодов маркировки остатков для товарной группы «Предметы одежды, белье постельное, столовое, туалетное и кухонное».
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ. На основании ИНН организации формируется запрос в ГИС МТ.  
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ. 
  • Документ основание. Ссылается на документы «Заказ поставщику», «Ввод в оборот кодов маркировки», «Перемаркировка».  Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов. 
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Идентификатор производства. Реквизит является обязательным для заполнения для товарных групп «Табачная продукция». Тип значения: «Строка». Глобальный номер места нахождения производства.
  • Наименование производства. Реквизит для заполнения для товарных групп «Табачная продукция». Тип значения: «Строка». Наименование производства товара.
  • Адрес производства. Реквизит для заполнения для товарных групп «Табачная продукция». Тип значения: «Строка». Адрес производства товара.
  • Страна производителя. Реквизит для заполнения для товарных групп «Табачная продукция». Ссылается на справочник «Страны мира». Страна производства товара.
  • Идентификатор производственной линии. Реквизит для заполнения для товарных групп «Табачная продукция». Тип значения: «Строка». Идентификатор производственной линии.
  • Код продукта (SKU). Реквизит для заполнения для товарных групп «Табачная продукция». Тип значения: «Строка». Код произведенного товара.
  • Описание продукта. Реквизит для заполнения для товарных групп «Табачная продукция». Тип значения: «Строка». Описание произведенного товара.
  • Номер производственного заказа. Реквизит для заполнения для товарных групп «Табачная продукция». Тип значения: «Строка». Номер производственного заказа.
  • Дата начала производства продукции. Реквизит для заполнения для товарных групп «Табачная продукция». Тип значения: «Дата». Ожидаемая дата начала производства продукции.
  • Тип

Табличная часть «Товары»

  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Характеристика. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п. 
  • GTIN. Реквизит является уникальным и обязательным для заполнения. Тип значения: «Строка». Код товара, используется при формировании кода маркировки.
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество кодов маркировки для заказа.
  • Код ТНВЭД. Реквизит является обязательным для заполнения при указании способа выпуска в оборот «Маркировка остатков». Ссылается на справочник «Классификатор ТН ВЭД». Код товара по классификатору ТН ВЭД. 

  • Модель. Реквизит является обязательным для заполнения при указании способа выпуска в оборот «Маркировка остатков» и товарной группы «Шины и покрышки пневматические резиновые новые». Тип значения: «Строка». Наименование модели для упрощенного ввода карточки товара при формировании документа «Описание остатков» в ГИС МТ.
  • Целевой пол. Реквизит является обязательным для заполнения при указании способа выпуска в оборот «Маркировка остатков» и товарной группы «Предметы одежды, белье постельное, столовое, туалетное и кухонное» с кодом ТН ВЭД вначале 42, 61, 62, 64. Ссылается на перечисление «Целевой пол». Описывает реквизит для упрощенного ввода карточки товара при формировании документа «Описание остатков» в ГИС МТ.
  • Возрастная категория. Реквизит является обязательным для заполнения при указании способа выпуска в оборот «Маркировка остатков» и товарной группы «Предметы одежды, белье постельное, столовое, туалетное и кухонное» с кодом ТН ВЭД вначале 63. Ссылается на перечисление «Возрастные категории». Описывает реквизит для упрощенного ввода карточки товара при формировании документа «Описание остатков» в ГИС МТ.
  • Способ выпуска товаров в оборот. Реквизит является обязательным для заполнения при указании способа выпуска в оборот «Маркировка остатков» и товарной группы «Предметы одежды, белье постельное, столовое, туалетное и кухонное». Ссылается на перечисление «Способы выпуска в оборот». Описывает реквизит для упрощенного ввода карточки товара при формировании документа «Описание остатков» в ГИС МТ.
  • Товарный знак. Тип значения: «Строка». Товарный знак товара для упрощенного ввода карточки товара при формировании документа «Описание остатков» в ГИС МТ.
  • Максимальная розничная цена. Реквизит является обязательным для заполнения для товарных групп «Табачная продукция». Тип значения: «Число». Максимальная розничная цена для данного товара.
Для способа выпуска в оборот «Маркировка остатков» добавлена кнопка Заказать GTIN. Она используется для формирования документа «Описание остатков» в ГИС МТ для генерации в системе GTIN товарам по упрощенному варианту по строкам в табличной части документа, где не заполнен GTIN. При установке настройки обмена по API ГИС МТ можно получить актуальные GTIN по введенным параметрам по кнопке подменю «Заполнить» по кнопке Заполнить GTIN. Иначе необходимо полученные GTIN заполнить в документ самостоятельно.
ВАЖНО! Если в сформированном документе «Описание остатков» будет описание товара с параметрами, по которым был ранее получен GTIN, то данный документ не загрузится в ГИС МТ корректно.

Кнопка Выгрузить заказ используется для формирования документа заказа в ГИС МТ. Признаком отправки документа в учетную систему является значение состояния документа «Отравлен».
Кнопка Загрузка кодов используется для загрузки эмитированных кодов маркировки для печати этикеток и фиксировании статусов кодов маркировки. Признаком завершения работы с заказом является значение состояния документа «Выполнен». 

Ввод на основании

Документ Заказ кодов маркировки может вводиться на основании следующих документов:

  • Заказ поставщику,
  • Ввод в оборот кодов маркировки,
  • Перемаркировка.
Документы, вводимые на основании:
  • Ввод в оборот кодов маркировки.

Печатные формы

  • Печать этикеток. Обработка, позволяющая получить печатную форму этикеток загруженных кодов маркировки.
