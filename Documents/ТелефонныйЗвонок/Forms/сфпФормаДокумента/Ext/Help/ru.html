Документ "Телефонный звонок"

Назначение

Документ служит для фиксации разговора при входящем и исходящем звонке через СофтФон. Создается только при использовании СофтФона или при загрузке истории звонков с АТС, посредством кнопки «Загрузить историю звонков» из списка Телефонных звонков. 

Из формы документа Телефонный звонок можно прослушать запись разговора, если такая настройка включена. Если у контакта есть несколько телефонных разговоров, то их количество указывается в документе Телефонный звонок и их можно прослушать из любого документа Телефонный звонок, данного контакта.

Если при входящем звонке контакт не определился, то непосредственно из созданного документа Телефонный звонок можно выбрать контакт из списков: Партнер, Контактное лицо, Пользователь, Физическое лицо.

В документе Телефонный звонок поля «Тема» и «Описание» заполняются автоматически из СофтФона. В поле «Тема» указываются входящий/исходящий звонок и время звонка. В поле «Описание» отображается длительность совершенного звонка.
