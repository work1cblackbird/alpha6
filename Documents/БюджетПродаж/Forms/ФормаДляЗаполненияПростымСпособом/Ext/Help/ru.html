Произвольная форма "Наивное прогнозирование"

Назначение

Произвольная форма документа «Бюджет продаж», предназначена для заполнения документа данными, используя метод наивного прогнозирования. Наивное прогнозирование заключается в получении простого арифметического среднего прогнозируемого ресурса: количество или сумма упр. от фактических или запланированных данных предыдущих периодов.

Способ вызова

Форма вызывается из меню командной панели табличной части документа «Бюджет продаж» : "Наивное прогнозирование" или из командной панели другой произвольной формы документа "Заполнение по документу основанию".

Принцип работы с формой

Для начала работы с формой, необходимо определиться - арифметические средние от каких данных необходимо получить.  Предусмотрены два типа данных для анализа: первое - Фактические данные продаж предыдущих периодов, которые используют оборотные значения регистра «Продажи», второе-  Спрогнозированные значения предыдущих периодов, которые используют оборотные значения регистра «Бюджет продаж». Выбор нужного типа осуществляется при помощи реквизита "Тип анализа".

Следующий важный параметр наивного прогнозирования - "Количество анализируемых периодов". Данный параметр фактически определяет количество значений, от которых будет получено среднее арифметическое. Периоды отсчитываются назад от даты планирования документа (не путать с датой документа!) «Бюджет продаж», из которого была вызвана данная форма. Вид периода (его размерность: месяц, квартал, год) определяется периодичностью выбранного в документе сценария планирования.  

Далее необходимо установить коэффициенты прогнозирования: Коэффициент роста продаж и коэффициент сезонности. Эти коэффициенты определяются экспертным путем на основании предположения о существенном росте\спаде продаж в планируемом периоде. Выражаются в натуральной величине (не в процентах!).

Показатель планирования - определяет ресурс, который будет запланирован: количество или сумма упр.., т.е. обороты какого ресурса будут рассчитаны. В дальнейшем через значение цены, соответствующей выбранному типу цен, будет найден второй ресурс. В случае категорийного бюджета  показатель планирования не доступен, т.к. жестко планируется сумма упр.

Способ округления  - определяет режим округления количества для весового товара. Невесовой товар округляется в любом случае.

Данное прогнозирование используется как для номенклатурного, так и для категорийного бюджетов продаж. Для категорийного бюджета (в случае если выбран тип анализа "по фактическим данным"), после получение результата, данные сворачиваются по категориям номенклатуры (по типу номенклатуры). В случае, если выбран тип анализа "по ранее планируемым данным", результат сразу получается по категориям (различие связано с различной структурой регистров «Бюджет продаж» и  «Продажи»)

Реквизиты

  • Количество прошлых периодов. Число. Количество анализируемых периодов.
  • Коэффициент роста продаж. Число. Экспертная оценка роста объема продаж в прогнозируемом периоде.
  • Коэффициент сезонности. Число. Экспертная оценка сезонной составляющей.
  • Показатель планирования. Булево. Вид прогнозируемого ресурса (количество или сумма упр.)
  • Способ округления. Число. Определяет способ округления ресурса "количество"
  • Сценарий планирования. Справочник  «Сценарий планирования». Соответствует сценарию планирования документа  «Бюджет продаж», из которого была вызвана форма. В форме не редактируется. 
  • Тип анализа. Перечисление  «Типы анализа». Определяет вид анализируемых данных: фактические данные продаж, либо ранее запланированные данные продаж.
  • Тип номенклатуры. Список значений. Содержит список типов номенклатуры. Используется для наложения ограничений на прогнозируемые данные.
  • Форма мастера. Произвольный. Используется при вызове данной формы из формы заполнения по документу основанию. Содержит ссылку на эту форму.
