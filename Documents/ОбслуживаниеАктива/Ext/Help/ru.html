Документ "Обслуживание актива"

Назначение

Документ Обслуживание актива предназначен для отражения в учете факта обслуживания нетоварного актива. Относится к подсистеме Прочие активы раздела Предприятие.

При проведении документ формирует хозяйственные операции:

  • Обслуживание актива на расходы подразделения. Хозяйственная операция уменьшает материальные активы (расходные материалы для обслуживания) на складах предприятия, хранимые на регистрах Остатки товаров компании и Партии товаров компании, с указанием статьи списания, отражает сумму списанных товаров на регистре Доходы и расходы, начисляет кредиторскую задолженность компании перед контрагентом на регистре Взаиморасчетов компании на сумму оказанных услуг (выполненных работ), отражает доход по другому подразделению компании, оказывающему услуги (выполняющему работы) нашему подразделению на регистре Доходы и расходы, отражает расход по нашему подразделению компании на регистре Доходы и расходы. Начисляется сумма обслуживания на регистре Прочие активы в эксплуатации.
  • Обслуживание актива на стоимость актива. Хозяйственная операция уменьшает материальные активы (расходные материалы для обслуживания) на складах предприятия, хранимые на регистрах Остатки товаров компании и Партии товаров компании, с указанием статьи списания, отражает сумму списанных товаров на регистре Доходы и расходы, начисляет кредиторскую задолженность компании перед контрагентом на регистре Взаиморасчетов компании на сумму оказанных услуг (выполненных работ), отражает доход по другому подразделению компании, оказывающему услуги (выполняющему работы) нашему подразделению на регистре Доходы и расходы. Начисляется сумма обслуживания и балансовая стоимость (на общую сумму обслуживания) на регистре Прочие активы в эксплуатации.

Реквизиты

  • Актив. Реквизит является обязательным для заполнения. Ссылается на справочник Прочие активы. Актив, подлежащий обслуживанию.
  • Тип обслуживания. Реквизит является обязательным для заполнения. Ссылается на справочник Типы обслуживания.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник Склады  компании. Склад компании, на котором происходит списание номенклатуры на обслуживание актива.
  • Статья расходов. Ссылается на справочник  Статьи доходов и расходов. Реквизит используется, если с помощью меню кнопки Операция выбрана хозяйственная операция Обслуживание на расходы подразделения. В этом случае автоматически после заполнения реквизита Тип обслуживания устанавливается статья расходов, указанная в справочнике Типы обслуживания для данного типа. При необходимости можно вручную выбрать другую статью.
  • Статья доходов. Ссылается на справочник  Статьи доходов и расходов. Статья доходов по обслуживанию актива (по подразделению, обслуживающему актив).

Вкладка «Обслуживание»

  • Контрагент. Ссылается на справочник Контрагенты и контакты. Контрагент, выполняющий работы по обслуживанию актива.
  • Договор. Ссылается на справочник Договоры взаиморасчетов. Договор со сторонней организацией, в рамках которого производится обслуживание.
  • Стоимость сторонних услуг. Стоимость работ по обслуживанию. Сумма денег, указанная в этом реквизите, будет учтена при взаиморасчетах
    с контрагентом как при обычном оприходовании услуг.

Табличная часть «Расходные материалы»

В табличной части Расходные материалы перечисляются все собственные товарно-материальные ценности предприятия, потраченные при обслуживании актива, количество и остаток этих товарно-материальных ценностей на складе.

Табличная часть «Оприходованные ценности»

На вкладке Оприходованные ценности документа Обслуживание актива перечисляются товарно-материальные ценности, оприходованные при обслуживании актива, количество, сумма оприходования.

Табличная часть «Зачет авансов»

Вкладка Зачет авансов содержит способ зачета оплаты/авансов, а так же список подобранных вручную документов оплаты и суммы оплаты для погашения задолженности. Так же в список можно добавить документы с помощью кнопки Подбор авансов. Данный список документов оплат доступен только при выборе способа зачета оплаты/авансов "По документу".
  • Документ аванса. Реквизит является уникальным и обязательным для заполнения. Документ, на основании которого будет погашена задолженность.
  • Сумма. Реквизит обязательным для заполнения. Сумма для зачета по документу аванса.

Ввод на основании

Документ Обслуживание актива вводится на основании следующих документов:

  • Ввод в эксплуатацию,
  • Ввод остатков прочих активов,
  • Перемещение активов.

Документы, вводимые на основании:

  • Расходный кассовый ордер,
  • Корректировка движений.

Печатные формы

  • Обслуживание актива. Простая печатная форма, содержит параметры обслуживания и список списываемой со склада номенклатуры для обслуживания.
  • ОС-3 (Акт о приемке-сдаче отремонтированных, реконструированных, модернизированных объектов основных средств). Унифицированная печатная форма №ОС-3, утвержденная постановлением Госкомстата России.
