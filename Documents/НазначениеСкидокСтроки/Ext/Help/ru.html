Документ "Назначение строчных скидок"

Назначение

Документ Назначение строчных скидок предназначен для назначения или отмены действующих скидок/наценок на товарную строку документа.

При проведении документ формирует хозяйственные операции:

  • Назначение строчных скидок. Хозяйственная операция регистрирует установку скидки в регистре сведений Скидки на товарные строки.
  • Отмена строчных скидок. Хозяйственная операция регистрирует отмену скидки в регистре сведений Скидки на товарные строки.

Реквизиты

  • Действует с. Реквизит является обязательным для заполнения. Дата начала действия скидки.

Табличная часть «Скидки»

  • Скидка. Реквизит является обязательным для заполнения. Ссылается на справочник Типы скидок и наценок. Определяет тип назначаемой скидки.
  • Вытесняющая. Если этот флажок установлен, то при применении скидки остальные скидки не действуют.
  • Значение. Значение скидки. Для абсолютных скидок - сумма, для относительных - процент.
  • %. Графа не редактируется. Она содержит наименование единицы измерения скидки (например, для процентной скидки — «%»).
  • Номенклатура. Товар или группа товаров, на которые назначается скидка. Если не указан, то скидка действует для всех товаров. Поля Тип номенклатуры и Свойство очищаются при выборе номенклатуры.
  • Тип номенклатуры. Скидка назначается на все товары выбранного типа номенклатуры. При выборе типа номенклатуры поле Номенклатура очищается.
  • Свойство. Наименование свойства номенклатуры, для которого будет использоваться скидка.
  • Количество. Количество номенклатуры, на которое назначается скидка.
  • Правило. Правило расчета скидки по количеству. Выбирается из списка:
    • Не используется - скидка предоставляется на товар, без ограничения количества.
    • Больше - скидка предоставляется на количество товара, превышающее указанное.
    • Меньше - скидка предоставляется на количество товара, не превышающее указанное.
    • Упаковка - скидка предоставляется на упаковку.
  • Сумма строки. Сумма, начиная с которой действует скидка.
  • Дисконтная карта. Ссылается на справочник Карточки. Дисконтная карта, по которой предоставляется скидка.
  • На сумму покупок. Скидка начинает действовать, если сумма покупок по дисконтной карте не менее указанной суммы.
  • С/По. Время действия скидки.
  • Пн...Вс. Семь флажков, соответствующих дням недели действия скидки (с понедельника по воскресенье). Если некоторый флажок снят, то в этот день скидка не предоставляется.
  • Сумма подарка. Ограничение на стоимость предоставляемого подарка.
  • Склад. Ссылается на справочник Склады компании. Склад, на котором предоставляется данная скидка.

Ввод на основании

На основании документа Назначение строчных скидок могут быть введены документы:

  • Корректировка движений,
  • Электронное письмо,

  • Событие.

Печатные формы

  • Назначение строчных скидок. Простая печатная форма, содержит список установленных или отмененных скидок/наценок.
