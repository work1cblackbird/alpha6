Произвольная форма "Автозаполнение"

Назначение

Произвольная форма документа «Бюджет доходов и расходов», предназначена для заполнения табличной части документа. Включает в себя три раздельных заполнения:

Заполнение статьями доходов от основной деятельности

  • Заполнение на основании продаж. Анализируются обороты по регистру продаж. В этом случае необходимо выбрать количество анализируемых периодов, а также задать коэффициенты роста продаж и сезонности. Среднее значение за один период будет является итоговой суммой. Однако, если в справочнике «Статьи доходов и расходов» создано более чем один элемент с видом "Доход от основной деятельности", то сумма будет распределена по этим статьям пропорционально оборотам по этим статьям в регистре «Доходы и расходы»
  • Заполнение по документу основанию. Сумма дохода в этом случае определяется как итоговая сумма, по выбранному документу «Бюджет продаж». Аналогично, если статей несколько, то будет произведено автоматическое распределение.

Заполнение статьями расходов от основной деятельности

  • По фактическим данным продаж. Анализируются оборот (себестоимость упр.) по регистру продаж. В этом случае необходимо выбрать количество анализируемых периодов, а также задать коэффициенты роста себестоимости и сезонности. Среднее значение за один период будет является итоговой суммой. Однако, если в справочнике «Статьи доходов и расходов» создано более чем один элемент с видом "Расходов от основной деятельности", то сумма будет распределена по этим статьям пропорционально оборотам по этим статьям в регистре «Доходы и расходы»
  • По фактическим данным закупок. Анализируется приход ресурса "Сумма упр" регистра «Партии товаров компании» (Фактически анализируются объем закупок)
  • Заполнение по документу основанию (Бюджет продаж). Сумма дохода в этом случае определяется как итоговая сумма реквизита "Себестоимость упр", выбранного документа «Бюджет продаж». Аналогично, если статей несколько, то будет произведено автоматическое распределение по статьям.
  • Заполнение по документу основанию (Бюджет закупок). Сумма дохода в этом случае определяется как итоговая сумма, по выбранному документу «Бюджет закупок». Аналогично, если статей несколько, то будет произведено автоматическое распределение.

Заполнение статьями накладных доходов и расходов

При активном флажке, табличная часть будет заполнена всеми статьями накладных расходов из справочника  «Статьи расходов и доходов», непомеченных на удаление.

Способ вызова

Форма вызывается из меню командной панели табличной части документа«Бюджет доходов и расходов»  ("Автозаполнение").

Реквизиты

  • Документ основание (выручка). Ссылка на документ основание «Бюджет продаж». 
  • Документ основание (себестоимость). Ссылка на документ основание «Бюджет продаж», либо   «Бюджет закупок»
  • Заполнить статьями "Выручка". Тип значения: «Булево». Признак использование заполнения статьями доходов от основной деятельности.
  • Заполнить статьями  "Себестоимость".  Тип значения: «Булево». Признак использование заполнения статьями расходов от основной деятельности.
  • Заполнить статьями накладных расходов и доходов. Тип значения: «Булево». Признак использования заполнения статьями накладных расходов и доходов.
  • Количество периодов (Выручка). Тип значения: «Число». Количество периодов для анализа оборотов по регистру «Продажи». Используется при заполнении статьями доходов от основной деятельности.
  • Количество периодов (Себестоимость). Тип значения: «Число». Количество периодов для анализа оборотов по регистру «Продажи». Используется при заполнении статьями расходов от основной деятельности.
  • Коэффициент роста (Выручка). Тип значения: «Число». Коэффициент увеличения продаж в прогнозируемом периоде. Используется при заполнении статьями доходов от основной деятельности.
  • Коэффициент роста (Себестоимость). Тип значения: «Число». Коэффициент увеличения себестоимости в прогнозируемом периоде. Используется при заполнении статьями расходов от основной деятельности.
  • Коэффициент сезонности (Выручка). Тип значения: «Число». Поправка на сезонность продаж в прогнозируемом периоде. Используется при заполнении статьями доходов от основной деятельности.
  • Коэффициент сезонности (Себестоимость). Тип значения: «Число». Поправка на сезонность продаж в прогнозируемом периоде. Используется при заполнении статьями расходов от основной деятельности
  • Способ расчеты выручки. Тип значения: «Булево». Определяет способ заполнения статьями доходов от основной деятельности.
  • Способ расчеты себестоимости. Тип значения: «Булево». Определяет способ заполнения статьями расходов от основной деятельности.
