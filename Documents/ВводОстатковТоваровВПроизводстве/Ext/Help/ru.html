Документ "Ввод остатков товаров в производстве"

Назначение

Документ Ввод остатков товаров в производстве предназначен для ввода начальных остатков по товарам в производстве при начальном заполнении информационной базы при вводе ее в эксплуатацию, либо при переносе первоначальных данных из другой информационной базы.

Относится к подсистеме Производство раздела Автосервис.

При проведении документ формирует хозяйственную операцию:

  • Ввод остатков товаров в производстве. Хозяйственная операция отражает начальные остатки товаров в производстве.

Реквизиты

  • Заказ-наряд. Реквизит является обязательным для заполнения. Заказ-наряд, по которому ведется производство.
  • Поставщик. Реквизит является обязательным для заполнения. Ссылается на справочник Контрагенты и контакты. Контрагент, который поставил или принял товар.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник Договоры взаиморасчетов. Договор взаиморасчетов с контрагентом.
  • Статья приходования ТМЦ. Ссылается на справочник Статьи доходов и расходов. Статья дохода, по которой приходуется товар.
  • Цех. Реквизит является обязательным для заполнения. Ссылается на справочник Цеха. Цех, в который приходуются товары.
  • По договору долг контрагента составляет. Гиперссылка отражает текущее состояние расчетов с контрагентом по договору. Переход производится на отчет Взаиморасчеты с контрагентом, сформированный для указанного контрагента.

Табличная часть «Товары»

Табличная часть документа может быть заполнена товарами из заказ-наряда при помощи пункта Заполнить товарами из заказ-наряда меню кнопки Заполнение.

Графы табличной части диалогового окна Ввод остатков товаров в производстве:

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Номенклатура. Номенклатура, для которой вводятся остатки.
  • Количество. Реквизит является обязательным для заполнения. Количество остатков номенклатуры.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Единицы измерения. Единица измерения номенклатуры.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета количества номенклатуры по отношению к базовой единице номенклатуры.
  • Цена. Цена номенклатуры.
  • Сумма. Сумма по номенклатуре без учета НДС.
  • Всего. Сумма по номенклатуре с учетом НДС.
  • % НДС. Ставка НДС для номенклатуры.
  • НДС. Сумма НДС по номенклатуры.  
  • ГТД. Номер грузовой таможенной декларации (для товаров, поступивших по импорту).

Ввод на основании

Вводится на основании документа Заказ-наряд.

Документы, вводимые на основании:

Возврат товаров поставщику,

Изменение цен компании и контрагентов,

Корректировка.

Печатные формы

  • Остатки товаров в производстве. Простая печатная форма документа.
