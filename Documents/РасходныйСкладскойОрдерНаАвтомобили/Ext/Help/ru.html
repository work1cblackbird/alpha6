Документ "Расходный складской ордер на автомобили"

Назначение

Отгрузка автомобилей с ордерного или ячеистого склада.

При проведении документ формирует хозяйственные операции:

  •  Расходный складской ордер на автомобили. Хозяйственная операция расходует со складов/стоянок предприятия, имеющие вид "Ордерный" или "Ордерный и ячеистый", автомобили, хранимые в регистре «Остатки автомобилей на ордерном складе».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Приходный складской ордер на автомобили», «Переразмещение автомобилей».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого отгружаются автомобили.
  • Владелец товара. Ссылается на справочники «Контрагенты и контакты». Контрагент, получающий автомобиль.
  • Документ основание. Ссылается на документы «Приходный складской ордер на автомобили», «Переразмещение автомобилей». Документ, на основании которого введен данный.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Юридическое лицо, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, создавший документ.
  • Операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Отгружаемый автомобиль.
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество автомобилей. Всегда = 1.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Место хранения ячеистого склада.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Расходный ордер.  Простая печатная форма, содержит список расходуемых со склада автомобилей. Если склад  ячеистый, содержит колонку, отображающую ячейки, из которых отгружаются автомобили.
  • Карта склада. Выводит графическое представление склада с отображением маршрута движения. Маршрут начинается в точке входа склада и заканчивается в точке выхода. Вывод карты склада возможен только для ячеистого склада.  

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Расходный складской ордер на автомобили"

Журналы, отображающие документ: «Складские ордера».

Движения

  • Остаточный регистр «Остатки автомобилей на ордерном складе».
