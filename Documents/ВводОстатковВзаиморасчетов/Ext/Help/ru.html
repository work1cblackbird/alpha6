Документ "Ввод остатков взаиморасчетов"

Назначение

Документ Ввод остатков взаиморасчетов служит для ввода начальных остатков по взаиморасчетам с контрагентами, имеющими дебиторскую или кредиторскую задолженность по различным договорам взаиморасчетов. Относится к подсистеме Ввод начальных остатков раздела Предприятие. 

При проведении документ формирует хозяйственную операцию:

  • Ввод остатков взаиморасчетов. Хозяйственная операция отражает начальные остатки взаимозачетов с контрагентом, имеющим дебиторскую или кредиторскую задолженность.

Табличная часть «Состав»

  • Контрагент. Контрагент, для которого вводятся остатки по взаиморасчетам.
  • Договор.  Договор взаиморасчетов с контрагентом.
  • Сделка. Сделка взаиморасчетов с контрагентом. Выбирается из перечня документов.
  • Курс взаиморасчетов. Курс валюты, по которой ведутся взаиморасчеты. 
  • Дебет. Сумма задолженности контрагента перед компанией по данному договору.
  • Сумма кредит. Сумма задолженности компании перед контрагентом по данному договору.
  • Заказ. Заказ, по которому произошли расчеты с контрагентом.
  • Тип расчета. Признак, по которому определяется вид оплаты: аванс или по документу.
  • Способ зачета авансов. Доступен только для договоров с признаком учета взаиморасчетов "По сделкам". Если установлено в значение "Автоматически", то при проведении документа все незакрытые сделки будут автоматически закрываться (в порядке ФИФО). В значении "Не зачитывать" - незакрытые сделки не будут погашены.

Ввод на основании

На основании документа Ввод остатков взаиморасчетов можно ввести документ Корректировка движений.

Печатные формы

  • Остатки взаимозачетов. Простая печатная форма документа.
