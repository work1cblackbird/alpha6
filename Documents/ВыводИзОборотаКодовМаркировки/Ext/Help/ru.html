Документ "Вывод из оборота кодов маркировки"

Назначение

Документ «Вывод из оборота кодов маркировки» предназначен для формирования факта вывода из оборота кодов маркировки и отправки документа в государственную информационную систему мониторинга за оборотом товаров (ГИС МТ). 

ВАЖНО! Перед использованием документа необходимо убедится в наличии личного кабинета в ГИС МТ для данной организации. Необходимо указать в настройках способ обмена с ГИС МТ через файловый обмен или API ГИС МТ.

ВАЖНО! При розничной реализации маркированных товаров сведения о выводе из оборота товаров с применением контрольно-кассовой техники (ККТ) должны быть переданы Оператору ГИС МТ. При наличии договора и поручения оператор фискальных данных (ОФД) осуществляет ежедневную передачу в режиме реального времени информации в ГИС МТ по каждой реализованной товарной единице, полученной от участника оборота товаров. При отсутствии договора с ОФД сведения о выводе товара из оборота с применением ККТ передаются самостоятельно не позднее 30 календарных дней со дня продажи товара. Таким образом, при розничной продаже товаров и наличии договора с ОФД данный документ выгружать в ГИС МТ не нужно.

Относится к подсистеме Маркировка товаров раздела Снабжение.

Реквизиты

  • Статус. Реквизит является обязательным для заполнения. Ссылается на перечисление «Статусы документов маркировки». Отображает текущее состояние документа.
  • Товарная группа. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы маркировки». Определение товарной группы маркируемого товара.
  • Идентификатор документа. Тип значения: «Строка». Заполняется автоматически при отправке документа в ГИС МТ при использовании обмена через API.
  • Причина. Реквизит является обязательным для заполнения. Ссылается на перечисление «Причины вывода из оборота». Выбранная причина вывода из оборота товара.
  • Дата вывода. Тип значения: «Дата». Дата вывода из оборота кода маркировки товара.
  • Вид первичного документа. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды первичных документов». Вид документа, на основании которого производится вывод товара из оборота.
  • Наименование первичного документа. Реквизит является обязательным для заполнения при указании вида первичного документа «Прочее». Тип значения: «Строка». Наименование документа, на основании которого производится вывод товара из оборота.
  • Дата первичного документа. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата документа, на основании которого производится вывод товара из оборота.
  • Номер первичного документа. Реквизит является обязательным для заполнения. Тип значения: «Строка». Номер документа, на основании которого производится вывод товара из оборота.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ. На основании ИНН организации формируется запрос в ГИС МТ.  
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ. 
  • Документ основание. Ссылается на документ «Чек», «Списание товаров», «Реализация товаров», «Заказ-наряд», «Корректировка реализации».  Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов. 
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Контрагент. Реквизит является обязательным для заполнения при выборе причины вывода из оборота «Безвозмездная передача». Ссылается на справочник «Контрагенты». Покупатель товаров с кодами маркировки.
  • Описание причины. Реквизит является обязательным для заполнения при выборе причины вывода из оборота «Другое». Тип значения: «Строка». Описание иной причины вывода из оборота.
  • Импортер. Ссылается на справочник «Контрагенты». Участник оборота товаров ЕАЭС, которому отгружен товар. Заполняется при выборе причины вывода из оборота «Трансграничная продажа в страны ЕАЭС». Код страны импортера должен соответствовать одному из: 112 - Беларусь; 398 - Казахстан; 417 - Киргизия; 051 - Армения. Длина налогового номера у импортера должна соответствовать стране регистрации: Беларусь - 9 цифр; Казахстан - 12 цифр; Киргизия - 14 цифр; Армения - 8 цифр. При нарушении данных условий импортер не будет передан при выгрузке документа. Данные об импортере выгружаются только при выгрузке документа через API.
  • Адрес места выбытия. Тип значения: «Строка». Адрес места выбытия товаров с кодами маркировки. Указывается только в соответствии с данными ФИАС, иначе адрес не будет выгружен с документом. Заполняется при выборе причин вывода из оборота «Возврат физическому лицу», «Дистанционная продажа», «Другое», «Конфискация», «Продажа по образцам», «Розничная продажа», «Уничтожение», «Утилизация», «Утрата».
  • Идентификатор государственного контракта. Тип значения: «Строка». Идентификатор содержит 25 цифр, при этом 13 разряд принимает значение от 1 до 3 включительно, 14 — 25 разряды могут быть заполнены нулями. Реквизит является обязательным для заполнения при выборе причин вывода из оборота «Продажа по сделке с государственной тайной», «Продажа по государственному (муниципальному) контракту».

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Характеристика. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п. 
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество товаров с кодами маркировки.
  • Единица измерения. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Количество кодов маркировки для заказа.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент единицы измерения.
  • Цена. Тип значения: «Число». Цена за единицу товара. Указывается обязательно для всех причин вывода из оборота за исключением «Экспорт за пределы стран ЕАЭС».
  • Вид первичного документа. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды первичных документов». Вид документа, на основании которого производится вывод товара из оборота.
  • Наименование первичного документа.  Тип значения: «Строка». Наименование документа, на основании которого производится вывод товара из оборота.
  • Дата первичного документа. Тип значения: «Дата». Дата документа, на основании которого производится вывод товара из оборота.
  • Номер первичного документа. Тип значения: «Строка». Номер документа, на основании которого производится вывод товара из оборота.

Табличная часть «Коды маркировки»

  • Код маркировки. Реквизит является уникальным и обязательным для заполнения. Тип значения: «Строка». Код маркировки товара, которому изменяется состояние.

Кнопка Выгрузить используется для формирования документа «Вывод из оборота» и его отправки (при настройке обмена через API) в ГИС МТ. Для отображения факта отправки документа в учетную систему необходимо перевести документ в состояние «Отправлен». После успешной загрузки документа необходимо установить статус «Выполнен» и для перевода указанных кодов маркировки в новый статус в том случае когда документ-основание не изменил.

Ввод на основании

Документ Вывод из оборота кодов маркировки может вводиться на основании следующих документов:

  • Чек, 
  • Списание товаров,
  • Реализация товаров,
  • Заказ-наряд,
  • Корректировка реализации.
