Документ "Корректировка поступления автомобилей"

Назначение

Документ Корректировка поступления автомобилей предназначен для регистрации исправленных первичных документов, полученных от поставщика. Например, при выявлении ошибки при оформлении первичных документов или в случае согласованного между продавцом и покупателем изменения стоимости ранее поступивших автомобилей.

Документ относится к подсистеме Закупка раздела Снабжение автосалона.

Документ Корректировка поступления автомобилей формирует следующие хозяйственные операции:

  • Исправление в первичных документах. Используется для регистрации исправленных первичных документов поставщика. При проведении документа выполняется аннулирование движений по исходному первичному документу в том налоговом периоде, в котором он был зарегистрирован, и формируются движения по исправленному счету-фактуре.
  • Корректировка по согласованию сторон. Для отражения в учете покупателя операции по изменению стоимости ранее приобретенных автомобилей, предусмотренного договором с поставщиком, например, при получении корректировочного счета-фактуры. Используется при перегрузке в бухгалтерию.

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник Склады компании. Склад по документу поступления.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник Контрагенты и контакты. Поставщик по документу поступления.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник Договоры взаиморасчетов. Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Основание. Реквизит является обязательным для заполнения. Документ, на основании которого введена корректировка.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник Автомобили. Поступивший автомобиль.  
  • VIN. VIN-номер автомобиля. 
  • Количество. Количество автомобилей.
  • Количество по документу поступления. Количество автомобилей по документу поступления.
  • Количество разница. Разница между Количеством автомобилей и количеством автомобилей по документу поступления.
  • Цена. Цена автомобиля в валюте документа.
  • Сумма. Сумма автомобиля. Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена.
  • Сумма по документу поступления. Сумма автомобиля из документа поступления. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена. 
  • Сумма разница. Разница между суммой автомобиля и суммой автомобиля согласно документу поступления.
  • % НДС. Ставка НДС. Ссылается на справочник Ставки НДС.
  • % НДС по документу поступления. Ставка НДС согласно документу поступления. Ссылается на справочник Ставки НДС.
  • Сумма НДС. Сумма НДС автомобиля.
  • Сумма НДС по документу поступления. Сумма НДС автомобиля согласно документу поступления.
  • Сумма НДС разница. Разница между суммой НДС и суммой НДС согласно документу поступления.
  • Всего. Итоговая сумма по строке документа со всеми налогами.
  • Всего по документу поступления. Итоговая сумма по строке документа со всеми налогами согласно документу поступления.
  • Всего разница. Разница между итоговыми суммами по строке документа со всеми налогами и суммами согласно документу поступления.

Ввод на основании

Вводится на основании документов:

  • Корректировка поступления автомобилей,
  • Поступление автомобилей.

Документы, вводимые на основании:

  • Корректировка поступления автомобилей,
  • Счет-фактура полученный,
  • Корректировка движений.
