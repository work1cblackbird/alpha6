Документ "Уведомление об остатках прослеживаемых товаров"

Назначение

Документ "Уведомление об остатках прослеживаемых товаров" предназначен для оформления уведомления Федеральной налоговой службы (ФНС) об имеющихся остатках на предприятии прослеживаемых товаров к дате запуска учета.

Данный документ позволяет сформировать уведомление о товарах в разрезе кода ТН ВЭД и первичного документа. Обмен с ФНС происходит с помощью выгрузки файла из документа, который необходимо передать через оператора ЭДО в ФНС. Из полученного ответа от ФНС на уведомление необходимо внести в систему регистрационный номер партии товаров (РНПТ) для учета прослеживаемых товаров.

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ. 
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ. 
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в которой был выписан документ. 
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Код ТН ВЭД. Реквизит является обязательным для заполнения. Ссылается на справочник «Классификатор ТН ВЭД». Код прослеживаемого товара по классификатору ТН ВЭД.
  • РНПТ. Реквизит является обязательным для заполнения при установке состояния документа в «Получено подтверждение из ФНС». Ссылается на справочник «ГТД». РНПТ прослеживаемых товаров, который был получен от ФНС.
  • Страна. Реквизит является обязательным для заполнения. Ссылается на справочник «Страны мира». Страна происхождения прослеживаемого товара для указания при формировании РНПТ.
  • Первичный документ. Реквизит является обязательным для заполнения. Ссылается на документы «Инвентаризация», «Инвентаризация автомобилей». Документ, на основании которого была произведена инвентаризация прослеживаемого товара.
  • Единица измерения по первичному документу. Ссылается на справочник «Единицы измерения». Единица измерения товара по тому как указан в первичном документе. Если не заполнена, то при выгрузке файла уведомления указывается кодом "796".
  • Единица измерения прослеживаемости. Ссылается на справочник «Классификатор единиц измерения». Единица измерения, в которых происходит прослеживаемость товаров. Если не заполнена, то при выгрузке файла уведомления указывается кодом "796".
  • Код ОКПД2. Ссылается на справочник «Классификатор ОКПД2». Код прослеживаемого товара по классификатору ОКПД2. 
  • Состояние. Реквизит является обязательным для заполнения. Ссылается на перечисление «Состояния уведомления прослеживаемости товаров». Текущее состояние документа устанавливается пользователем. При создании нового уведомления ему присваивается состояние "Новый". Если уведомление было отправлено в ФНС, то необходимо установить состояние "Отправлен в ФНС". При получении ответа с РНПТ - его необходимо внести в документ и установить состояние "Получено подтверждение из ФНС". В всех остальных случаях устанавливать состояние "Отклонен".
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочники «Номенклатура», «Автомобили». Номенклатура (товар) или автомобиль, который подлежит прослеживаемости. 
  • Код ОКПД2. Ссылается на справочник «Классификатор ОКПД2». Код прослеживаемого товара по классификатору ОКПД2. 
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения по первичному документу номенклатуры. 
  • Количество прослеживаемости. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения прослеживаемости номенклатуры. 
  • Сумма без НДС. Тип значения: «Число». Сумма прослеживаемого товара без НДС. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа. 
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Прослеживаемость товаров --> Уведомление об остатках прослеживаемых товаров".

 

Движения

Движения формируются после ввода РНПТ и установке состояния документа "Получено подтверждение из ФНС". В зависимости от первичного документа выполняются следующие движения по регистрам накопления:
  • «ГТД партии товаров». 
  • «Остатки автомобилей».
