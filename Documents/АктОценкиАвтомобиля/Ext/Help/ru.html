Документ "Акт оценки автомобиля"

Назначение

Документ предназначен для сбора информации об автомобиле:

  • Комплектация
  • Идентификационные номера
  • Документы на автомобиль
  • Техническое состояние
  • Повреждения
  • Фотоматериалы

Входит в состав подсистемы: Автомобили с пробегом

Реквизиты


Если в поле автомобиль указано значение поля Марка, Модель и Вариант комплектации блокируются на изменение. Команды создания новых документов ПТС и Свидетельство о регистрации проверяют заполнение поля автомобиль.

В полях группы Цена по рынку указываются диапазон для ограничения возможного значения цены выкупа. Обычно заполняется на основании данных из внешних систем.

Поле Итоговая цена служит для фиксации цены выкупа предлагаемой оценщиком, согласование и фиксация цены происходить документом Изменение цен автомобилей.

Пока не созданы владелец и автомобиль лучше не переходить к дальнейшему заполнению, так как переключатель Осмотр произведен будет не доступен.

Вкладка Осмотр содержит в себе группу вложенных вкладок с табличными частями содержащими свойства и характеристики автомобилей к осмотру. Их состав и расположения на закладках зависит от настроек выполненных в Свойства и характеристики автомобилей. В новый документ добавляются свойства и характеристики с установленным признаком Добавлять по умолчанию в акт осмотра. Состав вложенных страниц определяется по родителям добавленных свойств, если у свойства нет родителя оно попадает на предопределенную страницу Прочее. Перечень свойств может быть расширен в ручную.

Кроме указания значения для свойства автомобиля для строки можно прикрепить файл с фотографией. Просмотр выполняется в отдельной форме галерее. Для удаления необходимо удалить файл из формы просмотра. Пометка на удаление файла в форме списка присоединенных файлов не даст результата.
