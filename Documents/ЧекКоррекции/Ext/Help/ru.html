Документ "Чек коррекции"

Назначение

Чек коррекции является одновременно и подтверждением исполнения обязанности по исправлению чеков, пробитых с ошибками, и уведомлением о нарушении со стороны пользователя кассы. Сообщать налоговикам о нарушении дополнительно не требуется. Так же чеки коррекции не отражаются в отчете о розничных продажах, не попадают в документ «Закрытие кассовой сменой», что является правильным поведением. Для исправления ошибок управленческого и бухгалтерского учета есть возможность сторнировать документы и использовать корректирующие документы. Соответственно чеки коррекции работают только в кассовом учете и служат для передачи корректной информации об операции в ОФД. 

Документ относится к подсистеме "Розница" раздела "Продажи".

Реквизиты

  • Дата корректируемого расчета. Дата совершения корректируемого расчета. Не должна превышать текущую дату. Берется из документа-основания. Если документа-основания нет, то дата должна попадать в расчетный период, в котором был осуществлен корректируемый расчет.
  • Тип коррекции. Может принимать только значения: "Самостоятельно" и "По предписанию налогового органа". 
  • Номер корректируемого чека. В данный реквизит указывается фискальный признак корректируемого документа.
  • Отменяющий чек коррекции. Признак того, что данный документ является сторнирующим. При установленном признаке данные документа полностью дублируют корректируемый документ за исключением реквизита "Способ расчета".
  • Способ расчета. Определяет вид операции внесения или изъятия денежных средств коррекции. Для отменяющего документа может принимать значения "Возврат прихода", "Возврат расхода".

Табличная часть «Товары»

Табличная часть Товары содержит информацию о номенклатурных позициях, проданных по данному чеку.

Табличная часть «Оплаты»

На вкладке Оплаты отображаются данные, связанные с оплатой по данному чеку на оплату: сумма, уплаченная наличными, тип
платежной карты, по которой была произведена безналичная оплата (выбирается из справочника Типы платежных карт), и т. д.
