Документ "Уведомление о перемещении прослеживаемых товаров"

Назначение

Документ "Уведомление о перемещении прослеживаемых товаров" предназначен для оформления уведомления Федеральной налоговой службы (ФНС) о вывозе прослеживаемых товаров на территорию другого государства-члена Евразийского экономического союза с территории Российской Федерации и иные территории, находящиеся под ее юрисдикцией.

Данный документ позволяет сформировать уведомление о товарах в разрезе покупателей и сопроводительных документов. Обмен с ФНС происходит с помощью выгрузки файла из документа, который необходимо передать через оператора ЭДО в ФНС. 

Относится к подсистеме: «Прослеживаемые товары».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ. 
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ. 
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в которой был выписан документ. 
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Состояние. Реквизит является обязательным для заполнения. Ссылается на перечисление «Состояния уведомления прослеживаемости товаров». Текущее состояние документа устанавливается пользователем. При создании нового уведомления ему присваивается состояние "Новый". Если уведомление было отправлено в ФНС, то необходимо установить состояние "Отправлен в ФНС". 
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.

Табличная часть «Контрагенты»

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Контрагент, которому был прослеживаемый товар продан.

Табличная часть «Товары»

В данной табличной части описываются прослеживаемые товары, которые были проданы каждому контрагенту из табличной части «Контрагенты»
  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочники «Номенклатура», «Автомобили». Номенклатура (товар) или автомобиль, который подлежит прослеживаемости. 
  • Код ТН ВЭД. Ссылается на справочник «Классификатор ОКПД2». Ссылается на справочник «Классификатор ТН ВЭД». Код прослеживаемого товара по классификатору ТН ВЭД. 
  • Единица измерения по первичному документу. Ссылается на справочник «Единицы измерения». Единица измерения товара по тому как указан в сопроводительном документе. 
  • Единица измерения прослеживаемости. Ссылается на справочник «Классификатор единиц измерения». Единица измерения, в которых происходит прослеживаемость товаров. 
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения по сопроводительному документу номенклатуры. 
  • Количество в единицах прослеживаемости. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество в единицах измерения прослеживаемости номенклатуры. 
  • Сумма без НДС. Реквизит является обязательным для заполнения. Тип значения: «Число». Сумма прослеживаемого товара без НДС.
  • РНПТ. Реквизит является обязательным для заполнения. Ссылается на справочник «ГТД». РНПТ товара, который был продан.
  • Сопроводительный документ. Реквизит является обязательным для заполнения. Ссылается на документы «Реализация товаров» и «Реализация автомобилей». Документ, на основании которого произошла передача прослеживаемого товара контрагенту.
  • Порядковый номер. Реквизит является обязательным для заполнения. Тип значения: «Число». Номер строки товара в сопроводительном документе (счет-фактуре).

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа. 
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Прослеживаемость товаров --> Уведомление о перемещении прослеживаемых товаров".
