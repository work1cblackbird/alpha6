Документ "Перемещение товаров в производство"

Назначение

Документ Перемещение товаров в производство служит для оформления перемещения деталей, требуемых для выполнения работ по заказ-наряду, со склада компании в производство. Этот документ может быть создан только на основании заказ-наряда.

Для завершения работ по заказ-наряду (закрытия заказ-наряда) в производство должны быть перемещены детали в полном объеме, необходимом для выполнения ремонтных работ.

Список хозяйственных операций:

  • Перемещение товаров в производство. Хозяйственная операция производит расход деталей для выполнения работ по заказ-наряду со склада компании (регистр Остатков товаров компании) и приходование их и деталей заказчика на регистр ТоварыВПроизводстве. Для деталей заказанных у поставщиков для выполнения работ по заказ-наряду производится закрытие выполненных заказов покупателей.

Реквизиты

  • Склад. Склад компании, с которого производится перемещение деталей в производство по заказ-наряду.
  • Заказ-наряд. Ссылка на документ, на основании которого введено перемещение. В соответствии с заказ-нарядом, справа от этого поля отображается название автомобиля.
  • Цех. Цех, в который производится перемещение деталей для выполнения работ.
  • Отпустил. Сотрудник компании, который выдает детали со склада в производство.
  • Получил. Сотрудник, получивший детали в производство.

Табличная часть «Товары»

Вкладка Товары содержит список запчастей, перемещаемых в производство.

Существует несколько дополнительных способов ввода документа Перемещение товаров в производство:

  • Выполнением команды В производство в меню кнопки Действия, расположенной в панели инструментов вкладки Товары в документе Заказ-наряд.
  • При записи документа Заказ-наряд, если для подразделения установлено право Автоматическое формирование перемещений в производство.

В обоих случаях в документ Перемещение товаров в производство будут перемещены имеющиеся в наличии запчасти, для которых при формировании заказ-наряда указано, что они поставляются со склада предприятия, а также детали, поступившие по заказам покупателей,
сформированным на основании этого заказ-наряда.

Если для пользователя установлено право Перемещение деталей в производство, то пользователь сможет создавать и проводить документ Перемещение товаров в производство любым доступным способом. При отсутствии данного права пользователь сможет только создать и записать этот документ, но не сможет его провести.

После закрытия документа Заказ-наряд документы перемещения, введенные на его основании, не могут быть введены дополнительно, а также не могут быть отменены ранее введенные. При открытии форм этих документов они недоступны для редактирования.

Ввод на основании

Вводится на основании документа Заказ-наряд.

Документ Перемещение товаров в производство может являться основанием для документов:

  • Расходный складской ордер,
  • Корректировка движений.

Печатные формы

  • Перемещение товаров в производство. Накладная на детали, перемещенные в производство.
  • ТОРГ-13 (накладная на внутреннее перемещение). Унифицированная печатная форма №Торг-13, утвержденная постановлением Госкомстата России.
  • 1-Т (Товарно-транспортная накладная). Унифицированная печатная форма №1-Т, утвержденная постановлением Госкомстата России.
  • Приложение 4 (Транспортная накладная). Унифицированная печатная форма, утвержденная постановлением Госкомстата России.
  • М-11 (требование-накладная). Унифицированная форма № М-11, утвержденная постановлением Госкомстата России.
