Документ "Возврат товаров поставщику"

Назначение

Документ Возврат товаров поставщику предназначен для отражения операции по возврату поставщику ранее купленных или принятых от него на комиссию товаров. В табличной части документа не могут фигурировать услуги, работы и т.п. Относится к подсистеме Закупка раздела Снабжение.

При проведении документ формирует хозяйственные операции:

  • Возврат товаров поставщику. Хозяйственная операция является обратной от хозяйственной операции Поступление товаров:
    • Отражает сторнирование прихода товаров на склад.
    • Немедленно изменяет задолженность компании перед поставщиком (снимает задолженность за товары).
  • Возврат товаров поставщику комиссия. Хозяйственная операция является обратной хозяйственной операции Поступление товаров (комиссия). Отражает сторнирование прихода комиссионных товаров на склад.

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник Склады компании. Склад компании, с которого списываются возвращаемые поставщику товары.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Поставщик или комитент, которому возвращается товар.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Документ поставки. Документ, согласно которому были поставлены возвращаемые товары. Если документ не указан, то для каждого товара в таблице требуется указать документ поставки.
  • По договору долг контрагенту составляет. Гиперссылка отражает текущее состояние расчетов с контрагентом по договору. Переход производится на отчет Взаиморасчеты с контрагентами, сформированный для указанного контрагента. 
  • Ввести счет-фактуру. Если гиперссылка содержит текст «Ввести счет-фактуру», то при переходе по гиперссылке на основании данного документа будет создан счет-фактура выданный. Если счет-фактура был создан ранее, то гиперссылка будет содержать дату и номер выданного счета-фактуры; она позволяет открыть соответствующий документ.

Документ Возврат товаров поставщику содержит добавочную графу Партия. Это ссылка на документ, по которому был поставлен товар.

Если в правах пользователя установлен основной тип цен закупки, то в документе Возврат товаров поставщику будет устанавливаться основной тип цен закупки, даже в случае другого установленного типа цен в документе-основании.

Кнопка Подбор позволяет подобрать позиции справочника Номенклатура с помощью специальной формы подбора.

Кнопка Подбор по партиям позволяет заполнить таблицу товарами, купленными ранее у поставщика, указанного в документе. 

Ввод на основании

Документ Возврат товаров поставщику вводится на основании следующих документов:

  • Поступление товаров,
  • Возврат от покупателя,
  • Авансовый отчет,
  • Ввод остатков товаров.

На основании документа Возврат товаров поставщику могут быть введены документы:

  • Приходный кассовый ордер,
  • Счет-фактура выданный,
  • Расходный складской ордер,
  • Корректировка движений.

Печатные формы

  • Возвратная накладная. Простая печатная форма, содержит список возвращаемых товаров поставщику или комитенту.
  • ТОРГ-12 (Товарная накладная за поставщика). Унифицированная печатная форма №Торг-12, утвержденная постановлением Госкомстата России.
  • Универсальный передаточный документ.

Журналы, отображающие документ
