Документ "Извлечение товаров из производства"

Назначение

Документ Извлечение товаров из производства служит для учета возврата обратно на склад компании деталей, ранее переданных для выполнения работ по заказ-наряду, но не использованных. Из производства нельзя извлечь деталей более, чем ранее было перемещено туда для выполнения работ. Документ относится к подсистеме Производство раздела Автосервис.

Список хозяйственных операций:

  • Извлечение товаров из производства. Хозяйственная операция производит расход деталей для выполнения работ по заказ-наряду с регистра ТоварыВПроизводстве и приходование их на склад компании (регистр Остатков товаров компании).

Если для пользователя установлено право Перемещение деталей в производство, то пользователь сможет создавать и проводить документ Извлечение товаров из производства. Если данное право не установлено, пользователь сможет только создать и записать этот документ, но не сможет его провести.

Реквизиты

  • Цех. Реквизит является обязательным для заполнения. Ссылается на справочник Цеха. Цех, из которого производится извлечение деталей.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник Склады компании. Склад компании, на который производится перемещение деталей из производства по заказ-наряду.
  • Заказ-наряд. Ссылка на документ, на основании которого введен данный.
  • Отпустил. Сотрудник, выдавший детали из производства.
  • Получил. Сотрудник, получивший детали из производства.

При извлечении деталей на розничный склад устанавливаются розничные цены и возможна переоценка товаров на данном складе согласно новым ценам, если в справочнике Подразделения компании у подразделения установлен флажок При оприходовании товаров.

Табличная часть «Товары»

Вкладка Товары содержит список деталей, извлекаемых из производства.

Ввод на основании

Документ Извлечение товаров из производства вводится на основании документа Заказ-наряд.

Документ Извлечение товаров из производства может являться
основанием для документов:

  • Приходный складской ордер,
  • Корректировка движений.

Печатные формы

  • Извлечение товаров из производства;
  • ТОРГ-13 (Накладная на внутреннее перемещение);
  • 1-Т (Товарно-транспортная накладная);
  • Приложение №4 (Транспортная накладная).
