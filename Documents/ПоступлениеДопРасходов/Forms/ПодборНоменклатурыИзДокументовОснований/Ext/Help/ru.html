Произвольная форма "Подбор номенклатуры из документов-оснований"

Назначение

Произвольная форма документа «Поступление дополнительных расходов», предназначена для подбора номенклатуры из документов-оснований документа для распределения дополнительных расходов.

Способ вызова

Форма вызывается из меню командной панели табличной таблицы "Номенклатура" из произвольной формы "Распределение дополнительных расходов по товарам".

Принцип работы с формой

На форме выведено дерево. На верхнем уровне описаны документы-основания из соответствующего документа "Поступление дополнительных расходов". На втором уровне - номенклатура документов-оснований. У каждой строки номенклатуры имеется поле для выбора данной номенклатуры для распределения суммы дополнительных расходов. 

Дерево "Номенклатура"

  • Документ-основание. Документ, на который будут распределяться дополнительные расходы.
  • Номенклатура. Ссылается на справочник «Номенклатура». Номенклатура (товар) документа-основания.
  • Характеристика. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.   
  • Единица измерения. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • ГТД. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для товаров, поступивших по импорту).
