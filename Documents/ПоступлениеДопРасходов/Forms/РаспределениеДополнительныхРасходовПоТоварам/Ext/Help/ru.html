Произвольная форма "Распределение дополнительных расходов по товарам"

Назначение

Произвольная форма документа «Поступление дополнительных расходов», предназначена для ручного распределения суммы дополнительных расходов по номенклатуре документов-оснований.

Способ вызова

Форма вызывается из меню командной панели табличной части "Товары" документа для ручного распределения суммы дополнительных расходов для всех услуг, а также для конкретной услуги при двойном клике по ячейки "Способ распределения" соотвествующей строки.

 

Принцип работы с формой

Для начала работы с формой необходимо указать по какому способу будет распределена сумма дополнительных расходов - автоматичекое или ручное распределение. При автоматическом распределении данные будут распределены по способу распределения, который указан в самой услуге.

При выборе ручного распределения для каждой услуге форируется набор номенклатуры из документов-оснований, для которых пользователь может редактировать сумму услуги. Сумма услуги должна быть распределена полностью на товары. При этом имеется возможность распределенить всю суммы услуги на доходы и расходы, при этом необходимо наличия у услуги статьи дополнительных услуг.

При редактировании распределения всех улуг вся информация об услугах заносится в таблицу "Услуги", при редактировании одной услуги - данная таблица не выводится. При активизации строки таблицы "Услуги" строится дерево из документов-оснований документа и их номенклатуры для указания суммы дополнительных расходов.

Реквизиты

  • Режим распределения. Тип значения: «Число».. Количество анализируемых периодов.
  • Услуга. Ссылается на справочник «Номенклатура». Услуга, для суммы которой производится распределение.
  • Сумма. Тип значения: «Число».  Сумма услуги с учетом НДС.
  • Сумма без НДС. Тип значения: «Число».  Сумма услуги без учета суммы НДС.
  • Не распределено. Тип значения: «Число».  Разность между суммой услуги и распределенной по номенклатуре.
  • На доходы и расходы. Тип значения: «Булево». Признак того, что сумма услуги будет распределена на указанную в услуге статью дополнительных расходов. 

Таблица "Услуги"

  • Услуга. Ссылается на справочник «Номенклатура». Услуга, для суммы которой производится распределение.
  • Сумма. Тип значения: «Число». Сумма услуги с учетом НДС.
  • Сумма без НДС. Тип значения: «Число». Сумма услуги без учета суммы НДС.
  • Не распределено. Тип значения: «Число». Разность между суммой услуги и распределенной по номенклатуре.
  • На доходы и расходы. Тип значения: «Булево». Признак того, что сумма услуги будет распределена на указанную в услуге статью дополнительных расходов. 

Дерево "Номенклатура"

  • Документ-основание. Документ, на который будут распределяться дополнительные расходы.
  • Номенклатура. Ссылается на справочник «Номенклатура». Номенклатура (товар) документа-основания.
  • Характеристика. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.   
  • Единица измерения. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Сумма дополнительных расходов. Тип значения: «Число». Сумма дополнительных расходов, которая распределена для данной номенклатурной позиции. 
  • Сумма дополнительных расходов без НДС. Тип значения: «Число». Сумма дополнительных расходов без учета НДС, которая распределена для данной номенклатурной позиции. 
  • ГТД. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для товаров, поступивших по импорту).
