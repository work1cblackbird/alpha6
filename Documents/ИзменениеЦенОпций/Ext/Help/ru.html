Документ "Изменение цен опций"

Назначение

Документ предназначен для документального фиксирования изменения цен компании на опции автомобилей. При проведении документа информация о ценах записывается в регистр сведений "Цены опций" по каждой опции автомобиля.

При проведении документ формирует хозяйственные операции:

  • Установка цен на опции. Хозяйственная операция сохраняет цены опций автомобилей, указанные для каждой опции на регистре сведений «Цены опций».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Поступление товаров», «Ввод остатков автомобилей».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Действует с. Тип значения: «Дата». Дата начала действия цены.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Комментарий. Произвольный комментарий к документу.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен.
  • Хоз операция. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Сумма документа.
  • Расчет цен от. Служебный реквизит, указывающий относительно чего будут рассчитываться цены на опции автомобилей.
  • Базовый тип цен. Ссылается на справочник «Типы цен». Тип цен, относительно которого производим расчет.
  • Документ основание. Ссылается на документ «Поступление автомобилей». Документ, на основании которого введен данный.
  • Процент наценки. Процент наценки на базовую цену.
  • Округлять до. Округлять расчетную цену до....

Табличная часть «Опции»

  • Модель. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Модели автомобилей». Модель автомобиля, для опции которой производиться назначение цены.
  • Вариант комплектации. Ссылается на справочник «Варианты комплектации». Вариант комплектации модели автомобиля, для опции которой производиться назначение цены.
  • Опция. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Опции автомобилей». Опция автомобиля, на которую производиться назначение цены. 
  • Цена. Реквизит является обязательным для заполнения. Новая цена опции автомобиля.
  • Цена базовая. Цена относительно которой производится расчет новой цены.
  • Процент наценки. Наценка на базовую цену для вычисления устанавливаемой цены.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Изменение цен на опции. Простая печатная форма, содержит список товаров с измененными ценами.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Ценообразование на автомобили --> Изменение цен опций".

Движения

  • «Цены опций». Цены на опции автомобилей.
