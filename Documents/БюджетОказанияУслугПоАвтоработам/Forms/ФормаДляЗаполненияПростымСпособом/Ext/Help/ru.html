Произвольная форма "Наивное прогнозирование"

Назначение

Произвольная форма документа «Бюджет оказания услуг по автоработам», предназначена для заполнения документа данными, используя метод наивного прогнозирования. Наивное прогнозирование заключается в получении простого арифметического среднего прогнозируемого ресурса (количество или сумма упр.) от фактических или запланированных данных предыдущих периодов.

Способ вызова

Форма вызывается из меню командной панели табличной части документа «Бюджет оказания услуг по автоработам» : "Наивное прогнозирование".

Принцип работы с формой

Для начала работы с формой необходимо определиться - арифметические средние от каких данных необходимо получить.  Предусмотрены четыре типа данных для анализа:

  1. По фактическим данным произведенных работ. Анализируется оборот по регистру  «Продажи».
  2. По ранее планируемым данным будущих работ. Анализируется оборот по регистру  «Бюджет оказания услуг по автоработам».
Выбор нужного типа осуществляется при помощи реквизита "Тип анализа".

Выбрать количество анализируемых периодов.  Данный параметр фактически определяет количество значений, от которых будет получено среднее арифметическое. Периоды отсчитываются назад от даты планирования документа «Бюджет оказания услуг по автоработам», из которого была вызвана данная форма. Вид периода (его размерность: месяц, квартал, год) определяется периодичностью выбранного в документе сценария планирования.  

Далее необходимо установить коэффициенты прогнозирования: Коэффициент роста и коэффициент сезонности. Эти коэффициенты определяются экспертным путем на основании предположения о существенном увеличении\уменьшении работ в планируемом периоде. Выражаются в натуральной величине (не в процентах).

Показатель планирования - определяет ресурс, который будет запланирован (количество или сумма упр.). Т.е. обороты какого ресурса будут рассчитаны. В дальнейшем через значение цены (соответствующей выбранному типу цен) будет найден второй ресурс.

Способ округления  - определяет режим округления количества работ.

Реквизиты

  • Смещение планирования. Число. Количество отступаемых периодов от планируемого периода.
  • Количество периодов. Число. Количество анализируемых периодов.
  • Количество сезонов. Число. Количество анализируемых сезонов.
  • Коэффициент роста. Число. Экспертная оценка роста объема работ в прогнозируемом периоде.
  • Коэффициент сезонности. Число. Экспертная оценка сезонной составляющей.
  • Показатель планирования. Булево. Вид прогнозируемого ресурса ("количество" или "сумма упр.")
  • Способ округления. Число. Определяет способ округления ресурса "количество".
  • Сценарий планирования. Справочник  «Сценарий планирования». Соответствует сценарию планирования документа  «Бюджет оказания услуг по автоработам», из которого была вызвана форма. В форме не редактируется. 
  • Тип анализа. Перечисление  «Типы анализа». Определяет вид анализируемых данных.  
  • Форма мастера. Произвольный. Используется при вызове данной формы из формы заполнения по документу основанию. Содержит ссылку на эту форму.
