Документ "Корректировка реализации автомобилей"

Назначение

Документ предназначен для регистрации исправленных первичных документов, полученных от поставщика. Например, при выявлении ошибки при оформлении первичных документов или в случае согласованного между продавцом и покупателем изменения стоимости ранее реализованных автомобилей. Имеет две хоз. операции "Исправление в первичных документах" и "Корректировка по согласованию сторон", используются при перегрузке в бухгалтерию.

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: Корректировка реализации автомобилей, «Реализация автомобилей».

Объекты, вводимые на основании: Корректировка реализации автомобилей, «Счет-фактура выданный».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Менеджер. Ссылается на справочник «Сотрудники». Сотрудник, заключивший сделку с контрагентом.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Основание. Реквизит является обязательным для заполнения. Документ на основании которого введена корректировка.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Курс валюты упр. Реквизит является обязательным для заполнения. Курс валюты управленческого учета.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Поставщик по документу реализации.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад по документу реализации.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор по которому ведутся взаиморасчеты с контрагентом.
  • Сумма. Сумма документа.

Табличная часть «Товары»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Реализованный автомобиль.  
  • VIN. VIN-номер автомобиля. 
  • Цена. Цена автомобиля в валюте документа.
  • Сумма. Сумма автомобиля. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена.
  • Сумма по документу реализации. Сумма автомобиля из документа реализации. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена. 
  • Разница сумма. Разница суммы автомобиля и суммы автомобиля, согласно документа реализации.
  • % НДС. Ставка НДС. Ссылается на справочник «Ставки НДС».
  • % НДС по документу реализации. Ставка НДС, согласно документа реализации. Ссылается на справочник «Ставки НДС».
  • НДС. Сумма НДС автомобиля.
  • НДС по документу реализации. Сумма НДС автомобиля, согласно документа реализации.
  • Разница сумм НДС. Разница сумм НДС и суммы НДС, согласно документа реализации.
  • Всего. Итоговая сумма по строке документа со всеми налогами.
  • Всего по документу реализации. Итоговая сумма по строке документа со всеми налогами, согласно документа реализации.
  • Разница всего. Разница итоговых сумм по строке документа со всеми налогами и суммы, согласно документа реализации.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма документа. Используется для ввода и редактирования документов данного типа.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Отгрузка автомобилей --> Корректировка реализации автомобилей".
