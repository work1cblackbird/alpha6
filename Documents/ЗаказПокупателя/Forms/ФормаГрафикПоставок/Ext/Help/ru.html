Произвольная форма "Форма график поставок"

Назначение

Произвольная форма документа «Заказ покупателя», предназначена для формирования графика поступлений товарной строки заказа по нескольким поставкам, у которых различные даты поставок и/или различные поставщики.

Способ вызова

Вызывается при двойном нажатии, по полю колонки "График поставок", в товарной строке документа «Заказ покупателя».

Индикация «Состояние заказа»

  • Заказано.  Тип значения: «Число». Остаток заказа на текущий момент, по данному «Заказу покупателя».
  • В резерве.  Тип значения: «Число».  Остаток в резерве на текущий момент, по данному «Заказу покупателя».
  • Распределено.  Тип значения: «Число».  Остаток распределения на текущий момент, по данному «Заказу покупателя».

Таблица «График поставок»

  • Количество. Тип значения: «Число». Количество заказанной номенклатуры, по графику.
  • Дата поставки. Тип значения: «Дата». Плановая дата поступления товара.
  • Контрагент. Ссылается на справочник «Контрагенты». Поставщик товара.
  • Дата поступления. Тип значения: «Дата». Фактическая дата поступления заказанного товара в полном объеме.
  • Информация. Тип значения: «Строка». Дополнительная информация.

Таблица «Список поставщиков»

  • Поставщик. Ссылается на справочник «Контрагенты». Поставщик, который хоть раз поставлял данный товар.
  • Срок поставки. Тип значения: «Число». Срок поставки указанный в регистре «Сведения компании» для данного поставщика.
  • Заказано. Тип значения: «Число». Количество заказанного товара, по данному поставщику на текущий момент.
  • Распределено. Тип значения: «Число». Количество уже распределенного товара, по данному поставщику на текущий момент.
