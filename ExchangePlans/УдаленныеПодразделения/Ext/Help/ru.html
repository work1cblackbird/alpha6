План обмена "Удаленные подразделения"

Назначение

Предназначен для обмена данными между территориально распределенными информационными системами на основе информационных баз 1С:Предприятия 8.1. Например, для организации работы главного офиса, филиалов и складов предприятия в единой информационной базе.

План обмена содержит информацию об узлах, которые могут участвовать в обмене данными, определяет способ доставки и приема сообщений обмена. Для каждого узла можно задать код, наименование и необходимый перечень реквизитов, описывающих узел.

Механизм распределенных информационных баз предназначен для обмена данными только с идентичными конфигурациями 1С:Предприятия 8.1 и жестко регламентирует структуру создаваемой системы.

Параметры

Длина кода: 2

Длина наименования: 30

Реквизиты

  • Префикс. Тип  значения "Строка". Префикс узла распределенной информационной базы.
  • Профили архива чеков. Тип значения «Хранилище значений». Профили архива чеков.
  • Префикс. Тип  значения "Строка". Префикс узла распределенной информационной базы.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора узлов обмена.
  • Форма узла. Форма, используемая для заполнения обязательных реквизитов распределенного узла обмена.

Способ вызова

Вызывается из меню  "Справочники--> Структура компании -->Удалённые подразделения".

Дополнительная информация

Распределенная система должна иметь древовидную структуру, в которой существует корневой узел и определено отношение "главный - подчиненный" для каждой пары связанных узлов. Изменения конфигурации распространяются в сообщениях обмена вместе с изменениями данных и передаются от главного узла к подчиненным. Изменения данных передаются между любыми связанными узлами.
Внесение изменений в конфигурацию возможно только в одном (корневом) узле распределенной системы.
