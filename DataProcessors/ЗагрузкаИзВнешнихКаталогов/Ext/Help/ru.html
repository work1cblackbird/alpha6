Обработки "Загрузка из внешних каталогов"

Назначение

Обработка "Загрузка из внешних каталогов" позволяет загружать данные в заказ-наряд из внешних программ калькуляции. Поддерживается загрузка из следующих программ калькуляции:

  • Silver Dat II
  • AudaTex
  • AutoData
  • Автонормы
  • AutoSoft

Способ вызова

Вызывается из документа "Заказ-наряд", закладка "Работы", меню "Заполнение --> Заполнить из внешних каталогов"

Дополнительная информация

При вызове из документов сначала открывается диалог выбора файла загрузки, после выбора файла производиться загрузка данных в табличную часть обработки и открывается основная форма обработки, где можно указать те строчки, которые хочется загрузить и непосредственно загрузить их в документ.

Также в основной форме есть возможность снова выбрать файл для загрузки, тогда будет произведена повторная загрузка и ее данные добавлены к данным предыдущей загрузки.

Для форматов AutoSoft и Автонормы при добавлении строк в документ создаются(если таковых нет) и подставляются элементы справочника НормоЧасы соответствующие данным добавляемых строк.
