Обработка "Автоматическое планирование ресурсов"

Назначение

Обработка предназначена для автоматического распределения работ по ресурсам с учетом их загрузки. Работает совместно с обработкой Запись на ремонт, самостоятельная работа не предусмотрена. Реализованы два режима работы:

  • Упрощенный  - расчет происходит без участия пользователя
  • Экспертный - перед расчетом у пользователя запрашиваются параметры расчета
