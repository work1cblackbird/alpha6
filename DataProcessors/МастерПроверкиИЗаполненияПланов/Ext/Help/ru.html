Обработка "Мастер проверки и заполнения планов"

Назначение

Автоматизирует полный цикл планирования (заполнение планов и фактов).

После заполнения реквизитов шапки: подразделение, периодичность, значение конкретного периода. При нажатии на кнопку "Заполнить", на закладке "Виды планов" заполняется табличная часть виды планов в соответствии с имеющимися видами планов, по выбранному подразделению. Светофор указывает:

  • зеленый есть все необходимые документы;
  • желтый частично;
  • красный нет документов;
  • серый не подлежит заполнению по отсрочке.

После нахождения видов планов переходим к закладке "Заполнение планов". Тут интерактивно создаются и проводятся документы План. При нажатии кнопки "Автоформирование" заполняется табличная часть заполнение планов. Светофор указывает:

  • зеленый документ есть и проведен;
  • желтый документ сохранен, не проведен;
  • красный нет документов.

После чего последовательным нажатием кнопки "Документ". Последовательно открываются документы для корректировки и проведения.

Закладка "Заполнение фактов" Функционально, такая же, как и "Заполнение планов", только соответственно для документов Факт.
