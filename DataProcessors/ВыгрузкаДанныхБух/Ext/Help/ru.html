Обработка "Выгрузка данных в бухгалтерию"

Назначение

Служит для выгрузки документов, с последующей их загрузкой в бухгалтерию предприятия версии 4.5, 1.5, 1.6, 2.0 и 3.0.

Расположение файла выгрузки следует указать в поле "Каталог формирования файла-выгрузки".

Обработка выгружает документы за указанный период, если период выгрузки не задан, выгружаются документы за всю работу текущей базы.

 

  • Флажок «Выгружать данные в XML-документ». Установлен по умолчанию. Выгрузка в текстовый формат не поддерживается.
  • Флажок «Показывать выгруженные объекты». После выгрузки объектов в файл будет выведено дерево выгруженных объектов распределённых по структуре метаданных на закладке «Выгруженные объекты». При обработке документов, так же выгружаются справочники, указанные в документах, для последующей загрузки. Выгрузка справочников происходит по ссылкам.
  • Флажок "Выгружать документы по движениям партий". При установленном флажке происходит выгрузка по ссылкам документов участвующих в движениях партий выгружаемых документов. Документы по ссылкам выгружаются независимо от попадания их в выбранный период, выбранную организацию или выбранные виды документов. Этот флажок имеет смысл при перегрузке в конфигурацию "Бухгалтерия предприятия" версии 1.6, 2.0 и 3.0 с включенной возможностью "Списание с учетом партий" (обработка загрузки не ниже версии 1.2.2).
  • Флажок «Учет ГТД». Включенный флажок показывает, что выгрузка документов происходит с переносом номеров ГТД.
  • Поле ввода «Дата ввода остатков в бухгалтерии». Эта дата имеет очень важное значение в случае если перегрузка в бухгалтерию началась не за все существующие периоды, а со ввода остатков например на начала года. в таком случае эту дату нужно заполнять датой ввода остатков, и в этом случае партии, которые были раньше, чем введен ввод остатков в бухгалтерию, выгружены не будут. И, таким образом, не будет задвоения остатков в результате загрузки партиеобразующих документов с датой меньшей чем дата ввода остатков. Если не использовался ввод остатков в бухгалтерии то «Дата ввода остатков в бухгалтерии» заполнять не следует.

Закладка "Виды документов"

На данной закладке можно выбрать виды документов для выгрузки. Документы выгружаются по заданной на форме организации, если таковая не указана, выгружаются документы всех организаций, текущей базы. Также можно выгрузить документы по конкретным контрагентам. Если список контрагентов не задан, то выгрузка документов происходит по всем контрагентам в соответствии с  настройками, установленными в форме. Документы, у которых нет реквизита "Контрагент", выгружаются в соответствии с настройками, установленными в форме обработки. В таблице документов, необходимо указать виды документов предназначенных для выгрузки.

Закладка "Объекты"

На закладке "Объекты" можно указать несколько отдельных объектов для выгрузки: документов или элементов справочников.

Закладка "Виды справочников"

При использовании обработки есть возможность выгрузить справочники. На закладке «Виды справочников» находится табличная часть, содержащая следующие виды справочников: «Банки», «Банковские счета», «Валюты», «Контрагенты», «СтатьиДДС», «Номенклатура», «Договоры взаиморасчетов», «Сотрудники». Для выгрузки справочника его необходимо выбрать на данной закладке в табличной части. Есть возможность выгружать информацию о конкретной организации. Нужную организацию можно выбрать в графе «Организация». При выгрузке справочник «Банковские счета» можно выгружать только вместе со справочником «Банки» и «Валюта». Если ни один из справочников выбран не будет, то производится остановка выгрузки и выдается сообщение об ошибке. При выгрузке справочник «Договоры взаиморасчетов» выгружаются в соответствии с выбранной организаций на закладке, выгрузку данного справочника можно производить только совместно со справочниками «Контрагенты» и «Валюты».

  • Флажок «Выгружать справочники для нахождения соответствия». Если флажок установлен, то выгрузка производится для определения соответствий справочников и проверки их наличия в «Таблице соответствий» при загрузке в Бухгалтерию предприятия 1.6, 2.0 и 3.0. В этом режиме загрузка прекращается в момент нахождения соответствия загружаемому справочнику в Таблице соответствий. Если прямое соответствие не найдено, то поиск продолжается - по наименованию и другим реквизитам - и производится попытка установки соответствия. Если и по реквизитам объект не найден, то загрузка также прекращается.
  • Флажок «Выгружать справочники по ссылкам». Если флажок установлен, то производится выгрузка выбранных справочников и справочников, взаимосвязанных с ними.

Закладка "Выгруженные объекты"

На данной закладке отображаются выгруженные объекты: документы и элементы справочников.

Дополнительная информация

При выгрузке документов в бухгалтерию предприятия на 1С версии 7.7 следует пользоваться файлом выгрузки с форматом "*.xml". Для бухгалтерии предприятия на 1С версии 8.0, 8.1, 8.2, 8.3 можно пользоваться любым форматом выгрузки.
