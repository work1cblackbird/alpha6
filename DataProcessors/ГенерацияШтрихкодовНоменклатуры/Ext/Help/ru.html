Обработка "Генерация штрихкодов номенклатуры"

Назначение

Обработка предназначена для формирования штрихкодов номенклатуры и записи в регистр Штрихкоды.

Заполнение товарами для формирования штрихкодов возможно по отбору и из документов. Заполняется только теми товарами, для которых нет записанных штрихкодов в базе.

Для заполнения по отбору необходимо выбрать пункт меню "Подбор" - "По отбору". Открывается окно с настройкой  отбора и по нажатию на кнопку "ОК" таблица заполняется по установленным настройкам отбора.

Для заполнения из документов необходимо выбрать пункт меню "Подбор" - "Из документов" и выбрать нужный тип документа. 

Типы документов, из которых можно заполнить:

  • Заказ покупателя.
  • Заказ поставщику.
  • Поступление товаров.
  • Реализация товаров.

При нажатии на кнопку "Сформировать" происходит формирование новых штрихкодов для строк с незаполненным полем "Штрихкод". Также можно сформировать штрихкод для конкретной строки, нажав кнопку выбора в поле "Штрихкод" или при помощи сканера штрихкодов.
Можно включить/отключить раскраску строк таблицы по кнопке . 

Есть возможность очистить таблицу с выбранными товарами или удалить строки с незаполненными штрихкодами по кнопке "Удалить" меню таблицы.

Для записи сформированных штрихкодов необходимо нажать кнопку "Записать штрихкоды". Перед записью происходит проверка на заполненность поля "Штрихкод". Если есть строки с незаполненным штрихкодом, то выводится сообщение что не заполнено поле и запись не происходит. При записи производится проверка на уникальность штрихкодов и при нахождении неуникального штрихкода выводится сообщение и такой штрихкод не записывается в базу.
