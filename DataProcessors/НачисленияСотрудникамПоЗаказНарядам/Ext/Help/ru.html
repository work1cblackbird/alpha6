Обработка "Начисления сотрудникам по заказ-нарядам"

Назначение

    Обработка предназначена для редактирования параметров и суммы начислений сотрудникам по заказ-нарядам. Данные изменения можно зафиксировать в обработке через запись изменений в документ и обновление записей по регистру накопления  «Начисления сотрудникам по заказ-нарядам». Обработка доступна только при наличии включенной опции «Использовать расчеты начислений сотрудникам в заказ-нарядах». Также у пользователя должно быть настроено право «Уровень доступа к начислениям сотрудникам в заказ-нарядах» в режиме «Просмотр» или «Редактирование».
