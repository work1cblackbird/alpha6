Обработка "Помощник закрытия договора аренды"

Назначение


Используется при закрытии договора аренды в случае использования депозита. Присутствует возможность перенести депозит на другой договор аренды или зачесть на задолженности клиента, которые могут возникнуть при использовании дополнительных услуг, получении штрафов, увеличении сроков аренды. При помощи обработки можно оформить возврат остатка депозита клиенту удобным для него способом - перевести остаток депозита на расчетный счет, получить остаток наличными из кассы предприятия или оформить возврат денежных средств по кассе ККМ. После настройки необходимых действий необходимо нажать команду «Выполнить распределение» для ввода необходимых документов, при окончании работы созданные документы отобразятся на отдельной странице формы. При полном распределении депозита для договора будет установлено состояние «Закрыт».


Относится к подсистеме «Аренда автомобилей»


При открытии обработки происходит проверка использования депозита по договору аренды, наличия остатка депозита, возврат автомобилей клиентом и выставление актов об оказании услуг.


Для отражения переноса депозита на другие документы используется документ «Корректировка долга» . При возврате денежных средств на банковский счет создается документ «Платежное поручение», при получении денег из кассы организации — документ «Расходный кассовый ордер», возврате по ККМ — документ «Чек на оплату» с хоз. операцией «Возврат по чеку на оплату».


Созданные в процессе работы документы корректировки взаиморасчетов и возврата денежных средств прикрепляются к счету на оплату депозита аренды.
