Обработка "Выгрузка данных в файл"

Обработка Выгрузка данных в файл предназначена для выгрузки данных справочника или табличной части документа во внешний файл. При запуске обработки Выгрузка данных в файл появляется окно Помощника выгрузки данных.

Обработка относится к разделу Администрирование.

Порядок работы пользователя с обработкой:

  1. В поле Тип объекта пользователь указывает тип выгружаемых данных («Справочник» или «Документ»).
  2. В поле Объект указывается объект выгрузки (название документа или справочника).
  3. На следующей форме Помощника выгрузки данных указывается настройка выгрузки данных (выбирается из справочника Настройки выгрузки/загрузки данных) и файл, в который будут выгружаться данные.
  4. После нажатия кнопки Далее происходит выгрузка данных.
