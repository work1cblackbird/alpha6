Форма "Настройка параметров учета"

Назначение

Форма предназначена для заполнения основных параметров программы (констант). После начала работы с системой дальнейшее изменение значений некоторых констант может быть запрещено для поддержки целостности учета.

Значения констант

  • Оперативное проведение документов. Ссылается на перечисление  «Использование оперативного проведения документов». Настройка позволяет управлять режимом оперативного проведения. Варианты значений: 1. "Разрешено оперативное проведение". При этом поведение документа такое же как и предусмотрено платформой 1С, то есть при оперативном проведении дата документа будет сдвигаться на текущий момент времени. 2. "Оперативное проведение для новых и непроведенных" - оперативно будут проводиться непроведенные документы и новые. Т.е. если документ на этот день был записан, а его проводят, тогда время будет смещено на текущее. 3. "Оперативное проведение только для новых" - оперативно будут проводиться только новые документы (которые записываются первый раз).
  • Валюта регламентированного учета организаций. Ссылается на справочник «Валюты». Национальная валюта страны, в которой будет происходить выгрузка в бухгалтерию и проводиться выписка большинства первичных документов. Должна иметь единичный курс. Валюта устанавливается только один раз при первом запуске конфигурации. Менять регламентированную валюту в дальнейшем не рекомендуется.
  • Валюта управленческого учета организаций. Ссылается на справочник «Валюты». Валюта, в которой будет строиться управленческий баланс, а так же оцениваться деятельность предприятия. Управленческий учет может вестись в произвольной валюте. Все активы и пассивы компании имеют обязательную оценку в этой валюте. В валюте управленческого учета может быть получен Управленческий баланс.
  • Основная единица измерения массы. Ссылается на справочник «Классификатор единиц измерения». Обычно килограмм. Используется для расчета массовых показателей в товарных документах. 
  • Основная единица измерения количества. Ссылается на справочник «Классификатор единиц измерения». Обычно штука. Используется для помощи при вводе новых товаров. 
  • Основная единица размерности ордерного склада. Ссылается на справочник «Классификатор единиц измерения». Обычно метр. Используется для автоматической подстановки в качестве единицы размерности ячеистого склада и его ячеек.
  • Партия товаров отрицательных остатков. Задает партию товаров для партии товаров отрицательных остатков.

  • Режим проведения по партиям. Ссылается на перечисление "РежимыПроведенияПартий". При выбранном режиме "Партионные движения есть и выполняется допроведение" будут учитываться партионные движения и можно выполнять допроведения документов по партиям. Список документов, которые необходимо допровести по партиям хранится в регистре сведений «Допроведения по партиям». Режим "Партионные движения отсутствуют" означает, что учет партионных движений на предприятии не ведется. Режим "Партионные движения есть (допроведения НЕ выполняются)" означает, что на предприятии ведется партионный учет, но допроводить документы по партиям нельзя.

  • Валюта накопительных сумм. Ссылается на справочник «Валюты». Валюта, в которой будут накапливаться суммы для расчета накопительных скидок.
  • Операция с чеками после закрытия смены. Эта настройка определяет действия с чеками после закрытия смены. При выбранном значении "Архивировать" - для чеков будет установлен признак «Архивный». При значении "Удалить" - чеки после обработки будут удалены. Значение "Пометить на удаление" означает, что для чеков будет установлен признак «Архивный», и чеки будут помечены на удаление.
  • Формат вывода количества. Эта настройка определяет формат вывода количества в печатных формах документов (за исключением тех случаев, где формат явно определен).
  • Формат вывода суммы. Эта настройка определяет формат вывода суммы (и цены) в печатных формах документов (за исключением тех случаев, где формат явно определен).
  • Выводить полный номер документа. Если установлено значение истина, то в печатных формах документов его номер будет выводиться как он указан в самом документе. В противном случае, номер будет выводиться без начального префикса и лидирующих нулей в номере.
  • Префикс весового штрих-кода. Эта настройка определяет префикс внутреннего штрихкода весового товара. По умолчанию принимает значение 22.
  • Префикс штучного штрих-кода. Эта настройка определяет префикс внутреннего штрихкода штучного товара. По умолчанию принимает значение 20.
  • Количество одновременно загружаемых прайс-листов. Эта настройка указывает какое количество прайс-листов может одновременно загружатся при автоматическом обновлении данных прайс-листов в базе.
  • Путь к архиватору 7-Zip. Эта настройка указывает путь к исполняемому файлу программы 7-Zip для выполнения команды распаковки архива в том случае, когда загружаемые прайс-листы упакованы в архив.

Способ вызова

Вызывается из меню "Администрирование --> Основные параметры системы --> Настройка параметров учета"
