Обработки "Управление лицензированием"

Назначение

Предназначена для первоначальной настройки и управления лицензированием конфигурации. Управляет настройкой подключения к серверу лицензирования, активацией ключей защиты и пакетов лицензий. Позволяет получить информацию о текущем состоянии системы лицензирования. Позволяет установить сервер лицензирования конфигураций на локальный компьютер. Автоматически запускается при первом запуске конфигурации и в случае невозможности работы с текущим указанным сервером лицензирования.

Относится к подсистеме: Управление лицензированием.

Формы

  • Состояние системы защиты. Основная форма. Отображает текущий статус системы защиты конфигурации. Из этой формой вызываются все функции управления лицензированием конфигурации.
  • Форма настройки сервера защиты. Мастер настройки подключения к серверу лицензирования. Позволяет выбрать способ подключения к серверу защиты, указать адрес сервера и код доступа к ключу защиты.
  • Форма активации ключа защиты. Мастер активации программного ключа защиты. Позволяет выбрать сервер, на котором будет активирован ключ, указать способ активации ключа (автоматический, через файловый запрос или по телефону), заполнить данные для активации ключа и активировать ключ защиты.
  • Форма активации лицензий. Мастер активации лицензий. Предназначен для активации пакета лицензий. Активация лицензий используется для изменения лицензионных ограничений, заданных в уже существующем ключе защиты. Мастер позволяет активировать дополнительные лицензии через интернет, либо через файловый запрос.
  • Информация о параметрах лицензирования. Предназначена для просмотра текущих параметров лицензирования. Параметры можно сохранить в текстовый файл.
