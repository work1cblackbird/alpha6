Обработки "Проверка номенклатуры по типу"

Назначение

Обработка "Проверка номенклатуры по типу" предназначена для исправления ошибок в логической структуре данных, связанных с типизацией номенклатуры, а именно: использование единиц измерений, использование характеристик, параметры штрихкодирования и проверка для учета цен. Данная обработка предназначена для специалистов по внедрению и опытных пользователей.
