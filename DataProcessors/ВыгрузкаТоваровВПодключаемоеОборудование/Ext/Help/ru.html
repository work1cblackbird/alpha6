Выгрузка товаров в подключаемое оборудование

Рабочее место предназначено для выгрузки данных о товарах в подключаемое оборудование.

  • Для заполнения списка товаров используется кнопка Заполнить выгружаемые товары. В список заполняются только те товары, на которые назначены штрихкоды.
  • Для выгрузки товаров в оборудование используется кнопка Выгрузить товары в оборудование.
  • Для очистки товаров в оборудовании используется кнопка Очистить товары в оборудовании.
