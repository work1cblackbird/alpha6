Обработка "Настройка поля расширенной информации"

Назначение

Используется для настройки работы поля расширенной информации. Поле расширенной информации дает возможность пользователю видеть нужную ему информацию об объекте в форме списка объекта.

С помощью формы настройки можно вывести значение любого поля объекта.
