Обработка "Остатки прослеживаемых товаров"

Назначение

Обработка "Остатки прослеживаемых товаров" предназначена для проведения инвентаризации прослеживаемых товаров и автомобилей на складах по организации на указанную дату. А также формирование документов "Уведомление об остатках прослеживаемых товаров" для выгрузки информации о прослеживаемых товаров на складах для получения регистрационного номера партии (РНПТ). 

ВАЖНО! Выполнить инвентаризацию на складе с прослеживаемым товаром можно только один раз.
