Обработка "Автозаполнение ячеистого склада"

Назначение

Обработка предназначена для автоматического создания ячеек выбранного склада. Создавать ячейки можно только для склада с видом «Ордерный и ячеистый». Реализованы две стратегии заполнения:

  • Рассчитывать количество ячеек, исходя из указанных для них размеров.
  • Рассчитывать размеры ячеек, исходя из указанного количества и размеров склада.

Способ вызова

Вызывается из формы элемента справочника «Склады (места хранения) компании»  по кнопке "Заполнить ячейки".
