Обработка "Печать этикеток и ценников"

Назначение


Сервис предназначен для формирования списка номенклатуры, сертификатов, карт, для которой нужно распечатать этикетки и ценники для товаров, подарочных сертификатов, дисконтных карт и регистрационных карт, и, собственно, для их печати по заранее созданным шаблонам. При необходимости доступно редактирование макета шаблона через СКД (Система Компоновки Данных).

Способ вызова


Шаблоны можно создать в списке, который открывается по ссылке Розница - См. также - Шаблоны этикеток и ценников. Шаблоны можно группировать – команда Создать новую группу и искать (с помощью сервиса Расширенный поиск (Alt+F), который запускается из командной панели списка, Пользователь может выбрать нужные шаблоны, задавая поиск по значениям различных реквизитов.

Сервис можно запустить по ссылке Розница – Сервис - Печать этикеток и ценников, а также из меню Печать документов Авансовый отчет, Ввод остатков товаров, Заказ кодов маркировки, Кассовая смена, Изменение цен компании и контрагентов, Комплектация, Перемещение товаров, Переоценка, Поступление товаров, Приходный складской ордер, Разукомплектация, Расходный складской ордер, справочника Номенклатура.

Изменение режима печати

В меню Установить командной панели формы можно выбрать различные режимы и в зависимости от выбранного режима изменяется состав колонок в табличной части.

Заполнение основных реквизитов

Дата и тип цены для формирования этикеток и ценников указывается в шапке сервиса.
Если тип цены не указан, используется тип цены продажи пользователя по умолчанию.

Заполнение табличной части

Если сервис Печать этикеток и ценников был запущен из документов, то табличная часть автоматически заполняется номенклатурой исходных документов.

Если сервис был запущен вручную, то можно настроить отбор товаров, нажав кнопку Показать отбор в командной панели табличной части. После настройки параметров отбора можно заполнить табличную часть по ссылке Заполнить по отбору из меню Заполнить. Также можно заполнить табличную часть с помощью стандартного сервиса Подбор, который запускается по кнопке Подбор.

Редактирование табличной части

Выделять строки табличной части можно стандартными способами с помощью мыши и комбинаций клавиш на клавиатуре.
Для выделенных строк из меню Установить для выделенных строк можно выбрать следующие пункты:

· Количество ценников - вводится фиксированное значение;

· Шаблон ценников - выбирается из списка всех шаблонов с типом Этикетка (ценник);

· Количество этикеток - вводится фиксированное значение;

· Шаблон этикеток - выбирается из списка всех шаблонов с типом Этикетка (ценник);

· Количество по умолчанию - устанавливается количество по умолчанию, для ценников - 1, для этикеток - количество, равное остатку на складе или количеству из исходного документа, в зависимости от того, откуда был запущен сервис;

· Новый штрихкод EAN-13, если штрихкод не заполнен - генерирует новый внутренний штрихкод в формате EAN-13. 

Можно выбрать шаблон или изменить количество для конкретной позиции в соответствующей ячейке табличной части.

В первой колонке табличной части отмечаются позиции, для которых будет выполнена печать. Все поля с шаблонами и количествами в этих позициях должны быть заполнены. Отмечать позиции можно вручную и с помощью сервисных кнопок командной панели.

Печать этикеток и ценников

По нажатию кнопки Печать в командной панели сервисной формы для отмеченных позиций генерируются печатные формы ценников и этикеток. Перед отправкой на принтер при необходимости пользователь может внести исправления в созданные печатные формы.
