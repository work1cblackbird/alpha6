Справочник "Типы маркировки"

Назначение

Хранит список товарных групп маркируемого товара. Используется для формирования запроса кодов маркировки в ГИС МТ.

Относится к подсистеме: Розница

Реквизиты

  • Код маркировки. Код типа маркировки для формирования тега 1162 в чеке при пробитии соответствующего товара. Если код маркировки равен 0 (нулю), то в чек не будет выведена информация о маркировке. 
  • Используется Единый метод создания документов. При включенном флаге тип маркировки отображается в документах связанных с маркировкой товаров.
