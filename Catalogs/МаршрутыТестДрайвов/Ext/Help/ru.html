Справочник «Маршруты тест-драйвов»

Назначение

Справочник возможные маршруты проведения тест-драйвов автомобилей.

Параметры

Длина кода: 10, тип кода: Строка.

Длина наименования: 100.

Не иерархический.

Реквизиты

  • Дата создания. Тип значения: «Дата». Дата создания маршрута.
  • Автор. Ссылается на справочник «Пользователи». Пользователь, создавший запись о маршруте.
  • Актуальность. Тип значения: «Булево». Актуальность маршрута для проведения тест-драйвов.
  • Комментарий. Тип значения: «Строка». Дополнительная информация о маршруте.
  • Карта. Тип значения: «Строка». Ссылка на карту маршрута. Это может быть путь к файлу с картой маршрута или ссылка на web-ресурс с картами. 
  • Продолжительность. Тип значения: «Число». Продолжительность маршрута в минутах.
  • Протяженность. Тип значения: «Число». Протяженность маршрута в километрах.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается при выборе маршрута в документе «Тест-драйв».
