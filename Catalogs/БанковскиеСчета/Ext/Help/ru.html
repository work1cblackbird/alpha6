Справочник "Банковские счета"

Назначение

В справочнике "Банковские счета" хранятся банковские счета всех юридических и физических лиц: собственных и сторонних.

Список банковских счетов контрагента, собственной организации или подразделения определяется теми банковскими счетами, которые подчинены соответствующему элементу справочника «Контрагенты и контакты», «Организации» или «Подразделения компании».

Для каждого контрагента или собственной организации можно выбрать т.н. «основной» банковский счет, который будет подставляться по умолчанию в платежные документы.

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 150.

Не иерархический.

Владельцы справочника: «Контрагенты и контакты», «Организации»,  «Подразделения компании».

Реквизиты

  • Номер счета. Реквизит является уникальным и обязательным для заполнения. Тип значения: «Строка». Номер расчетного счета.
  • Банк. Реквизит является обязательным для заполнения. Ссылается на справочник «Банки». Банк, в котором открыт счет.
  • Валюта счета. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта, в которой хранятся денежные средства.
  • Вид счета. Реквизит является обязательным для заполнения. Тип значения: «Строка». Может принимать следующие значения: расчетный, депозитный, ссудный, иной. Активный при выборе Организации в качестве типа владельца.
  • Дата открытия. Тип значения: «Дата». Дата открытия счета.
  • Номер и дата разрешения. Тип значения: «Строка». Номер и дата разрешения открытия счета.
  • Банк для расчетов. Ссылается на справочник «Банки». Банк, в случае непрямых расчетов. Используется при оплате по QR-коду, если реквизит заполнен, то при оплате в поле "Наименование получателя платежа" выводится информация: ФИО/Наименование, р/с Номер счета в Наименование банка, Город банка.
  • Подразделение. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, к которому "привязан" счет. Если счет принадлежит контрагенту, то поле неактивно.
  • Месяц прописью. Тип значения: «Булево». Указывать в дате документа месяц прописью.
  • Сумма без копеек. Тип значения: «Булево». Указывать сумму без копеек, если она в целых рублях.
  • Дата закрытия. Тип значения: «Дата». Дата закрытия счета.
  • Владелец. Ссылается на справочники «Контрагенты и контакты», «Организации», «Подразделения компании». Контрагент (организация или подразделение компании), который является владельцем счета.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов. На форме есть кнопка которая открывает обработку «Клиент-Банк», которая служит для выгрузки указанных платежных поручений в файл, за указанный период, и по отмеченным счетам и загрузки платёжных поручений из файла импорта созданного программой клиент банка.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов. Указывается информация, характеризующая банковский счет (банк, номер счета, вид счета, валюта денежных средств, банк - корреспондент для непрямых расчетов), а также дополнительная информация, необходимая для корректного заполнения платежных документов по выбранному счету. На форме присутствует кнопка

Способ вызова

Вызывается из меню "Справочники --> Идентификационные --> Банковские счета".

Дополнительная информация

В нижнем левом углу форм объекта есть ссылка, при нажатии на которую открывается форма элемента владельца счета.
