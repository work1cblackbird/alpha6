Справочник "Договоры взаиморасчетов"

Назначение

Содержит перечень заключенных договоров с контрагентами. Для каждого контрагента может быть оформлено сколько угодно договоров. Удобство в ведении договоров состоит в том, что все проведенные операции с контрагентом можно разнести по разным договорам, в зависимости от смысла конкретной операции.

Например, можно отнести к одному договору операции приема товаров на комиссию, а к другому — покупки товаров. Тогда в каждом договоре можно увидеть не общие итоги по расчетам с данным контрагентом, а текущее состояние каждой конкретной операции.

Договор определяет валюту взаиморасчетов по данному договору и в случае ведения взаиморасчетов с одним контрагентом в разных валютах необходимо ввести соответственное количество договоров.

Относится к подсистеме "Финансы".

Реквизиты

  • Валюта. Реквизит является обязательным для заполнения. Валюта взаиморасчетов по договору. Значение не может быть изменено, если по договору проведен хотя бы один документ.
  • Кураторы. Сотрудник, курирующий указанного контрагента.
  • Отмена контроля суммы кредита. Установка контроля проверки превышения максимальной суммы кредита. При установке данного флага контроль проверки не производится. Для нового договора взаиморасчетов данный флаг возводится по умолчанию. 
  • Срок оплаты задолженности. Срок оплаты задолженности (в днях).
  • Процент комиссионного вознаграждения. Процент комиссионного вознаграждения для договоров комиссии. Поле активно при выборе вида договора "Комиссия".
  • Сумма отчета.  Способ заполнения сумм в отчете комитенту по сумме поступления или по сумме продажи комиссионного товара. ЛОЖЬ='По суммам продажи'; ИСТИНА='По сумме поставки'.
  • Статья ДДС.  Статья движения денежных средств, по которой производится платеж.
  • Расчеты в условных единицах. При установленном флаге оплата по валютному договору производится в условных единицах. Используется только при синхронизации.
  • Профиль скидки. Ссылается на справочник «Карточки». Профиль скидки договора. 
  • Единица измерения авторабот в печатных формах.  Используется для заполнения единицы измерения авторабот в печатных формах.
  • Способ зачета авансов. Способ зачета авансов для заполнения соответствующего реквизита при вводе документов оплат по умолчанию. 
  • Учет взаиморасчетов. Определяет способ ведения учета взаиморасчетов с контрагентом.
