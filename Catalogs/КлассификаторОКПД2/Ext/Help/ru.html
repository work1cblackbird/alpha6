Справочник "Классификатор продукции по видам экономической деятельности"

Назначение

В справочнике кодов продукции по видам экономической деятельности  хранятся коды, которые используются при заполнении реквизитов номенклатуры.

Данный справочник может быть заполнен с помощью подбора из классификатора ОКПД2, в котором хранятся все коды продукции по видам экономической деятельности, а также вручную пользователем поэлементно. При использовании подбора из классификатора кодов существенно снижается трудоемкость ввода информации, ввод информации о коде сводится к выбору из классификатора, соответственно сводится к минимуму количество ошибок при вводе.

Параметры

Длина кода: 12, тип кода: Строка.

Длина наименования: 150.

Не иерархический.

Реквизиты

  • Полное наименование. Тип значения: «Строка». Полное наименование кода ОКПД2. 

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется создания новых элементов справочника.
  • Форма подбора из классификатора. Используется для подбора элементов из классификатора кодов ОКПД2. 

Печатные формы

  • Классификатор ОКПД2. Используется не для печати, а для подбора новых кодов.

Способ вызова

Вызывается из меню "Справочники --> Классификаторы --> Классификатор ОКПД2".
