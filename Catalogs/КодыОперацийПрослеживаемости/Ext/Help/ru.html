Справочник "Коды операций прослеживаемости"

Назначение

Справочник "Коды операций прослеживаемости" предназначен для ведения списка доступных операций и их кодов в соответствии с приказом ФНС о порядке заполнения отчета об операциях с товарами, подлежащими прослеживаемости. Относится к разделу Прослеживаемые товары.

Реквизиты

  • Описание. Тип значения: «Строка». Описание вида операции. 
  • Условие. Тип значения: «Строка». Условие применения операции.  

Формы

  • Форма списка. Основная форма выбора группы. Используется для выбора групп. 
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
