Справочник "Хоз. операции"

Назначение

Содержит список используемых хозяйственных операций типового решения. Хозяйственная операция является дополнительной аналитикой, в разрезе которой можно анализировать движения всех регистров учетных блоков базы данных, порождаемые документами. Каждый вид документа может выполнять одну или несколько (по выбору пользователя) хозяйственных операций. Каждая хозяйственная операция характеризует свой уникальный набор проводок. 

Параметры

Длина кода: 6, тип кода:Строка.

Длина наименования: 50.

Иерархический

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Данный справочник не вызывается интерактивно.

Однако во многих документах может быть выбрана необходимая хозяйственная операция из ограниченного списка по кнопке "Операция"

Дополнительная информация

Является "общим" реквизитом практически всех документов и регистров.
