Справочник "Группы товаров оборудования"

Назначение

Справочник Группы товаров оборудования предназначен для объединения нескольких экземпляров оборудования одинакового функционального назначения, требующих загрузки товарного состава. (Так, например, несколько ККМ, обслуживающих один торговый зал, будут использовать одну товарную группу «Товары торгового зала»).

Относится к подсистеме Управление ассортиментом раздела Склад.

Реквизиты

  • Режим заполнения. Режим заполнения группы товарами. Значение выбирается из списка:
  • Все товары. Группа заполняется любыми товарами, ограничения отсутствуют.
  • Только весовые товары. В группу можно добавлять только весовые товары.
  • Только штучные товары. В группу можно добавлять только штучные товары.
  • Только товары перечисленных типов. В таблице указываются товарные категории, группу можно формировать только из товаров этих категорий.
  • Только товары, входящие в группы. В таблице указываются группы номенклатуры, группу можно формировать только из товаров этой номенклатурной группы.
  • Автоматически заполнять новым товаром. Если флажок установлен, то после ввода нового номенклатурного элемента этот элемент автоматически добавляется в текущую группу товаров оборудования (если этот элемент соответствует параметрам заполнения группы).

Табличная часть «Ограничения заполнения группы»

  • Тип или группа номенклатуры. Реквизит является уникальным и обязательным для заполнения при режимах заполнения Только товары перечисленных типов и Только товары, входящие в группы. Тип или группа номенклатуры, выгружаемые в оборудование.

 

Вызываемый из меню кнопки дополнительных функций регистр сведений Товары в оборудовании хранит сведения о товарах, которые выгружаются в оборудование и хранятся в логических ячейках памяти экземпляров оборудования. Информация в каждом элементе справочника задает значение одной ячейки PLU оборудования, использующего определенную товарную группу.

Перечень команд, исполняемых при использовании меню кнопки Заполнение формы регистра сведений Товары в оборудовании:

  • Очистить все товары. Удаляет все строки таблицы.
  • Вся номенклатура. Заполняет таблицу всеми товарами номенклатуры.
  • Заполнить по группе номенклатуры. Открывает справочник Номенклатура. Заполняет таблицу всеми товарами выбранной группы.
  • Заполнить складскими остатками. Открывает справочник Склады компании. Заполняет таблицу товарами, имеющимися на выбранных складах.
  • Заполнить остатками по группе номенклатуры. Открывает поочередно справочники Номенклатура и Склады компании. Заполняет товарами выбранной группы, имеющимися на выбранных складах.
  • Заполнить по документу. Заполняет таблицу товарами из выбранного документа поступления или перемещения товаров.
  • Заполнить по ассортименту подразделения. Открывает справочник Подразделения компании. Заполняет таблицу ассортиментом товаров выбранного подразделения компании.
