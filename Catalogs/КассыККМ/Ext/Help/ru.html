Справочник "Кассы ККМ"

Назначение

Справочник Кассы ККМ содержит список контрольно-кассовых машин компании. Используется при оформлении документов розничной продажи. В качестве неоперационных касс компании
могут использоваться денежные ящики контрольно-кассовых машин (ККМ) при розничной торговле. Может быть вызван из разделов Розница, Предприятие. 

Денежные средства, полученные при продаже товара в розницу (в торговом зале, удаленной торговой точки, розничном магазине и т.д.), поступают в кассу ККМ.

Реквизиты

  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник Подразделения компании. Подразделение компании, к которому относится ККМ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник Организации. Организация, к которой относится ККМ.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник Валюты. Задает валюту работы ККМ и наличности в денежном ящике. Валюта должна соответствовать национальной валюте страны.
