Справочник "Характеристики номенклатуры"

Назначение

В конфигурации можно вести количественный учет материальных ценностей в разрезе различных характеристик, серий и инвентарных номеров номенклатуры. Назначение и физический смысл характеристик назначаются в соответствующем типе номенклатуры. Характеристики назначаются только для тех позиций, у которых установлен флаг "Вести учет по характеристикам".

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 150.

Не иерархический.

Владельцы справочника: «Номенклатура», «Типы номенклатуры».

Реквизиты

  • Серийный номер. Тип значения: «Строка». Серийный номер.
  • Качество. Ссылается на справочник «Категории качества». Категория качества характеристики номенклатуры.
  • Дата изготовления. Тип значения: «Дата». Дата изготовления серии номенклатуры для серийного учета.
  • Срок годности. Тип значения: «Дата». Срок годности .
  • Номер и дата сертификата. Ссылается на справочник «Подтверждающие документы». Номер и дата сертификата.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка (сроки и сертификаты). Форма списка и выбора при использовании ограничения по сроку годности и сертификату (задается в справочнике «Типы номенклатуры»).
  • Форма элемента (сроки и сертификаты). Форма элемента при использовании ограничения по сроку годности и сертификату (задается в справочнике «Типы номенклатуры»).
  • Форма списка (серийный номер и свойства). Форма списка и выбора при использовании ограничения по серийному номеру и свойствам (задается в справочнике «Типы номенклатуры»).
  • Форма элемента (серийный номер и свойства). Форма элемента при использовании ограничения по серийному номеру и свойствам (задается в справочнике «Типы номенклатуры»).
  • Форма списка (свойства). Форма списка и выбора при использовании ограничения по свойствам (задается в справочнике «Типы номенклатуры»).
  • Форма элемента (свойства). Форма элемента при использовании ограничения по свойствам (задается в справочнике «Типы номенклатуры»).
  • Форма списка (качество). Форма списка и выбора при использовании ограничения по качеству (задается в справочнике «Типы номенклатуры»).
  • Форма элемента (качество). Форма элемента при использовании ограничения по качеству (задается в справочнике «Типы номенклатуры»).

Способ вызова

Вызывается из контекстного меню справочников "Номенклатура", "Типы номенклатуры" вызывается по кнопке "Перейти"

Дополнительная информация

Для параметризации характеристик номенклатуры обычно используется механизм свойств, позволяющий определить состав свойств номенклатурных характеристик для каждого типа номенклатуры. При выборе характеристики номенклатуры в документы может использоваться фильтрация по значениям свойств.
