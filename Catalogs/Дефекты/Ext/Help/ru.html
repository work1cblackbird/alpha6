Справочник "Дефекты"

Назначение

В справочник заносится информация о типичных дефектах различных узлов автомобиля. Используется документе «Заявка на хранение шин» в поле "примечание".

Параметры

Длина кода: 10, тип кода:Строка.

Длина наименования: 100.

Иерархический

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.  

Дополнительная информация

Загрузка дефектов выполняется из макета при первом запуске системы. Для загрузки используется макет справочника "Дефекты".
