В справочнике кодов товарной номенклатуры внешнеэкономической деятельности хранятся коды, которые используются при заполнении реквизитов номенклатуры.

Данный справочник может быть заполнен с помощью подбора из классификатора кодов ТН ВЭД, в котором хранятся все коды товарной номенклатуры внешнеэкономической деятельности, а также вручную пользователем поэлементно. При использовании подбора из классификатора кодов существенно снижается трудоемкость ввода информации, ввод информации о коде сводится к выбору из классификатора, соответственно сводится к минимуму количество ошибок в вводе.

Коды, которые уже были выбраны из классификатора подбора, в списке отмечаются синим цветом.

Открывается в меню Администрирование -> Основные параметры системы в пункте Базовые классификаторы.

В справочнике выводится:

  • Наименование кода - наименование кода товарной номенклатуры внешнеэкономической деятельности;
  • Полное наименование - более подробное описание кода товарной номенклатуры внешнеэкономической деятельности;
  • Код - непосредственно код товарной номенклатуры внешнеэкономической деятельности;
  • Сырьевой товар - признак принадлежности товара к сырьевым.

 

Ввод вручную

В некоторых случаях может возникнуть необходимость ввести какой-либо код вручную.

  • Для того чтобы добавить новый код в список, нажмите  Создать.
  • Не рекомендуется вводить вручную коды, имеющиеся в классификаторе, так как ручной ввод увеличивает возможность ошибок.
  • Программа предлагает выбрать код из классификатора. Подтвердите свой выбор, в ответ на вопрос программы нажав кнопку Да.
  • Для того чтобы ввести код вручную, нажмите  Нет. Заполните необходимые поля. Нажмите Записать или Записать и закрыть для сохранения текущего кода.
