Справочник "Категории качества"

Назначение

Справочник "Категории качества" предназначен для классификации номенклатуры (и ее характеристик) по степеням качества. Справочник состоит из 5-ти предопределенных элементов, определяющих степень качества. Элемент с наименьшим кодом соответствует наивысшей степени качества. Добавлять элементы в справочник нельзя. Разрешено только изменение наименования элементов.

Предопределенные элементы

  • Категория качества № 1 - 5. Пять предопределенных категорий качества.

Параметры

Длина кода: 9, тип кода: Число.

Длина наименования: 50.

Не иерархический.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.

Способ вызова

Вызывается из меню "Справочники --> Классификаторы --> Категории качества"
