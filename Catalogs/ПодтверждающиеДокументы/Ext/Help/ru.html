Справочник "Подтверждающие документы"

Назначение

Справочник содержит список подтверждающих документов для некоторых объектов конфигурации (контрагентов, номенклатуры, организаций,  сотрудников). Под такими документами понимают: для физических лиц, прежде всего паспорт, удостоверение, военный билет, доверенность, справка; для юридических лиц - лицензия; для номенклатуры - сертификат.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Не иерархический.

Владельцы справочника: «Контрагенты и контакты», «Номенклатура», «Организации», «Сотрудники».

Вводится на основании: «ЗаказПоставщику», «ПоступлениеТоваров».  

 

Реквизиты

  • Вид документа. Ссылается на перечисление «Виды документов». Реквизит является обязательным для заполнения. Вид подтверждающего документа.
  • Серия. Тип значения: «Строка». Серия документа.
  • Номер. Тип значения: «Строка». Реквизит является обязательным для заполнения. Номер документа.
  • Дата выдачи. Тип значения: «Дата». Реквизит является обязательным для заполнения. Дата выдачи подтверждающего документа.
  • Дата оформления. Тип значения: «Дата». Реквизит является обязательным для заполнения. Дата оформления документа.
  • Номер бланка. Тип значения: «Строка». Номер бланка подтверждающего документа. 
  • Дата окончания. Тип значения: «Дата». Дата окончания срока действия подтверждающего документа.
  • Кем выдан. Тип значения: «Строка». Структура выдавшая подтверждающий документ.
  • Комментарий. Тип значения: «Строка». Дополнительная информация о документе.
  • Текущий. Тип значения: «Булево». Признак того, что документ актуален в данный момент.
  • Устаревший. Тип значения: «Булево». Признак того, что документ устарел и не используется.
  • Показывать устаревшие. Тип значения: «Булево». Если флаг установлен, в списке будут отображаться подтверждающие документ, у которых стоит признак устаревшего документа.
  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Контрагент, на получение материальных ценностей у которого выдана доверенность.

Табличная часть «Товары»

  • Номенклатура. Ссылается на справочник «Номенклатура». Реквизит является обязательным для заполнения. Товар, подлежащий получению по доверенности (если вид подтверждающего документа - "доверенность").
  • Количество. Тип значения: «Число». Реквизит является обязательным для заполнения. Количество номенклатуры.
  • Единица. Ссылается на справочник «Единицы измерения». Реквизит является обязательным для заполнения. Единица измерения номенклатуры.
  • К. Тип значения: «Число». Реквизит является обязательным для заполнения. Коэффициент.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры. Назначение и физический смысл характеристик назначаются в соответствующем типе номенклатуры. Характеристики назначаются только для тех позиций, у которых установлен флаг "Вести учет по характеристикам".

Печатные формы

  • М-2. Печать регламентированных форм доверенности (Регламентированные формы М-2 и М-2а по учету материальных ценностей).
  • Договор купли-продажи автомобиля. Печать договора купли-продажи автомобиля, являющегося владельцем элемента справочника.
  • Регистрационный знак. Печать регистрационного знака "Транзит".
  • ПТС. Печать Паспорта транспортного средства. Заполняется внутренняя и оборотная сторона ПТС в зависимости от выбранных секций в форме выбора секций. При печати поля секции заполняются по данным документа "Реализация автомобилей": владелец автомобиля (контрагент из документа), фактический адрес владельца, дата,  документ на право собственности (договор с контрагентом из документа). При выборе варианта печати для комиссионного автомобиля доступна печать "секции дилера", которая заполняется по данным документа "Поступление автомобилей": владелец автомобиля (организация из документа), юридический адрес организации, дата, документ на право собственности (договор с контрагентом из документа).

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов. В форме отображается список подтверждающих документов. Для того чтобы в списке отображались устаревшие документы необходимо установить флаг "Показывать устаревшие" находящийся на верхней панели формы. Также список можно отсортировать по виду документа указанному в поле "Вид документа" также находящийся на верхней панели формы.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.  Форма содержит несколько закладок, каждая из которых устанавливает соответствующие параметры подтверждающего документа.
    • Закладка "Данные". Позволяет заполнить данные о документе: номер бланка, кем выдан, конечный срок действия документа и признак актуальности документа.
    • Закладка "Товары". Активна при выбранном виде документа "Доверенность". Содержит табличную часть "Товары", в которую вносятся товары для выдачи на них доверенности.

Способ вызова

Вызывается из меню "Справочники --> Статьи и свойства --> Подтверждающие документы".
