Справочник "Графики работы"

Назначение

Справочник Графики работы используется для хранения графиков работ всего предприятия. В справочнике хранятся временные интервалы работы по количеству строк табличной части шаблона графика. Количество строк табличной части шаблона графика определяется периодичностью графика. Одна строка табличной части определяет временные показатели одного дня. При необходимости разбития на временные интервалы одного дня (строки табличной части), используется реквизит табличной части Смены. Может быть фиксированным, в этом случае элемент справочника выступает в роли шаблона графика. Детальный, заполняется на каждую конкретную дату с использованием данных шаблона.

Справочник Графики работы относится к подсистеме Планирование раздела Предприятие.

Реквизиты

  • Периодичность. Реквизит является обязательным для заполнения. Может принимать следующие значения:
    • неделя,
    • декада,
    • месяц,
    • квартал,
    • год,
    • по циклам.
  • Дата начала. Дата начала действия графика, актуально только для графика с периодичностью декада и по циклам. 
  • Ручное заполнение. При ручном заполнении, график заполняется на каждую конкретную дату с использованием данных шаблона графика. Иначе график заполняется только как шаблон графика, расчет на конкретную дату производится по нему.
  • Календарь. Указывает производственный календарь, на основании которого могут заполняться праздничные и предпраздничные дни.
  • Учет праздников. Отражает способ учета праздничных дней:
    • Замещать - из состава рабочих и выходных дней будут исключаться дни, выпадающие на праздники, 
    • Сдвигать - рабочие и выходные дни будут сдвинуты с учетом праздников.

Табличная часть «Шаблон графика»

  • День. Номер дня в ряде планируемых дней. Конкретная дата для произвольного графика определяется как дата начала действия графика плюс номер дня, для графиков с другой периодичностью номером дня.
  • Рабочий. Установка/снятие флажка указывает рабочий день или нет.
  • Смена. Ссылается на справочник Смены. Используется для создания временных интервалов для одного дня.
  • Начало. Время суток, от которого начинается действие графика.
  • Окончание. Время суток, в котором заканчивается действие графика.
  • Часы. Продолжительность графика в текущем дне.
  • Комментарий. Любая дополнительная информация.

Табличная часть «Ручное заполнение» 

Вкладка отображается, только если справочник заполняется вручную.

  • Дата. Дата дня графика.
  • Вид дня. Ссылается на перечисление Вид дня. Может принимать следующие значения: рабочий, предпраздничный, праздник, выходной.
  • Смена. Ссылается на справочник Смены. Служит для хранения временных интервалов рабочих смен.
  • Вид интервала. Ссылается на справочник Вид интервалов. Служит для указания вида интервала: рабочее время, обеденный перерыв, технический перерыв и т.д.
  • Начало. Начало рабочего времени.
  • Окончание. Конец рабочего времени.
  • Часы.  Продолжительность рабочего времени, разница между началом рабочего времени и окончанием рабочего времени.
  • Комментарий. Любая дополнительная информация.

Примеры заполнения графиков работы

Сотрудники работают посменно, 1 день через 2, с 9-00 до 20-00. Дополнительных выходных (праздники) нет.

Создаем новый элемент справочника. Периодичность - по циклам (будет определяться количеством строк (дней) в шаблоне). Указываем начальную дату. Например, с 1 января 2007 года сотрудник А работает 1, 4, 7 и т.д., сотрудник Б - 2, 5, 8 и т.д., сотрудник В - остальные дни. Тогда начальная дата для сотрудника А будет - 01.01.2007. Шаблон графика содержит 3 строчки: первая - рабочий день, вторая и третья - выходные (вид дня). Если в смене оговорен обеденный перерыв, то необходимо рабочий день разбить на интервалы (работа-обед-работа). Это нужно сделать с помощью задания смены в строке шаблона. Вводим смену (элемент справочника Смены), добавляем в него интервалы: 1) рабочее время 9-00/12-00, 2) обеденный перерыв 14-00/15-00, 3) рабочее время 15-00/20-00. И в текущем графике заполняем этой сменой все строки шаблона (возможно и не все дни будут биться на интервалы смены, например выходные дни, поскольку обеда там не будет). При получении календарного графика смены учитываются таким образом: если заполнена смена в строке шаблона, то интервалы будут взяты из элемента смены, если смена не заполнена, то из элемента графика работы. Расчет осуществляется по номеру строки относительно начальной даты графика.

  1. Ручное заполнение.

Сотрудники работают без смен, рабочие дни Пн-Пт, выходные Сб и Вс, с 9-00 до 18-00. Праздники учитываются. Иметь возможность введения дополнительных выходных/рабочих дней.

Создали элемент справочника. Используя кнопку Ручное заполнение активируем ручное заполнение. Периодичность - Неделя (период повторения выходных). Для периодичностей Неделя, Месяц, Квартал и Год заполнение начальной даты не требуется (год, например, всегда начинается с 1 января). Если необходима разбивка дней шаблона на интервалы, то добавляем смену (см. фиксированный график). Автозаполнение позволяет автоматически заполнить шаблон графика. Так называемый шаблон графика готов. Для не ручного заполнения ввод графика на этом календарь завершен (т. е. шаблон для графика и будет графиком). Для ручного графика нужно ещё заполнить детальный календарный план, который находится на странице Ручное заполнение.

На странице Ручное заполнение есть возможность автозаполнения календарного графика - кнопки Очистить, Заполнить по основным параметрам и Вытеснить по графику. При нажатии на заполнение открывается форма заполнения календарного графика, в которой можно указать период заполнения. Устанавливаем периодичность Год (если заполняем в начале года на год). Далее нажимаем Применить, календарный график готов.

Полученный детальный график можно отредактировать вручную по каждому дню или интервалу (если идет разбивка дня по интервалам смены). Имеется возможность очистки графика за указанный период.
