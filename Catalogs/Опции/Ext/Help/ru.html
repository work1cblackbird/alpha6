Справочник "Опции"

Назначение

В справочник заносится информация о всех опциях, устанавливаемых на автомобилях. Справочник "Опции" может содержать так же наборы, состоящие из отдельных элементов опций.

Параметры

Длина кода: 8, тип кода:Строка.

Длина наименования: 80.

Иерархический

Реквизиты

  • Полное наименование. Полное наименование, например, для печати.
  • Номер по каталогу. Код производителя опции. Если право "Номер по каталогу уникальный" установлено в ИСТИНА, то данный реквизит является обязательным для заполнения.
  • Ставка НДС. Реквизит является обязательным для заполнения. Ссылается на справочник «Ставки НДС». Ставка НДС по умолчанию для оформления документов.
  • Валюта учета. Ссылается на справочник «Валюты». Используется для ведения дополнительного учета валютной себестоимости и цен.
  • Комментарий. Любая дополнительная информация. 
  • Признак набора. Признак, указывающий что данная опция представляет собой совокупность нескольких других элементов. 

Табличная часть «Состав набора»

  • Опция. Ссылается на справочник «Опции автомобилей». Опции, входящая в состав набора. Для набора данный реквизит является обязательным для заполнения.
  • Количество. Количество опций в наборе. Для набора данный реквизит является обязательным для заполнения. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.  

Способ вызова

Вызывается из меню "Справочники --> Автомобили --> Опции автомобилей".
