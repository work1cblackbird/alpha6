Справочник "Виды аренды"

Назначение


Справочник предназначен для разделения оказываемых услуг аренды автомобилей по видам, каждый из которых имеет особенности по ценообразованию, порядку контроля выдачи и приемки автомобиля, выставления документов клиенту.


Относится к подсистеме «Аренда автомобилей»


Тип аренды — платный, бесплатный. При выборе платного вида аренды доступна настройка тарификации на соответствующей странице, также становится доступна страница «Дополнительные услуги». Для бесплатного типа аренды невозможен расчет стоимости аренды, выставление счетов и актов по услугам аренды, за исключением счета на депозит.


Основное


Тип цен — используемый в документах аренды тип цен. Также из типа цен определяется, включен ли НДС в цену или НДС необходимо начислять сверху. Поле обязательно для заполнения. Выбор расчетного типа цен не влияет на цены аренды - в таком случае необходимо вводить документ «Изменение цен аренды» и указывать цены для моделей и вариантов комплектаций. Тип цен виляет на стоимость дополнительных услуг в документах аренды, для дополнительных услуг расчетные цены будут работать.


Валюта — валюта, в которой устанавливаются цены аренды в документе «Изменение цен аренды». В документах аренды при расчете стоимости происходит перерасчет из валюты вида аренды в валюту документа. Поле обязательно для заполнения.


График аренды — график, использующийся при расчете аренды. Поле обязательно для заполнения.


Обязательно указание водителей — при установке признака документы «Договор аренды» и «Изменение договора аренды» не будут проводиться, если не указан хотя бы один водитель в соответствующей табличной части.


Не облагается НДС — если признак указан, то в документах аренды используется ставка Без НДС.


Тарификация


Изменения настроек на странице доступно только для платных видов аренды.


Используется тарификация по периодам: Минута, Час, Сутки, Неделя, Месяц, Квартал, Год — набор признаков расчета стоимости аренды в зависимости от входящих в период интервалов. При установке признака становятся доступными поля в документе «Изменение цен аренды» и поля Цена и Количество на закладке «Тарификация» в документах аренды. Установка одного из признаков обязательна для платного вида аренды


Услуга для выставления счетов на оплату - услуга из справочника «Номенклатура», которая будет использоваться при выставлении счета на оплату на основании договора аренды. Для платного вида аренды заполнение услуги обязательно.


Услуга для выставления актов об оказании услуг - услуга из справочника «Номенклатура», которая будет использоваться при выставлении акта об оказании услуг на основании договора аренды. Для платного вида аренды заполнение услуги обязательно.


Депозит


Использовать депозит — признак получения обеспечительного платежа от клиента при передаче автомобиля в аренду. При установке признака становятся доступными поля депозита при изменении цен аренды и поля в заявке на аренды и договоре аренды. Признак может указываться как для платного вида аренды, так и бесплатного.


Услуга — услуга из справочника «Номенклатура», которая будет использоваться при выставлении счета на оплату по депозиту аренды. При установке признака «Использовать депозит» указание услуги обязательно.


Выдача и приемка


Контролировать выдачу и приемку — при включении признака в документе «Договор аренды» появляются команды «Выдать автомобиль» и «Принять автомобиль», а в табличной части «Автомобили» отображается текущий статус автомобиля. При использовании команд фиксируется дата выдачи и приемки автомобиля.


Шаблон анкеты при выдаче автомобиля — при заполнении шаблона для отражения выдачи автомобиля в системе необходим ввод документа «Дефектовочная ведомость» с хоз. операцией «Выдача в аренду». Документ вводится при помощи команды «Выдать автомобиль» в форме договора аренды.


Шаблон анкеты при приемке автомобиля - при заполнении шаблона для отражения приемки автомобиля в системе необходим ввод документа «Дефектовочная ведомость» с хоз. операцией «Приемка из аренды». Документ вводится при помощи команды «Принять автомобиль» в форме договора аренды.


Дополнительные услуги


Содержит список услуг из справочника «Номенклатура», список доступен для быстрого выбора в документах аренды. Ввод доступен только для платных видов аренды. При переключении типа на бесплатный, таблица очищается.
