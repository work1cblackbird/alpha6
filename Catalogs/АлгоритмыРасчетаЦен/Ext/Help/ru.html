Справочник "Алгоритмы расчета цен"

Назначение

Справочник алгоритмов расчета цен хранит информацию о возможных алгоритмах в виде кода 1С для расчета цены.

При создании нового или открытии существующего элемента справочника "Алгоритмы расчета цен", для пользователя автоматически создается 5 стандартных шаблонов, которые можно использовать при написании алгоритма, выбрав один из предложенных шаблонов в одноименном меню.

В нижней части формы элемента пользователю предоставляется возможность проверить корректность работы построенного алгоритма расчета цен. Для этого необходимо подставить данные в поля "Цена в прайс листе", "Цена покупки", "Цена номенклатуры" и нажать на кнопку "Проверить". После выполнения расчета, сформированную по алгоритму цену можно увидеть в поле "Итог".

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 100.

Иерархический.

Реквизиты

  • Алгоритм. Реквизит является обязательным для заполнения. Код 1С для расчета цены.

Формы

  • Форма списка. Основная форма списка, выбора и редактирования. Используется для отображения списка, выбора элементов и редактирования справочника.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Маркетинг --> Ценообразование --> Алгоритмы расчета цен".
