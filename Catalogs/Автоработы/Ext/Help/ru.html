Справочник "Автоработы"

Назначение

В справочник авторабот заносится информация обо всех работах, выполняемых на предприятии, включаемых в документы заказ-наряд.

Параметры

Длина кода: 10, тип кода: Строка.

Длина наименования: 100.

Иерархический.

Реквизиты

  • № по каталогу. Номер автоработы по каталогу.
  • Наименование полное. Полное наименование автоработы.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Базовая номенклатура автоработы для задания ставок налогов.
  • Время выполнения. Время выполнение работы, которое будет подставлено по умолчанию, если для модели и варианта комплектации не указано время выполнения в табличной части "Нормы времени".
  • Способ расчета стоимости работы. Используется при расчете суммы автоработы в документах «Заявка на ремонт» и «Заказ-наряд».

Табличная часть «Нормы времени»

  • Модель. Ссылается на справочник «Модели». Модель, для которой указывается норма времени выполнения автоработы.
  • Вариант комплектации. Ссылается на справочник «Варианты комплектации». Вариант комплектации, для которой указывается норма времени выполнения автоработы. Если вариант комплектации не задан, то данная норма времени действует для всех вариантов комплектации для модели указанной в строке.
  • Время выполнения. Тип значения «Число». Норма времени выполнения автоработы в часах.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.
  • Форма выбора группы. Основная форма выбора группы. Используется для выбора групп.
  • Форма подбора. Используется для подбора работ в документы «Заявка на ремонт» и «Заказ-наряд». В нижней части формы расположен список подбираемых работ. При выборе товара в этом списке он переносится в документ. Вызывается из форм документов «Заявка на ремонт» и «Заказ-наряд» по кнопке "Подбор" на закладке "Работы".
    В верхней части формы расположено дерево групп справочника работ.
    В группе Запрашивать есть два флажка: Цена и Количество. Если оба флажка сняты, то при выборе работы, она (работа) добавится в документ с количеством=1 и ценой которая установлена для класса автомобиля, выбранного в документе. Если хотя бы один из флажков установлен, то при выборе работы будет открыта форма Ввод цены и количества в которой можно ввести количество и цену работы в нормочасах. 
  • Ввод количества и цены. Дополнительная форма для ввода количества и цены подбираемой работы.

Способ вызова

Вызывается из меню "Справочники --> Автосервис --> Автоработы".
