Справочник "Скрипты разговоров"

Назначение

Используется для помощи оператору при разговоре. 

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 50.

Реквизиты

  • Содержание. Тип значения: «ХранилищеЗначений». Описание содержания разговора.  

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
