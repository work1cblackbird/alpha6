Справочник "Виды ремонта"

Назначение

Справочник "Виды ремонта" предназначен для классификации работ по заказ-нарядам и контроль оплаты по выполненным работам и примененным деталям для выполнения работ по заказ-нарядам.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 100.

Не иерархический.

Реквизиты

  • Тип ремонта. Реквизит является обязательным для заполнения. Типы ремонта:
    • Платный ремонт. Используется при выполнении работ для автомобиля контрагента с последующей оплатой за их выполнение.
    • Бесплатный ремонт. Выполнение работ на автомобиле контрагента в рамках гарантийного ремонта, сервисного обслуживания, когда контрагент не должен их оплачивать.
    • Комплектация автомобилей. Предназначен для комплектования автомобиля, который находится на складе компании, дополнительным оборудованием. Доступен при наличии лицензии "Автосалон".
    • Сборка автомобиля. В результате установки дополнительного оборудования собран автомобиль. При этом собранный автомобиль будет оприходован на склад компании.  Доступен при наличии лицензии "Автосалон".

Формы

  • Форма списка. Основная форма списка, выбора и редактирования. Используется для отображения списка, выбора элементов и редактирования справочника.

Способ вызова

Вызывается из меню "Справочники --> Автосервис --> Виды ремонта".
