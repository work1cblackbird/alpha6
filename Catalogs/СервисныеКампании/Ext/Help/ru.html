Справочник "Сервисные кампании"

Назначение

Справочник предназначен для учета мероприятий, которые необходимо выполнить по инициативе производителя автомобилей. Используется при получении от производителя списка VIN-номеров автомобилей, на которых необходимо выполнить определенные действия по ремонту или устранению каких-либо недостатков.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 100.

Иерархический.

Реквизиты

  • Модель. Реквизит является обязательным для заполнения. Ссылается на справочник «Модели автомобилей». Модель автомобиля, по которому осуществляется сервисная кампания.
  • Вариант комплектации. Ссылается на справочник «Варианты комплектации». Вариант комплектации автомобиля, по которому осуществляется сервисная кампания. 
  • Дата начала, Дата окончания. Период действия сервисной кампании.
  • Вид ремонта. Ссылается на справочник «Виды ремонта». Вид ремонта, который подставляется в документы при выборе сервисной кампании.
  • Комментарий. Тип значения: «Строка». Дополнительная информация.

Табличная часть «Автомобили»

  • VIN. Тип значения: «Строка». VIN автомобиля, по которому назначена сервисная кампания.
  • Дата выполнения. Дата, когда для автомобиля  была выполнена сервисная кампания. Если значение не заполнено, то сервисная
    кампания не проводилась.
  • Документ выполнения. Ссылается на документ «Заказ-наряд». Заказ-наряд, по которому была проведена сервисная кампания. Если значение не заполнено, то сервисная кампания не проводилась или была проведена другой организацией.

Табличная часть «Работы/детали»

  • Авторабота. «Авторабота» или «Номенклатура», которая подставляется в документ при выборе сервисной кампании в соответствующую табличную часть.
  • Количество. Тип значения: «Количество». Количество авторабот или деталей.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка и выбора сервисной кампании.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.

Способ вызова

Вызывается из меню "Справочники --> Автосервис --> Сервисные кампании".
