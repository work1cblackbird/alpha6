Справочник «Программы страхования»

Назначение

Справочник содержит возможные программы страхования. Программа страхования – это гарантированный договором страхования комплекс услуг, предлагаемых страхователю или застрахованному при наступлении страхового случая.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Не иерархический.

Реквизиты

  • Агентская услуга. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Агентская услуга, которая используется в реализации агентских услуг.
  • Страховая компания. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Страховая компания, используемая в программе страхования.
  • Вид страхования. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды страхования». Вид страхования программы страхования.
  • Вознаграждение. Тип значения: «Число». деньги, которые оставляет себе автосалон при продаже страховки.
  • Описание. Тип значения: «Строка». Описание программы страхования.
  • Актуальность. Тип значения: «Булево». Актуальность данной программы страхования.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Взаимоотношения с клиентами --> Программы страхования".
