Справочник "Типы номенклатуры"

Назначение

Хранит список типов номенклатуры. Тип номенклатуры - основной признак номенклатурной позиции, определяющий её назначение и поведение в конфигурации, способ формирования и декодирования штрихкода.

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 50.

Иерархический.

  • Штучный. Штучный товар.
  • Весовой. Весовой товар.
  • Размерный. Размерный товар.
  • Серийный. Серийный товар.
  • Услуга. Услуга.
  • Тара. Тара.

Реквизиты

  • Вид номенклатуры. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды номенклатуры». Вид номенклатуры. Может принимать следующие значения:  товар, услуга, тара, комплект, набор, прочие активы.
  • Основная базовая единица измерения. Ссылается на справочник «Классификатор единиц измерения». Основная единица измерения, подставляемая по умолчанию для номенклатурных позиций данного типа.
  • Основной тип эксплуатации. Ссылается на справочник «Типы эксплуатации». Основной тип эксплуатации для номенклатуры, являющейся прочим активом.
  • Весовой. Тип значения: «Булево». Признак весового товара. Используется для определения вида штрихкодирования.
  • ЕНВД/УСН. Тип значения: «Булево». Признак ведения учета ЕНВД/УСН. Используется при перегрузке в 1С:Бухгалтерию. 
  • Использование единиц измерения. Реквизит является обязательным для заполнения. Тип значения: «Число». 1 - Единицы измерения подчинены данному типу номенклатуры, 2 - Единицы подчинены номенклатурным позициям.
  • Использование характеристик. Реквизит является обязательным для заполнения. Тип значения: «Число». 1 - Характеристики подчинены данному типу номенклатуры, 2 - Характеристики подчинены номенклатурным позициям, 3 (и любой другой) - Учет по характеристикам не ведется.
  • Ограничение данных характеристик. Ссылается на перечисление «Ограничения данных характеристики». Задает ограничение на использование характеристик для данной номенклатуры или ее типа. 
  • Уникальность серийного номера. Тип значения: «Булево». Уникальность серийного номера.
  • Порядок списания характеристик. Ссылается на перечисление «Режимы автоматического списания характеристик». Указывает способ списания номенклатуры по характеристикам. 
  • Использовать разрешительный режим при розничной продаже. Тип значения: «Булево».  Признак того, что перед розничной продажей товара с кодом маркировки необходимо производить проверку кода маркировки в разрешительном режиме.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.
  • Форма настройки характеристик. Форма используется для задания порядка списания характеристик номенклатуры и видов свойств характеристик.

Способ вызова

Вызывается из меню "Справочники --> Номенклатурные --> Типы номенклатуры".
