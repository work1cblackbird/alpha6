Справочник "Дополнительные коды Audapad Web"

Назначение

Используется для хранения перечня дополнительных кодов из программы AutoPad Web, по которым необходимо вести расчет в загружаемых «Заказ - нарядах».

Предопределенные группы

  • Товары и работы. По умолчанию помещаются доп. коды с типом распределения на товары и работы.
  • Товары. По умолчанию помещаются доп. коды с типом распределения на товары.
  • Работы. По умолчанию помещаются доп. коды с типом распределения на работы.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Иерархический (иерархия групп и элементов).

Реквизиты

  • Код по классификатору. Реквизит является обязательным для заполнения. Тип значения: «Строка». Номер доп. кода в системе AutoPad Web.
  • Тип начисления. Реквизит является обязательным для заполнения. Ссылается на перечисление "Типы начислений доп. кодов". Тип начисление доп кода.
  • Тип распределения. Реквизит является обязательным для заполнения. Ссылается на перечисление "Типы распределения доп. кодов". Тип распределения доп. кода.
  • Комментарий. Тип значения: «Строка». Любая дополнительная информация.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Сервис --> Все операции --> Справочники --> Дополнительные коды Audapad Web".
