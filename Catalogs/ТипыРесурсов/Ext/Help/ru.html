Справочник "Типы ресурсов"

Назначение

Справочник Типы ресурсов хранит список доступных для планирования типов авторабот и автомобилей. Используется в автоматическом планировании. Относится к подсистеме Автосервис раздела Автоматическое планирование.

 

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Иерархический

 

Форма элемента справочника содержит закладки:

  • Доступные типы авторабот . На вкладке заполняются типы авторабот, которые будут доступны для автоматического планирования. Также, присутствует возможность автоматического заполнения.
  • Доступные типы автомобилей. Содержит сведения о типах автомобилей, которые будут доступны для автоматического планирования.
