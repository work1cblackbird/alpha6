Справочник «Кредитные программы»

Назначение

Справочник предназначен для предоставления информации о возможных программах автокредитования. Используется в документе "Рабочий лист кредитного отдела".

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Не иерархический.

Реквизиты

  • Банк. Реквизит является обязательным для заполнения. Ссылается на справочник «Банки». Указывает на банк кредитования.
  • Годовая ставка. Реквизит является обязательным для заполнения. Тип значения: «Число». Реквизит является обязательным для заполнения. Годовая ставка кредита.
  • Ссылка на калькулятор. Тип значения: «Строка». Строка, содержащая ссылку на калькулятор.
  • Описание. Тип значения: «Строка». Описание программы кредитования.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Взаимоотношения с клиентами --> Кредитные программы".
