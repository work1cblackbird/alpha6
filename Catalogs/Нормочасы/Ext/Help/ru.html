Справочник "Нормочасы"

Назначение

Справочник предназначен для хранения данных о стоимости нормочасов выполнения работ. Стоимости единицы времени выполнения различных работ могут быть различные и их стоимость может задаваться в различных валютах.

Предопределенные элементы

  • Рубль. Нормочасы авторабот.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 20.

Не иерархический.

Реквизиты

  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта измерения цены нормочаса.
  • Цена. Реквизит является обязательным для заполнения. Цена нормочаса в валюте нормочаса.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Автосервис --> Нормочасы".
