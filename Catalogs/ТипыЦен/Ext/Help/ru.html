Справочник "Типы цен"

Назначение

В справочнике Типы цен содержатся названия типов цен, которые используются в компании (например: «розничная», «оптовая», «дилерская»).

Для каждой позиции номенклатуры можно назначить столько цен, сколько их хранится в справочнике Типы цен. Количество цен на элементы номенклатуры в решении не ограничено.

Для просмотра истории цен номенклатуры в разрезе типов цен и поставщиков предусмотрен регистр сведений Цены.

С одним контрагентом может быть оформлено множество договоров, причем в каждом из договоров может быть назначен свой тип цен. Этот тип цен будет по умолчанию устанавливаться в документах, оформленных по данному договору.

Предопределенные элементы

  • Основной тип цен закупки. Тип цен закупки по умолчанию. Наименование можно изменить.
  • Основной тип цен продажи. Тип цен продажи по умолчанию. Наименование можно изменить.
  • Нормативная цена. Тип цен для произведенного товара. Наименование можно изменить.

Реквизиты

  • Расчет цены. Переключатель на две позиции:
    • Цены назначаются и хранятся для каждой позиции номенклатуры - вводятся и хранятся цены для элементов номенклатуры.
    • Цены автоматически рассчитываются от базового типа цен - цены не хранятся, но автоматически рассчитываются на основе базового типа цен. Если валюта базового типа цены и расчетного типа цены не совпадает, то значение цены пересчитывается на дату записи цены по курсу, определенному в справочнике валют.
  • Базовый тип цен. Поле доступно, если выбрана вторая позиция предыдущего переключателя. Устанавливает базовый тип цен для автоматического расчета данного типа цен.
  • % наценки. Поле доступно, если для переключателя Расчет цены выбрана позиция Цены автоматически рассчитываются от базового типа цен. Служит для установки наценки/скидки в процентах относительно базового типа цен. Положительный процент означает наценку, отрицательный - скидку.
  • % наценки по типам.  Поле доступно, если для переключателя Расчет цен выбрана позиция Цены автоматически рассчитываются от базового типа цен. Назначает процент наценки/скидки для типа номенклатуры. Положительный процент означает наценку, отрицательный - скидку. 
  • Валюта цены. Реквизит является обязательным для заполнения. Ссылается на справочник Валюты. Цены хранятся в этой валюте. Если установлен флажок В валюте товара, то это поле недоступно, тогда цены хранятся в валюте указанного товара.
  • В валюте товара. При установленном флажке - вести учет в валюте учета номенклатуры, при снятом - в валюте типа цены.
  • Цена включает НДС. Определяет, включается ли в данный тип цен налог на добавочную стоимость. Влияет на расчет сумм «НДС» и «Всего» в документах.
  • Метод округления. Выбирается вариант округления цены по арифметическим правилам, либо всегда в большую сторону.
  • Точность. Количество значащих знаков цены после запятой.
  • Документы закупки товаров. Если флажок установлен, то можно использовать данный тип цен в качестве цен закупки. Служит для отбора в документах закупки.
  • Документы продажи. Если флажок установлен, то можно использовать данный тип цен в качестве цен продажи. Служит для отбора в документах отгрузки.
  • Автоматически регистрировать цены по приходу. Флажок активен, если переключатель порядка назначения цен установлен в позиции Цены назначаются и хранятся для каждой позиции номенклатуры. Позволяет автоматически регистрировать цены поставщиков при поступлении товара на склад. 
  • Включать в прайс-лист. Если флажок установлен, то данный тип цен будет использоваться при печати прайс-листа.
  • Аналитика учета цен. Реквизит является обязательным для заполнения. Выбирается из перечисления Алгоритм получения цены:
    • По номенклатуре.
    • По характеристике.
    • По единице измерения.
  • Комментарий. Любая дополнительная информация.
