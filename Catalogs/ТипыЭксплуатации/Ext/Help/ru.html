Справочник "Типы эксплуатации"

Назначение

Справочник «Типы эксплуатации» предназначен для хранения списка типов эксплуатации нетоварных активов.

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 50.

Не иерархический

Реквизиты

  • Способ начисления амортизации. Реквизит является обязательным для заполнения (если снят флаг консервирования). Ссылается на перечисление «Способы начисления амортизации». Способ начисления амортизации. Может принимать следующие значения: 100%-й при вводе, линейный, уменьшаемого остатка, по сумме чисел лет, по объему выработки.
  • Амортизационная группа. Ссылается на перечисление «Амортизационные группы». Амортизационная группа.
  • Консервирование. Тип значения: «Булево». Флаг "консервирования" актива. При консервировании амортизация не начисляется.
  • Статья расходов по амортизации. Ссылается на справочник «Статьи доходов и расходов». Статья расходов по амортизации.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Прочие активы --> Типы эксплуатации".
