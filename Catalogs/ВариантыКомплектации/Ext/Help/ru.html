Справочник "Варианты комплектации"

Назначение

Справочник "Варианты комплектации" описывает возможные варианты комплектации различных моделей автомобилей. Справочник является подчиненным справочнику «Модели автомобилей». Согласно этому справочнику для различных вариантов поставки автомобиля можно задавать различные дополнительные опции его комплектации.

Параметры

Длина кода: 8, тип кода:Строка.

Длина наименования: 50.

Не иерархический.

Владельцы справочника: «Модели автомобилей».

Реквизиты

  • № по каталогу. Тип значения: «Строка». Номер по каталогу (артикул) варианта комплектации.
  • Полное наименование. Тип значения: «Строка». Полное наименование комплектации.
  • Тип кузова. Ссылается на справочник «Типы кузовов». Тип кузова данного варианта комплектации автомобиля.
  • Тип двигателя. Ссылается на справочник «Типы двигателей». Тип двигателя данного варианта комплектации автомобиля.
  • Тип КПП. Ссылается на справочник «Типы КПП». Тип КПП данного варианта комплектации автомобиля.
  • Межсервисный интервал ТО. Тип значения: «Число». Число месяцев, через которое автомобиль данной комплектации должен проходить техобслуживание.
  • Межсервисный пробег ТО. Тип значения: «Число». Число километров, через которое автомобиль данной комплектации должен проходить техобслуживание.
  • Модель двигателя. Ссылается на справочник «Модели двигателей». Модель двигателя данного варианта комплектации автомобиля.
  • Тип автомобиля. Ссылается на справочник «Типы автомобилей». Тип автомобиля данного варианта комплектации автомобиля.
  • Тип привода. Ссылается на перечисление «Типы приводов автомобилей». Тип привода данного варианта комплектации автомобиля.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Перейти --> Варианты комплектации" справочника «Модели автомобилей».
