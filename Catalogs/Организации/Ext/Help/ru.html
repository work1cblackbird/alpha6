Справочник "Организации"

Назначение

В справочнике Организации хранится список организаций, из которых состоит компания. Справочник содержит статичную или редко меняющуюся информацию, идентифицирующую эти организации, а также дополнительные сведения, необходимые для ведения учета по этим организациям в единой информационной базе. Относится к подсистеме Структура компании раздела Предприятие.

Каждая организация может формировать свой комплект отчетности. Каждой из них может быть назначен префикс номера документов - это позволяет в рамках единой информационной базы вести раздельную нумерацию для документов, оформленных от имени разных организаций.

В справочнике также хранится информация, необходимая для оформления первичных документов, ведения бухгалтерского учета и получения отчетов.

В список организаций введен предопределенный элемент. До начала работы его следует переименовать в соответствии с наименованием основной организации и заполнить необходимые реквизиты.

Форма элемента справочника Организации содержит несколько вкладок, каждая из которых устанавливает соответствующие параметры организации.

  • Вкладка Основные данные. Заполняются основные данные о предприятии: форма собственности, дата регистрации, плательщик НДС или нет, коды регистрации и ответственные лица организации. 
  • Вкладка Контактная информация. Предоставляет возможность просмотра и редактирования контактной информации, связанной с организацией: сведения о телефонах, адресах, адресах электронной почты и пр. 
  • Вкладка Счета содержит список банковских счетов организации. Один из банковских счетов основной - он выбирается кнопкой Основной. Строка таблицы, соответствующая основному счету, выделена полужирным шрифтом. Основной счет, если он задан, используется по умолчанию в документах данной организации. 
  • На вкладке Подтверждающие документы содержится список документов, подтверждающих разрешение деятельности организации.
  • Вкладка Дополнительно содержит признак автоматического закрытия сделок: если флажок установлен, то при вводе нового договора взаиморасчетов с контрагентом в этом договоре будет установлен признак автоматического закрытия сделок.
  • Вкладка Типы цен содержит типы проверяемых цен реализации при проведении документов поступления. Становится доступной, если хотя бы у одного пользователя право "Проверка превышения цены прихода над ценой реализации" установлено в значение "Предупреждать" или "Блокировать приход".
  • На вкладке Логотип и Печать отображаются присоединенные файлы. Также выбрав переключатель Печать и нажав кнопку Основная можно установить факсимиле печати для организации.

Реквизиты

  • Полное наименование. Полное наименование организации.
  • Префикс. При создании регламентированных документов, относящихся к этой организации, префикс номеров документов будет формироваться автоматически со сквозной нумерацией по каждой организации. В качестве префикса будут использованы символы (до двух), заданные в текстовом поле. Виды документов, которые помечаются данным префиксом, задаются в правах и настройках для компании в целом.
  • Форма собственности. Форма собственности организации:
    • Юридическое лицо.
    • Индивидуальный предприниматель.
    • Частное лицо.
    • Прочая форма собственности.
  • Банковский счет. Основной банковский счет организации.
  • Дата регистрации. Дата регистрации организации.
  • НЕ является плательщиком НДС. Если флажок установлен, то при заполнении документов учет НДС производиться не будет.
  • ИНН. Индивидуальный номер налогоплательщика.
  • КПП. Код причины постановки на учет.
  • Код по ОКПО. Код предприятия по общероссийскому классификатору предприятий и организаций.
  • ГНИ. Номер государственной налоговой инспекции.
  • Код по КОПФ. Код по классификатору организационно-правовых форм.
  • Служебная статья. Определяет служебную статью доходов и расходов, используемую при закрытии периода.
  • Код по ОКФС. Код по общероссийскому классификатору форм собственности.
  • Код по ОКВЭД. Код по классификатору видов экономической деятельности.
  • Код по ОКДП. Код по общероссийскому классификатору видов экономической деятельности, продукции и услуг.
  • Код по ОКОНХ. Код по общероссийскому классификатору отраслей народного хозяйства.
  • ОГРН. Основной государственный регистрационный номер.  
  • Контрагент интеркампани. Указывает на связь организации и контрагента (при включенной опции учета «Интеркампани» - это механизм конфигурации, который обеспечивает доступность для продажи товаров, принадлежащих другой организации). Для настройки передач товаров между организациями требуется перейти в раздел Предприятие – Интеркампани - Настройка передачи товаров между организациями.  
  • Код ИМНС. Код Инспекции Министерства Российской Федерации по налогам и сборам.
  • Ответственные лица. Ответственные лица организации. Информация об ответственных лицах организации сохраняется с учетом истории изменения. Для просмотра ранее установленных значений используется список регистра сведений Сведения компании.
