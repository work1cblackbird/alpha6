Справочник "Единицы измерения"

Назначение

Справочник предназначен для хранения данных о единицах измерения товарно-материальных ценностей.

Позиция номенклатуры характеризуется базовой единицей измерения. Кроме базовой единицы измерения, позиция номенклатуры может иметь необходимое число других единиц измерения (упаковок).
Например, сигареты могут иметь единицы измерения: "штука", "пачка", "блок", "коробка".

Каждая единица измерения (упаковка) имеет свой коэффициент пересчета относительно базовой единицы. Каждая единица измерения имеет свой штрихкод и свой вес.

Параметры

Длина кода: 9, тип кода:Строка.

Длина наименования: 25.

Не иерархический

Владельцы справочника: «Номенклатура», «Типы номенклатуры».

Реквизиты

  • Базовая единица. Реквизит является обязательным для заполнения. Ссылается на справочник «Классификатор единиц измерения». Базовая единица измерения для данной позиции номенклатуры из классификатора единиц измерения.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета единицы измерения относительно базовой единицы измерения.
  • Вес. Тип значения: «Число». Обычно соответствует массе Брутто в основной единице измерения массы для данной единицы или упаковки.
  • Точность. Тип значения: «Число». Количество значащих цифр после запятой, которое применимо для указания количества в этой единице измерения.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из справочников «Типы номенклатуры» и «Номенклатура» "Перейти --> Единицы измерения".

Дополнительная информация

При заполнении информации о единице измерения используется справочник «Классификатор единиц измерения». В этот справочник заносится общий список применяемых компанией единиц измерения.
