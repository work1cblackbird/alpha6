Журнал документов "Расчеты по зарплате"

Назначение

Журнал документов Расчеты по зарплате предназначен для группировки и отображения документов, с помощью которых в информационную базу вводятся данные о начислении и о выплате зарплаты сотрудникам. Является удобным инструментом для отбора и просмотра документов. С его помощью возможен отбор по конкретным реквизитам и видам документов.  Относится к подсистеме Расчеты по зарплате раздела Финансы.

Список регистрируемых документов:

  • Начисление зарплаты,
  • Выплата зарплаты.

Графы

  • Хоз. операция. Выполняемая документом операция.
  • Сумма. Сумма выданных / начисленных денежных средств.
  • Валюта. Валюта документа.
  • Касса. Касса компании, в которой проходила операция выдачи денежных средств.  
  • Автор. Автор документа - пользователь, создавший документ.
