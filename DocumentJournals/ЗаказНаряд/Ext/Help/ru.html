Журнал документов "Заказ - наряд"

Назначение

Журнал документов предназначен для группировки и отображения документов, имеющих общую учетную сущность. Является удобным инструментом для отбора и просмотра документов. С его помощью возможен отбор по конкретным реквизитам и видам документов.  

Журнал "Заказ-наряд" содержит документы, отражающие автоработы организации.

Список регистрируемых документов: «Заявка на ремонт», «Заказ-наряд», «Перемещение товаров в производство», «Извлечение товаров из производства», «Акт разногласий», «Ввод остатков товаров в производстве», «Разукомплектация автомобилей», «Перемещение незавершенного производства».

Графы

  • Операция. Хозяйственная операция документа.
  • Дата сводного ремонтного заказа. Дата сводного ремонтного заказа заказ-наряда.
  • Дата создания. Дата создания заказ-наряда.
  • Дата начала. Дата начала выполнения работ по заказ-наряду.
  • Дата окончания. Дата окончания работ по заказ-наряду.
  • Заказчик. Заказчик работ по заказ-наряду.
  • Контрагент. Плательщик по заказ-наряду.
  • Договор. Договор взаиморасчетов с плательщиком по заказ-наряду.
  • Автомобиль. Автомобиль, ремонт которого выполняется по заказ-наряду.
  • Пробег. Пробег автомобиля на дату выполнения ремонтных работ по заказ-наряду.
  • Состояние. Состояние выполнения работ по заказ-наряду.
  • Сумма. Сумма документа.
  • Сумма деталей. Сумма деталей, используемых в ремонте по заказ-наряду.
  • Сумма работ. Сумма выполняемых работ по заказ-наряду.
  • Валюта. Валюта документа.
  • Курс. Курс валюты документа.
  • Склад. Склад компании или цех, в котором осуществляется ремонт.
  • Организация. Юридическое лицо, от которого выписан документ.
  • Подразделение. Подразделение компании, в котором был выписан документ.
  • Тип цен. Тип цен деталей.
  • Автор. Автор документа - пользователь, создавший документ.
  • Вид ремонта. Вид ремонта по заказ-наряду.
  • Комментарий. Комментарий к документу.
  • Регламентированный учет. Флаг отражения документа в регламентированном учете.
  • Повторное обращение. Признак повторного ремонта автомобиля в сервисе.

Способ вызова

Вызывается из меню "Документы --> Автосервис --> Журнал - Заказ наряд".

Дополнительная информация

Документы в журнале раскрашиваются согласно правилам раскраски в списке документов «Заказ-наряд».
