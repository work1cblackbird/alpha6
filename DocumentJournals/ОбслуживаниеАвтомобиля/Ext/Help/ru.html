Журнал документов "Обслуживание автомобиля"

Назначение

Журнал документов предназначен для группировки и отображения документов, имеющих общую учетную сущность. Является удобным инструментом для отбора и просмотра документов. С его помощью возможен отбор по конкретным реквизитам и видам документов.  

Журнал "Обслуживание автомобиля" содержит документы, отражающие автоработы организации.

Список регистрируемых документов: «Заявка на ремонт», «Заказ-наряд».

Графы

  • Контрагент. Плательщик по заказ-наряду или заявке на ремонт.
  • Договор. Договор взаиморасчетов с плательщиком по заказ-наряду или заявке на ремонт.
  • Автомобиль. Автомобиль, ремонт которого выполняется по заказ-наряду или заявке на ремонт.
  • Состояние. Состояние выполнения работ по заказ-наряду.
  • Документ основание. Документ, на основании которого был создан заказ-наряд или заявка на ремонт.
  • Сводный ремонтный заказ. Сводный документ ремонта автомобиля.
  • Дата начала. Дата начала выполнения работ по документу.
  • Дата окончания. Дата окончания выполнения работ по документу.
  • Тип цен. Тип цен деталей.
  • Вид ремонта. Вид ремонта по документу.
  • Пробег. Пробег автомобиля из документа.
  • Гос. номер. Государственный номер автомобиля из документа.

Способ вызова

Вызывается из меню "Автосервис --> Журналы документов --> Обслуживание автомобиля".

Дополнительная информация

Документы типа «Заказ-наряд» в журнале раскрашиваются согласно правилам раскраски в списке документов «Заказ-наряд». Документы типа «Заявка на ремонт» раскраски не имеют.
