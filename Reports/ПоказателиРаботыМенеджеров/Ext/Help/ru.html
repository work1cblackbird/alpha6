Отчет "Показатели работы менеджеров"

Назначение

Этот отчет служит для анализа эффективности работы менеджеров автосалона.

Относится к подсистемам: Автосалон, Заказ автосалона, к подсистемам Выкуп и Продажи раздела Автомобили с пробегом

Показатели отчета

  • Карточки клиентов - Количество элементов справочника “Контрагенты”, созданных менеджером за период формирования отчета.
  • Вх. звонки, визиты, эл. обращение - Общее количество документов Событие в состоянии Завершено по менеджеру за период формирования отчета с видами событий Входящий звонок, Встреча и Электронное обращение.
  • Вх. звонки - Количество документов Событие с видом Входящий звонок и состоянием Завершено по менеджеру за период формирования отчета.
  • Вх. визиты - Количество документов Событие с видом Встреча и состоянием Завершено по менеджеру за период формирования отчета.
  • Вх. эл. обращение - Количество документов Событие с видом Электронное обращение и состоянием Завершено по менеджеру за период формирования отчета.
  • Исх. звонки - Количество документов Событие с видом Исходящий звонок и состоянием Завершено по менеджеру за период формирования отчета.
  • Рабочие листы - Количество созданных документов Рабочий лист по менеджеру за период формирования отчета.
  • Тест-драйвы - Количество документов Тест-драйв в состоянии Завершено по менеджеру за период формирования отчета.
  • Контракты - Количество документов Заказ на автомобиль по которым (или по документам Счет на оплату за автомобиль введенным на основании заказа) получена предоплата от клиента, по менеджеру за период формирования отчета. Предоплатой считается первое поступление денежных средств от клиента по заказу на автомобиль. Датой предоплаты считается дата первого поступления денежных средств от клиента по заказу на автомобиль.
  • Оформления (Выписки) - Количество проведенных документов Реализация автомобилей по менеджеру за период формирования отчета. Количество проданных автомобилей определяется по числу строк в документе реализации.
  • Выдачи - Количество документов Событие с видом Завершение сделки и состоянием Завершено по менеджеру за период формирования отчета. 
  • Несостоявшиеся (Сделки) - Количество документов Рабочий лист, которые достигли последнего по порядку статуса, но по ним так и не был создан документ Заказ на автомобиль, по менеджеру за период формирования отчета.
  • Отказы - Количество документов Заказ на автомобиль, на основании которых введена Отмена заказа на автомобиль, по менеджеру за период формирования отчета.
  • Trade-in (Трейд-ин) - Количество документов Поступление автомобилей, которыми был зарегистрирован выкуп клиентского автомобиля по документу Заказ на автомобиль, по менеджеру за период формирования отчета. Учитывается число строк в документе поступления автомобилей по трейд-ин.
  • Визиты к звонкам (события). Соотношение показателя отчета Вх. визитов к показателю Вх. звонкам.
  • Тест-драйвы к событиям. Соотношение показателя отчета Тест-драйвы к показателю Вх. звонки, визиты, эл. обращение.
  • Контракты к событиям. Соотношение показателя отчета Контракты к показателю Вх. звонки, визиты, эл. обращение.
  • Контракты к тест-драйвам. Соотношение показателя отчета Контракты к показателю Тест-драйвы.
  • Отказы к контрактам. Соотношение показателя отчета Отказы к показателю Контракты.
