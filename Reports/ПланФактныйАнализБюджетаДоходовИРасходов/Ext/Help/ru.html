Отчет "План-фактный анализ бюджета доходов и расходов"

Назначение

Этот отчет служит для сравнения запланированных и фактических данных движения доходов и расходов в разрезе статьи дохода и расхода.

Формы

  • План-фактный анализ бюджета доходов и расходов.  Позволяет сформировать отчет с возможностью гибкой настройки группировок строк, показателей, дополнительных полей и т.д.  

Назначение некоторых реквизитов формы

  • Сценарий планирования. Обязательный для заполнения. Справочник «Сценария планирования». Определяет сценария планирования, по которому будет производиться сравнение данных.
  • Разворачивать по периодам. Булево. Истина - данные разворачиваются по периодам (соответствующим периодичности выбранного сценария планирования).
  • Выводить незапланированные данные. Булево. Управляет отображением в отчете данных по статьям,  по которым есть фактические данные, но нет запланированных данных. 

Способ вызова

Вызывается из меню "Отчеты--> Бюджетирование -->План фактный анализ бюджета доходов и расходов"

Дополнительная информация

Отчет может формироваться только в виде табличного документа.

Отчет может быть сформирован в разрезе дополнительных свойств.
