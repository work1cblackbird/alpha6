Отчет "План-фактный анализ бюджета движения денежных средств"

Назначение

Этот отчет служит для сравнения запланированных и фактических данных движения денежных средств в разрезе статьи ДДС.

Формы

  • План-фактный анализ бюджета движения денежных средств.  Позволяет сформировать отчет "План-фактный анализ бюджета движения денежных средств" с возможностью гибкой настройки группировок строк, показателей, дополнительных полей и т.д.  

Назначение некоторых реквизитов формы

  • Сценарий планирования. Обязательный для заполнения. Справочник «Сценария планирования». Определяет сценария планирования, по которому будет производиться сравнение данных.
  • Разворачивать по периодам. Булево. Истина - данные разворачиваются по периодам (соответствующим периодичности выбранного сценария планирования).
  • Выводить незапланированные данные. Булево. Управляет отображением в отчете данных по статьям ДДС,  по которым есть фактические данные, но нет запланированных данных. 

Способ вызова

Вызывается из меню "Отчеты--> Бюджетирование -->План фактный анализ бюджета движения денежных средств"

Дополнительная информация

Отчет может формироваться только в виде табличного документа.
