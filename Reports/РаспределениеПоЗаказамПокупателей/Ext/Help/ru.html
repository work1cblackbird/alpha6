Отчет "Распределение по заказам покупателей"

Назначение

Отчет служит для анализа распределения по заказам покупателя для документов «Поступление товаров» и «Перемещение товаров». Относится к разделу Снабжение.

Показатели отчета:

  • Количество по документу. Количество товара, поступившее по документу закрытия.
  • Зарезервировано. Количество товара, зарезервированное документом закрытия по заказам.
  • Поступило без заказа. Количество товара, поступившее без резервирования по документу закрытия.

На форме настройки отчета находится поле Документ закрытия для указания документа поступления/перемещения товаров.

Дополнительная информация

Отчет может формироваться в виде табличного документа, сводной таблицы, диаграммы и сводной диаграммы.

Отчет может быть сформирован в разрезе дополнительных свойств.
