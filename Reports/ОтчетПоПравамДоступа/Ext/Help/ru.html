Отчет "Отчет по правам доступа"

Назначение

Отчет по правам доступа служит для отображения доступности объектов по профилям групп доступа, группам доступа. 

Варианты отчета:

  • По группам доступа. Выводятся данные по доступности объектов по группам доступа;
  • По профилям групп доступа.  Выводятся данные по доступности объектов по профилям групп доступа.

    Показатели

  • Доступность права. Показывает наличие (+) или отсутствие (-) права доступа к объекту на чтение, редактирование, добавление, удаление и т.д.

    • ВАЖНО! Для корректного формирования отчета рекомендуется использовать отборы и параметры отчета. Иначе при больших объемах данных отчет может не сформироваться из-за нехватки памяти.
