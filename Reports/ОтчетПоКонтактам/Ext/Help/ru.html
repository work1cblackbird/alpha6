Отчет "Отчет по контактам"

Назначение

Этот отчет служит для просмотра количества взаимодействий с видом "Событие".

Относится к подсистеме: «Маркетинг».   

Способ вызова

Вызывается из меню "Маркетинг --> Отчеты --> Отчет по контактам"

Дополнительно

Все события, в том числе, созданные автоматически (например из Контакт-центра при завершении), попадают в отчет.
