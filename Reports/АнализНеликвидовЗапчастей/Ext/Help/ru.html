Отчет "Анализ неликвидов запчастей"

Назначение

Отчет служит для анализа неликвидных товаров по складам и контрагентам.

Параметры

  • Неликвид после. Дата с которой нет продаж и выдач в производство, чтобы признать товар неликвидом.
  • Претендент после. Дата с которой нет продаж и выдач в производство, чтобы анализировать товар как претендента на неликвидность.
  • Критичный запас. Количество месяцев, на которое хватит товара имеющегося на складе(при текущем расходе - количество в остатке деленное на среднемесячный расход N месяцев), чтобы товар, рассматриваемый как претендент на неликвидность по сроку без продаж, был признан таковым.
  • Тип цен. Тип цен, для показателя себестоимость по типу цен.
  • Интервал расхода. Количество дней для расчета среднемесячного(текущего) расхода. Текущий расход считается, как количество проданного / количество дней для расчета текущего расхода деленное на 30 с округлением в большую сторону. 
  • Подразделение. Подразделение по которому будет выбираться цены для расчета показателя стоимость по типу цен.

Способ вызова

Вызывается из меню "Автосервис-->Отчеты-->Анализ неликвидов запчастей"
