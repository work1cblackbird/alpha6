Отчет "Рейтинг продаж"

Назначение

Этот отчет служит для анализа финансовых и количественных данных о проданных товарах и оказанных услугах.

Форма

Позволяет сформировать отчет "Рейтинг продаж" с возможностью настройки группировок строк, показателей, дополнительных полей и т.д.

Способ вызова

Вызывается из меню "Отчеты-->Отгрузка ТМЦ-->Рейтинг продаж"

Дополнительная информация

Отчет может формироваться в виде табличного документа, сводной таблицы и диаграммы.

Отчет может быть сформирован в разрезе дополнительных свойств.

Отчет для группировок строк с типом значения "Только иерархия" разрешает дополнительные настройки вывода групп: настройку списка групп, которые будут участвовать в получении результата,  настройку числа вывода первых групп (ограничение равное 0 отключает данное ограничение - устанавливается по умолчанию), а так же настройку получения информации, в виде общего итога, о прочих не вошедших в число первых групп.

Отчет может быть настроен с сортировкой строк выводимой информации, по показателю.
