Отчет "Оборачиваемость товаров"

Отчет предназначен для анализа отдельной группы показателей эффективности работы предприятия - показателей оборачиваемости товарных запасов.
В отчете выводятся следующие основные показатели:

1. Коэффициент оборачиваемости

КО = ФР / СО, где:
КО - коэффициент оборачиваемости,
ФР - фактический расход за период отчета,
СО - средний остаток за период отчета (см. 3)
Коэффициент оборачиваемости показывает, сколько в среднем оборотов совершают товарные запасы за один период.
Низкие значение могут говорить о нерентабельности данного товара: товар медленно превращается в деньги, а на складах находятся излишки.
Высокие значения могут сигнализировать либо о хороших продажах, либо о недостаточных товарных запасах.

2. Период оборачиваемости

ПО = СО / ФР * КД, где:
ПО - период оборачиваемости
СО - средний остаток за период отчета (см. 3),
ФР - фактический расход за период отчета,
КД - количество дней в периоде отчета
Период оборачиваемости выражается в днях и показывает, сколько в среднем дней товар хранится на складе.
Низкое значение показателя может означать высокую эффективность управления товарными запасами.
Высокое значение показателя говорит о возможно избыточном запасе товаров.

3. Средний остаток за период отчета

СО = СУММА((ОНi + ОКi) / 2) / КД, где:
СО - средний остаток за период отчета,
ОНi - остаток товаров на начало i-го дня,
ОКi - остаток товаров на конец i-го дня,
КД - количество дней в периоде отчета,
i = 1, 2 ... КД
Средний остаток за период показывает, сколько в среднем в день товаров лежит на складе.

4. Фактический расход за период

Фактический расход за период - это расход товара, означающий продажу (списание товаров в производство, реализация товаров и т.д.).
Остальной расход считается внутренними перемещениями. Какие расходы товаров являются внутренними перемещениями - определяется списком хоз. операций в параметре Внутренние перемещения.

Отчет имеет две настройки по умолчанию:
  • По количеству - показатели оборачиваемости рассчитываются по количеству товаров
  • По сумме - показатели оборачиваемости рассчитываются по себестоимости товаров
В параметр Внутренние перемещения включены следующие хоз.операции:
  • Перемещение товаров
  • Перемещение товаров в филиал
  • Перемещение товаров из филиала
Товары в обеих настройках отчета сортируются в первую очередь по возрастанию коэффициента оборачиваемости, а затем - по убыванию среднего остатка.
