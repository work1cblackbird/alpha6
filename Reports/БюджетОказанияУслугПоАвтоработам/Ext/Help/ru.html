Отчет "Бюджет оказания услуг по автоработам"

Назначение

Этот отчет служит для анализа данных о запланированных авторабот.

Формы

  • Бюджет оказания услуг по автоработам. Позволяет сформировать отчет "Бюджет оказания услуг по автоработам" с возможностью гибкой настройки группировок строк, показателей, дополнительных полей и т.д. Группировка колонок не настраивается и всегда выполняется по сценарию планирования и периодам планирования.  В отчет попадают данные по всем сценариям планирования, периодичность которых соответствует выбранной на форме (реквизит "Периодичность планирования" - обязательный для заполнения). Список возможных показателей отчета зависит от выбранной аналитики.

Способ вызова

Вызывается из меню "Отчеты--> Бюджетирование --> Бюджет оказания услуг по автоработам"

Дополнительная информация

Отчет может формироваться только в виде табличного документа
