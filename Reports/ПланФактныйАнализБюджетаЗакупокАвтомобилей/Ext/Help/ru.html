Отчет "План-фактный анализ бюджета закупок автомобилей"

Назначение

Этот отчет служит для сравнения запланированных и фактических данных закупок автомобилей в разрезе моделей и вариантов комплектации.

Формы

  • План-фактный анализ закупок автомобилей. Позволяет сформировать отчет "План-фактный анализ закупок автомобилей" с возможностью гибкой настройки группировок строк, показателей, дополнительных полей и т.д.  

Назначение некоторых реквизитов формы

  • Аналитика отчета. По вариантам комплектации, либо по моделям автомобилей.
  • Сценарий планирования. Обязательный для заполнения. Справочник «Сценария планирования». Определяет сценарий планирования, по которому будет производиться сравнение данных.
  • Разворачивать по периодам. Булево. Истина - данные разворачиваются по периодам (соответствующим периодичности выбранного сценария планирования).
  • Выводить незапланированные данные. Булево. Управляет отображением в отчете данных по вариантам комплектации\моделям,  по которым есть фактические данные закупок, но нет запланированных данных. 

Способ вызова

Вызывается из меню "Снабжение автосалона --> Отчеты --> Отчеты раздела "Снабжение автосалона" --> "План-фактный анализ бюджета закупок автомобилей".

Дополнительная информация

Отчет может формироваться только в виде табличного документа.
