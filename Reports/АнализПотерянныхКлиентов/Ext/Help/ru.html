Отчет "Анализ потерянных клиентов"

Назначение


Отчет служит для анализа произведенных ремонтов за период в разрезе заказчиков. Показателями отчета являются сумма и количество заказ-нарядов за период. В отчет попадают данные о заказ-нарядах, у которых дата создания/выполнения/закрытия находится в периоде указанном в параметрах отчета. Заказ-наряды распределяются по кварталам по дате создания. Количество кварталов определяет период указанный в параметрах отчета. Также к периоду отчета будут добавлены предыдущие кварталы, если в них есть заказ-наряды созданные в предыдущем квартале, а закрытые/выполненные в период указанный в параметрах отчета.

В отчете есть возможность, получить данные по клиентам, не попавшим в период отчета, для этого надо выбрать дополнительную детализацию по типам клиентов. Если дата создания Заказ-наряда входит в период отчета, то клиент относится к типу "Постоянные клиенты", иначе к типу "Потерянные клиенты".

Для получения информации, о прошедших ремонтных работах, не попавших в период отчета, надо выбрать дополнительную детализацию по типам данных.

Отчет позволяет группировать контрагентов по количеству обращений и суммам заказ-нарядов. Интервалы для группировки настраиваются по кнопке "Настройка интервалов".

Способ вызова

Вызывается из меню "Маркетинг-->Отчеты-->Анализ потерянных клиентов"
