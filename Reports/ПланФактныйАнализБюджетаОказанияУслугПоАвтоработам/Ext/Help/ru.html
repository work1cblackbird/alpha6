Отчет "План-фактный анализ бюджета оказания услуг по автоработам"

Назначение

Этот отчет служит для сравнения запланированных работ и выполненных по факту в суммовых, количественных и процентных показателях.

Формы

  • План-фактный анализ бюджета оказания услуг по работам.  Позволяет сформировать отчет "План-фактный анализ работ" с возможностью гибкой настройки группировок строк, показателей, дополнительных полей и т.д.  

Назначение некоторых реквизитов формы

  • Сценарий планирования. Обязательный для заполнения. Справочник «Сценария планирования». Определяет сценария планирования, по которому будет производиться сравнение данных.
  • Разворачивать по периодам. Булево. Истина - данные разворачиваются по периодам (соответствующим периодичности выбранного сценария планирования).
  • Выводить незапланированные данные. Булево. Управляет отображением в отчете данных о работах,  по которым есть фактические данные, но нет запланированных. 

Способ вызова

Вызывается из меню "Отчеты--> Бюджетирование -->План-фактный анализ бюджета оказания услуг по автоработам"

Дополнительная информация

Отчет может формироваться только в виде табличного документа.
