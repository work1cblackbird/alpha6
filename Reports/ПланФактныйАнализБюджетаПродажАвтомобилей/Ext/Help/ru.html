Отчет "План-фактный анализ бюджета продаж автомобилей"

Назначение

Этот отчет служит для сравнения запланированных и фактических данных продаж автомобилей в разрезе моделей и вариантов комплектации.

Формы

  • План-фактный анализ продаж автомобилей.  Позволяет сформировать отчет "План-фактный анализ продаж автомобилей" с возможностью гибкой настройки группировок строк, показателей, дополнительных полей и т.д.  

Назначение некоторых реквизитов формы

  • Аналитика отчета. По вариантам комплектации, либо по моделям автомобилей.
  • Сценарий планирования. Обязательный для заполнения. Справочник «Сценария планирования». Определяет сценария планирования, по которому будет производиться сравнение данных.
  • Разворачивать по периодам. Булево. Истина - данные разворачиваются по периодам (соответствующим периодичности выбранного сценария планирования).
  • Выводить незапланированные данные. Булево. Управляет отображением в отчете данных по вариантам комплектации\моделям,  по которым есть фактические данные продаж, но нет запланированных данных. 

Способ вызова

Вызывается из меню "Отчеты--> Бюджетирование -->План фактный анализ бюджета продаж автомобилей"

Дополнительная информация

Отчет может формироваться только в виде табличного документа.
