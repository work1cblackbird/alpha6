Регистр сведений "Состояния заявок на подпись"

Назначение

Регистр предназначен для хранения истории изменения статуса документов Заявка на подпись.

Параметры

Периодичность: В пределах секунды

Режим записи: Независимый

Измерения

  • Заявка на подпись. Ссылается на документ Заявка на подпись. Документ, для которого установлен статус. 

Ресурсы

  • Состояние. Ссылается на перечисление Статусы заявок на подпись. Установленный статус для документа.
  • Автор. Тип: Строка. Пользователь, который установил текущий статус документу.
  • Установлен вручную. Тип: Булево. Признак того, что статус был установлен пользователей информационной базы вручную, а не получен при синхронизации с сервисом ПЭП.
