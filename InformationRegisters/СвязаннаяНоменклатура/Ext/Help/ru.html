Регистр сведений "Связанная номенклатура"

Назначение

Регистр сведений "Связанные номенклатура" предназначен для хранения списка номенклатуры, которая может быть выбрана с номенклатурой с указанием их количества.

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Номенклатура. Ссылается на справочник «Номенклатура». Номенклатура, для которой задается связанная номенклатура.
  • Связанная номенклатура. Ссылается на справочник «Номенклатура». Связанная номенклатура.
  • Модель. Ссылается на справочник «Модели». Модель автомобиля, для которой предназначена связанная номенклатура.
  • Вариант комплектации. Ссылается на справочник «Варианты комплектации». В соответствии с выбранной моделью можно указать вариант комплектации, для которой можно указать связанную номенклатуру.

Ресурсы

  • Количество. Тип значения: «Число». Требуемое количество связанной номенклатуры.
