Регистр сведений "Штрихкоды"

Назначение

В регистре хранятся сведения о штрихкодах на товары и карточки. Позволяет автоматизировать процессы чтения штрихкодов и формирования табличных частей документов поступления и реализации товаров.

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Штрихкод. Реквизит является обязательным для заполнения. Тип значения: «Строка». Штрихкод товара или карточки.
  • Объект. Реквизит является обязательным для заполнения. Ссылается на справочники: «Номенклатура», «Карточки». Объект штрихкода (номенклатура или карточка).
  • Единица измерения. Ссылается на справочник «Единицы измерения». Единица измерения, для которой установлен данный штрихкод.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры, для которой установлен данный штрихкод.

Ресурсы

  • Выгружать в оборудование. Тип значения: «Булево». Признак выгрузки штрихкода в торговое оборудование с памятью.

Реквизиты

  • Запрет. Тип значения: «Булево». Действия с этим объектом запрещены. Если с помощью сканера будет считан штрихкод объекта с флагом запрета, этот объект не будет использоваться для подстановки.

Способ вызова

Доступен для просмотра и редактирования: из панели свойств справочника «Номенклатура», из формы элемента справочника «Карточки».
