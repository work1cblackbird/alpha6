Регистр сведений "Минимальная наценка"

Назначение

Регистр сведений "Минимальная наценка" содержит данные о минимальной наценке для указанного типа номенклатуры и выбранного пользователя.

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Номенклатура. Ссылается на справочник «Номенклатура». Номенклатура, для которой задается минимальная наценка.
  • Склад компании. Ссылается на справочник «Пользователи». Пользователь, который может использовать минимальную наценку для номенклатуры.
  • Процент наценки. Тип значения «Число». Процент наценки на номенклатуру.

Способ вызова

Редактируется в разделе "Маркетинг" -> "Ценообразование".
