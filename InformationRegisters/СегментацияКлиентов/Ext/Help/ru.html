Регистр сведений "Сегментация клиентов"

Назначение

Предназначен для хранения информации о том, к какому сегменту относится клиент. Одновременно клиент может относиться к нескольким сегментам. Регистр периодический, период день. Работа с сегментом выполняется с помощью обработки "Работа с сегментом".

Параметры

Периодичность: День

Режим записи: Независимый

Измерения

  • Клиент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Контрагенты, для которого назначается сегмент.
  • Сегмент клиента. Реквизит является обязательным для заполнения. Ссылается на справочник «Сегменты клиентов». Сегмент клиента.

Ресурсы

  • Принадлежность. Тип значения: «Булево». Значение "Да" - клиент принадлежит сегменту (при установке сегмента - сразу взводится). При исключении из сегмента - значение "Нет".
