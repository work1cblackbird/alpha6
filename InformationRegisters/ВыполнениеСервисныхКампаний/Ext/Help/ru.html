Регистр сведений "Выполнение сервисных кампаний"

Назначение

Регистр предназначен для хранения VIN автомобилей, для которых производителем назначены сервисные кампании.

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • VIN. Строка. VIN автомобиля, по которому назначена сервисная кампания.
  • Сервисная кампания. Ссылается на справочник «Сервисные кампании». Сервисная кампания.

Ресурсы

  • Дата выполнения. Дата. Дата, когда для автомобиля была выполнена сервисная кампания. Если значение не заполнено, то сервисная кампания не проводилась.
  • Документ выполнения. Ссылается на документ «Заказ-наряд». Заказ-наряд, по которому была проведена сервисная кампания. Если значение не заполнено, то сервисная кампания не проводилась или была проведена другой организацией. 

Способ вызова

Редактирование осуществляется из справочников «Сервисные кампании», «Автомобили».
