Регистр сведений "Состояния кодов маркировки"

Назначение

В данном регистре хранятся записи о состоянии кодов маркировки товаров.

Параметры

Периодичность: В пределах секунды

Режим записи: Независимый

Измерения

  • Документ основание. Является обязательным для заполнения. Ссылается на документы. Документ, который изменил состояние кода маркировки. 
  • Организация. Является обязательным для заполнения. Ссылается на справочник «Организации». Владелец товара с кодом маркировки. 
  • Номенклатура. Является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар). 
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры. 
  • GTIN. Тип значения: «Строка». Часть кода маркировки, которая описывает штрих-код товара. 
  • Серийный номер. Тип значения: «Строка». Уникальная последовательность символов для определения кода маркировки в указанном GTIN. 

Ресурсы

  • Состояние. Ссылается на перечисление «Состояния кодов маркировки». Установленное состояние кода маркировки товара документом основанием.
  • Код маркировки. Тип значения: «Строка». Представление кода маркировки товара.
