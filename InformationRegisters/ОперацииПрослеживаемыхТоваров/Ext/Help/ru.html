Регистр сведений "Операции прослеживаемых товаров"

Назначение

Регистр предназначен для хранения сведений об истории операций с товарами, подлежащими прослеживаемости.

Параметры

Периодичность: ПозицияРегистратора

Режим записи: ПодчинениеРегистратору

Измерения

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Организация, в которой произошла операция. 
  • Период отчета. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата начала квартала отчетности об операции. 
  • Отчетность операции. Реквизит является обязательным для заполнения. Ссылается на перечисление «Документы отображения операций». Вид отчетности для отображения текущей операции.
  • Код операции. Реквизит является обязательным для заполнения. Ссылается на справочник «Коды операций прослеживаемости». Организация, в которой произошла операция. 
  • Документ. Реквизит является обязательным для заполнения. Документ, на основании которого произошла операция. 
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочники «Номенклатура», «Автомобили». Прослеживаемый товар.  
  • РНПТ. Реквизит является обязательным для заполнения. Ссылается на справочник «ГТД». Регистрационный номер партии товаров.

Ресурсы

  • Количество прослеживаемости. Тип значения: «Число». Количество прослеживаемого товара в единицах прослеживаемости.
  • Сумма без НДС. Сумма прослеживаемого товара без налогов в российских рублях и копейках.
  • Вид документа. Ссылается на перечисление «Типы документов». Вид первичного документа, в котором произошла операция.
  • Номер документа. Тип значения: «Строка». Номер первичного документа, в котором произошла операция.
  • Дата документа. Тип значения: «Дата». Дата первичного документа, в котором произошла операция.
  • Контрагент. Ссылается на справочник «Контрагенты». Юридическое лицо или индивидуальный предприниматель, которому передается право собственности на товары, подлежащие прослеживаемости, или который передает право собственности на товары

Способ вызова

Интерактивный вызов не предусмотрен.
