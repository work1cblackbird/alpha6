Регистр сведений "Опции вариантов комплектации"

Назначение

Регистр предназначен для хранения списка опций вариантов комплектации моделей автомобилей. Состав опций разделяется на опции устанавливаемые по умолчанию, устанавливаемые производителем и устанавливаемые продавцом.

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Вариант комплектации. Ссылается на справочник «Варианты комплектации». Вариант комплектации, для которой определяется опция.
  • Опция комплектации. Ссылается на справочники «Опции автомобилей» и «Номенклатура». Опции, устанавливаемые на данный вариант комплектации.

Ресурсы

  • Количество опций. Тип значения: «Число». Количество опций в варианте комплектации.
  • Цена закупки. Тип значения: «Число». Цена закупки данной опции.

Реквизиты

  • Вид включения опции. Тип значения: «Число». 0 - Устанавливается продавцом, 1 - Устанавливается производителем, 2 - Устанавливается по умолчанию.

Способ вызова

Редактируется из формы элемента справочника «Варианты комплектации» в панели "Опции варианта комплектации".
