Регистр сведений "Настройки тест-драйва"

Назначение

Регистр предназначен для хранения настроек тест-драйвов по моделям.

Параметры

Периодичность: В пределах дня

Режим записи: Независимый

Измерения

  • Модель. Реквизит является обязательным для заполнения. Ссылается на справочник «Модели автомобилей». Модель автомобиля.

Ресурсы

  • Продолжительность эксплуатации. Тип значения: «Число». Хранит значение продолжительности эксплуатации по умолчанию для модели.
  • Максимальный пробег. Тип значения: «Число». Хранит значение максимально допустимого пробега для модели на тест драйве.
  • Анкета предварительная. Ссылка на справочник «Шаблоны анкет». Хранит анкету которую необходимо пройти перед началом тест-драйва.
  • Анкета итоговая. Ссылка на справочник «Шаблоны анкет». Хранит анкету которую необходимо пройти после окончания тест-драйва.
  • Срок поставки. Тип значения: «Число». Хранит значение срока поставки для модели .

Формы

  • Форма списка. Основная форма списка . Используется для отображения списка элементов.
  • Форма элемента. Основная форма редактирования. Используется для отображения данных и редактирования элементов. 

Способ вызова

Доступен для просмотра и редактирования в форме элемента справочника «Модели автомобилей».
