Регистр сведений "График работ ресурсов"

Назначение

Предназначен для отражения фактического рабочего времени ресурсов (сотрудники, пользователи, оборудование, залы,..).

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Измерения

  • Ресурс1. Составной тип значения: Сотрудник, Пользователь, Оборудование. Ресурс, для которого задаются временные интервалы.
  • Ресурс2. Составной тип значения: Сотрудник, Пользователь, Оборудование. Ресурс, для которого задаются временные интервалы.
  • Объект. Тип значения: Любая ссылка. Ссылка на объект конфигурации, прилагаемый к графику.
  • Дата. Тип значения: «Дата». Дата дня графика.
  • Начало рабочего времени. Тип значения: «Дата». Время суток, от которого начинается время.
  • Конец рабочего времени. Тип значения: «Дата». Время суток, в котором заканчивается время.
  • Смена. Ссылается на справочник «Смены». Служит для хранения временных интервалов рабочих смен.

Ресурсы

  • Продолжительность. Тип значения: «Дата». Продолжительность времени, разница между началом рабочего времени и окончанием рабочего времени.
  • Количество натуральных единиц. Тип значения: «Число». Количество занятых мест в зале, кабинете врача, болванок в станке и т. д.
  • Напомнить за. Тип значения: «Дата». Время, когда нужно напомнить о событии.

Формы

  • Форма списка. Основная форма списка и редактирования. Используется для отображения списка или редактирования элементов.
  • Форма заполнения. Форма для заполнения регистра.

Дополнительная информация

Регистр сведений график работ календарный можно заполнить по определённому графику или по всем имеющимся. Для этого на форме заполнения графика необходимо выбрать вид заполнения: по всем графикам, по выбранному графику. Указать периодичность заполнения: неделя, декада, месяц, квартал, год, произвольный период. И выбрать дату начала, при произвольном периоде и дату окончания.
