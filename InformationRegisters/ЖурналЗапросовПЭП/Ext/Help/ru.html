Регистр сведений "Служебный журнал обмена с doc4b.ru"

Назначение

Регистр предназначен для хранения информации об обмене запросов с сервисом ПЭП. 

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Таймстэмп. Тип значения: Число. Указывает когда произошел запрос к ПЭП. 
  • Дата. Тип значения: Дата и время. Дата и время формирования запроса.
  • Дата UTC. Тип значения: Дата и время. Дата и время формирования запроса относительно универсального часового пояса.
  • Путь. Тип значения: Строка. Путь к формированию запроса.
  • Тип запроса. Тип значения: Строка. Указывает какой запрос был отправлен.
  • Идентификатор запроса. Тип значения: УникальныйИдентификатор. Идентификатор, по которому определяется запрос.

Ресурсы

  • Тело запроса. Тип значения: Строка. Текст запроса, который отправляется на сервер ПЭП.
  • Тело ответа. Тип значения: Строка. Текст, который получен от ПЭП.
  • Статус. Тип значения: Число. Кодовое представление статуса запроса.
  • Длительность. Тип значения: Число. Время, которое было затрачено на получение ответа по запросу.
  • Пользователь. Тип значения: Строка. Пользователь информационной базы, от которого был выполнен запрос.
