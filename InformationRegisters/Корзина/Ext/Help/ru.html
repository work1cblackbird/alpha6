Регистр сведений "Корзина"

Назначение

Регистр сведений предназначен для хранения корзины товаров.

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Наименование. Является обязательным для заполнения. Тип значения: «Строка». Номер корзины.
  • Дата. Является обязательным для заполнения. Тип значения: «Дата». Дата формирования корзины.
  • Клиент. Является обязательным для заполнения. Тип значения: «Строка» или ссылка на справочник «Контрагенты». Клиент, для которого подобрана корзина.

Ресурсы

  • Значение. Тип значения: «ХранилищеЗначения». Содержит структуру с реквизитами корзины и её составом.

Формы

  • Форма выбора. Служит для просмотра списка корзин с возможностью наложения отборов. В верхней части расположен список корзин. В нижней части расположены данные корзины: реквизиты и состав.

Способ вызова

Вызывается из подменю "заполнить" документов в командной панели табличной части "Товары", либо из АРМ "Корзина" по кнопке "Загрузить".
