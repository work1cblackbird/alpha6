Регистр сведений "График поставок"

Назначение

Используется для хранения графика поставок товаров.

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Измерения

  • Заказ покупателя. Ссылается на документы «Заказ покупателя» или «Заказ внутренний». Заказ, для которого установлен график поставок.
  • Идентификатор строки. Реквизит является обязательным для заполнения. Уникальный идентификатор, для поиска товарной строки заказа.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры.
  • Дата поставки. Тип значения: «Дата». Плановая дата поставки товара поставщиком.
  • Контрагент. Ссылается на справочник «Контрагенты». Поставщик заказанного товара.

Ресурсы

  • Дата поступления. Тип значения: «Дата». Дата поступления заказанного товара в полном объеме.
  • Информация. Тип значения: «Строка». Дополнительная информация по заказу.
  • Количество. Тип значения: «Число». Количество товара, которое должно прийти на дату поставки.

Формы

  • Форма списка. Основная форма списка и редактирования. Используется для отображения списка поставок.
