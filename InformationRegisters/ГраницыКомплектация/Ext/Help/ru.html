Регистр сведений "Границы последовательности комплектации автомобилей"

Назначение

Используется для восстановления последовательности ввода и проведения документов. Предназначен для хранения информации о нарушениях ввода документов комплектации автомобилей. Применяется в обработке «Восстановление последовательностей».

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Измерения

  • Автомобиль. Ссылается на справочник «Автомобили». Автомобиль, ввод документов комплектации по которому нарушена.

Ресурсы

  • Момент времени. Тип значения: «Дата и время». Момент времени, на который нарушена последовательность ввода документов.
  • Документ ссылка. Тип значения: «ДокументСсылка». Документ, нарушивший последовательность ввода документов.

Формы

  • Форма списка. Основная форма списка и редактирования. Используется для отображения списка нарушений ввода документов.

Дополнительная информация

Регистр сведений "Границы последовательности комплектации автомобилей" является служебным и не предназначен для ручного редактирования. Сведения из данного регистра используются для восстановления последовательности проведения товарных документов.
