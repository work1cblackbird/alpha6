Регистр сведений "Использование штрихкодирования документов"

Назначение

Регистр предназначен для хранения настроек использования автоматического формирования штрихкода для документов при их записи.

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Подразделение компании. Ссылается на справочник  «Подразделения компании». Подразделение, к которому относится настройка.
  • Вид документов. Тип значения: «Строка». Вид документа, к которому относится настройка.

Ресурсы

  • Использование. Тип значения: «Булево». Использовать автоматическое штрихкодирование для данного вида документов и в данном подразделении.

Способ вызова

Редактируется из справочника «Подразделения компании» по кнопке «Настроить».
