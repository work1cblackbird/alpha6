Регистр сведений "Настройки начислений сотрудникам"

Назначение

Предназначен для хранения и использования в документе «Заказ-наряд» при подборе исполнителей для формирования суммы начисления. 

Параметры

Периодичность: непериодический

Режим записи: независимый

Измерения

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Организация, в которой используется настройка начисления сотруднику.
  • Сотрудник. Является обязательным для заполнения. Ссылается на справочник «Сотрудники». Сотрудник, для которого настраивается начисление.

Ресурсы

  • Вид начисления. Реквизит является обязательным для заполнения. Ссылается на план видов расчета «Начисления и удержания». Вид начисления, по которому выполняется начисление сотруднику.
  • Параметр расчета. Реквизит является обязательным для заполнения. Тип значения: «Число». В зависимости от способа расчета вида начисления процент/сумма для расчета начисления:
    • По нормочасам: сумма начисляемая за один нормочас в регламентированной валюте.
    • Суммой: сумма начисляемая за одну работу в регламентированной валюте.
    • Процентом: Процент от стоимости работы с учетом суммы скидок и без суммы НДС.
