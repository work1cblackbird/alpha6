Регистр сведений "Номера листов кассовой книги"

Назначение

Данный регистр предназначен для хранения номеров последних листов кассовой книги за конкретный день. Значения регистра можно менять вручную или с помощью обработки «Кассовая книга».

Параметры

Периодичность:День

Режим записи: Независимый

Измерения

  • Касса компании. Ссылается на справочник «Кассы компании». Касса компании, для которой строится кассовая книга. 

Ресурсы

  • Номер листа. Тип значения: «Число». Номер последнего листа кассовой книги за день.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма записи. Основная форма для редактирования записи регистра.
