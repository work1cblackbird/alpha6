Регистр сведений "Замены"

Назначение

Регистр сведений "Замены" предназначен для хранения информации о заменах номенклатуры. Замены хранятся в виде списка каталожных номеров.

Загрузка замен.

Замены могут быть загружены в регистр с помощью обработки "Загрузка данных из табличного документа". Загружаемый файл обязательно должен содержать 2 параметра: каталожный номер номенклатуры, для которой есть замена, и каталожный номер замены. Загрузка данных с пустыми каталожными номерами номенклатуры или замены не допустимы.

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Артикул номенклатуры. Строка. Каталожный номер номенклатуры, для которой есть замена.
  • Артикул номенклатуры замены. Строка. Каталожный номер замены для номенклатуры.
  • Производитель. Ссылается на справочник «Пользователи». Производитель деталей.
  • Группа. Строка. Определяет группу замены (Обратимая, Необратимая, Множественная, Одиночная).
  • Количество. Число. Количество номенклатуры для замены.

Реквизиты

  • Группа. Строка. Определяет группу замены (Обратимая, Необратимая, Множественная, Одиночная).
  • Количество. Число. Количество номенклатуры для замены.
