Регистр сведений "Параметры планирования"

Назначение

Регистр сведений Параметры планирования определяет набор необходимых операционных бюджетов для конкретных подразделений, а также порядок ввода конкретных видов бюджетов для подразделения за счет использования ресурса Порядок ввода, который нумеруется в пределах конкретного подразделения.

Порядок ввода имеет справочный характер и не обязателен для строгого соблюдения.

Измерения

  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник Подразделения компании. Определяет подразделение, для которого устанавливается порядок ввода бюджетов. 
  • Сценарий планирования. Реквизит является обязательным для заполнения. Ссылается на справочник Сценарии планирования. Сценарий планирования, в разрезе которого устанавливается порядок ввода.
  • Хоз. операция. Ссылается на справочник Хоз. операции. Вид бюджета, для которого устанавливается порядок ввода.

Ресурсы

  • Порядок ввода. Порядок ввода вида бюджета для подразделения компании в разрезе сценария планирования.

 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения и редактирования хранимой в регистре информации.
