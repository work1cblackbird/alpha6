Регистр сведений "Обмен с AudaPad Web"

Назначение

Регистр сведений "Обмен с AudaPad Web" предназначен для осуществления обмена с системой Audatex. Обмен основан на использовании веб-сервисов Audatex, для доступа к которым необходима учетная запись AudaPad Web. Получить данную учетную запись можно отправив заявку на приобретение доступа к Audatex.

Основной принцип работы с Audatex заключается в вводе необходимой информации непосредственно в AudaPad Web с последующей её загрузкой в документ "Заказ-наряд".

При работе с сервером Audatex обмен данными происходит через http (80) и https (443) tcp-порты. Для корректного обмена данными данные tcp-порты должны быть открыты в фаерволе (для клиент-серверного варианта информационной базы http/https порты должны быть открыты как на сервере, так и на клиенте).

Параметры

Периодичность:Периодический

Режим записи: Независимый

Измерения

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Пользователь, под учетной записью которого, осуществляется обмен данными.
  • Заказ-наряд. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ-наряд». Реквизит предназначен для хранения ссылки на документ, в который загружаются данные калькуляции.
  • Идентификатор дела. Тип значения: «Строка». Реквизит предназначен для хранения уникального идентификатора дела Audatex.
  • Идентификатор задания. Тип значения: «Строка». Реквизит предназначен для хранения уникального идентификатора задания Audatex.

Ресурсы

  • Калькуляция. Тип значения: «Хранилище значения». Реквизит содержит данные калькуляции, загруженные из Audatex.
  • Пользователь АХ. Тип значения: «Строка». Учетная запись пользователя Audatex.
  • Пароль АХ. Тип значения: «Строка». Пароль учетной записи Audatex.
  • Получатель АХ. Тип значения: «Строка». Учетная запись получателя задания АХ (страховой компании).

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма заявки на регистрацию. Форма ввода данных для формирования заявки на приобретение учетной записи Audatex.
  • Форма пользовательского сценария. Форма для осуществления обмена с Audatex по заданным сценариям.
  • Форма настроек. Форма для задания режимов поиска загружаемых данных и реквизитом по умолчанию для использования в документе-приемнике.

Способ вызова

Вызывается из меню "Сервис => Все операции -> Регистр сведений -> Обмен с AudaPad Web".
