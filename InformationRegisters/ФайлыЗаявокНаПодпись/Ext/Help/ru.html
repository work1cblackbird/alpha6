Регистр сведений "Файлы заявок на подпись"

Назначение

Регистр предназначен для хранения файлов документов Заявка на подпись.

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Заявка на подпись. Ссылается на документ Заявка на подпись. Документ, для которого загружен файл. 

Ресурсы

  • Данные файла. Тип: Хранилище значения. Данные файла документа.
  • Имя файла ПЭП. Тип: Строка. Представление имени файла в ПЭП.
  • Ссылка для скачивания в ПЭП. Тип: Строка. Ссылка для скачивания  текущего файла в ПЭП.
  • Хеш. Тип: Строка. Служебный ресурс для идентификации файла.
  • Идентификатор файла. Тип: Уникальный идентификатор. Служит для идентификации файла при выполнении операций.
