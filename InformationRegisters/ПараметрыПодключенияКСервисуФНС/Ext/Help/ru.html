Регистр сведений "Параметры подключения к сервису API ФНС"

Назначение

В регистре хранятся сведения о параметрах для подключения к отрытому API ФНС. Данные параметры используются для отправки запросов и получения ответов.

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник  «Организации». Организация, для которой получены параметры подключения к сервису ФНС. 

Ресурсы

  • Мастер-токен. Реквизит является обязательным для заполнения. Тип значения: «Строка». Токен, полученный от ФНС. Используется во всех запросах.
  • Тестовый контур. Тип значения: «Булево». Признак использования сервера отправки запросов: текстовый контур (koe-open-api.nalog.ru:443) или промышленный контур (openapi.nalog.ru:8090).
  • Время ожидания запроса. Тип значения: «Число». Время в секундах ожидания ответа от сервера ФНС на запрос. При создании новых настроек по умолчанию устанавливается 60.

Способ вызова

Доступен для просмотра и редактирования из настройки параметров учета.
