Регистр сведений "Цены"

Назначение

Предназначен для отражения ценовой политики компании. В нем хранится история цен номенклатуры в разрезе типов цен и поставщиков.

Параметры

Периодичность:Позиция регистратора

Режим записи: Подчинен регистратору

Измерения

  • Тип цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, для которого хранятся цены.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура, для которой сохраняются цены.
  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Поставщик, историю цен которого необходимо хранить. Если <пусто>, то цену установили мы.

Ресурсы

  • Цена. Тип значения: «Число». Цена.

Способ вызова

Доступен для просмотра из формы списка справочника «Номенклатура».
