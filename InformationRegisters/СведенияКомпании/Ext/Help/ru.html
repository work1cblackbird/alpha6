Регистр сведений "Сведения компании"

Назначение

Регистр предназначен для хранения некоторых важных сведений о компании, которые могут измениться со временем (например, гл. бухгалтер, руководитель...)

Параметры

Периодичность: День

Режим записи: Независимый

Измерения

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочники: «Контрагенты и контакты», «Организации». Организация.
  • Объект. Реквизит является обязательным для заполнения. Ссылается на перечисление "Виды объектов сведений". Вид сведения.

Ресурсы

  • Значение. Типы возможных значений: "Булево", "Дата", "Число", ссылка на перечисления: " Стратегия списания партий товаров по статусам", "Стратегия списания партий товаров по датам", ссылка на справочник: «Сотрудники». Значение сведения.

Способ вызова

Не предназначен для интерактивного вызова и редактирования.
