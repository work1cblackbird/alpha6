Регистр сведений "Применяемость дополнительных предложений по моделям"

Назначение

В регистре хранится состав дополнительных предложений для документа. Доступен просмотр и редактирование.


Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Модель. Ссылается на справочник «Модели автомобилей». Модель к которой относиться дополнительное предложение.
  • Дополнительное предложение. Реквизит является обязательным для заполнения. Ссылается на справочник «Автоработы».  Авторабота - дополнительное предложение.
  • GUID. Тип значения: «Строка». Уникальный идентификатор на мобильном приложении.

Ресурсы

  • Используется. Тип значения: «Булево». Признак использования.

Способ вызова

Вызывается из пункта меню "Сервис-> Все операции-> Регистры сведений-> Применяемость дополнительных предложений по моделям".
