Регистр сведений "Прайс-листы контрагентов. Временный"

Назначение

Регистр предназначен для загрузки и черновой обработки прайс-листов контрагентов.

Относится к подсистеме: «Рарус».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Прайс-лист контрагента. Реквизит является обязательным для заполнения. Ссылается на справочник: «Прайс-листы контрагентов». Прайс-лист контрагента.
  • Артикул для поиска. Тип значения: «Строка». Артикул для поиска номенклатуры.
  • Производитель. Ссылается на справочник «Производители». Распознанный производитель номенклатуры.
  • Дата записи. Тип значения: «Дата». Дата создания записи регистра.
  • Код предложения. Тип значения: «Число». Порядковый номер позиции в файле.

Ресурсы

  • Наименование номенклатуры. Тип значения: «Строка». Наименование позиции в прайс-листе.
  • Иностранное наименование. Тип значения: «Строка». Иностранное наименование позиции в прайс-листе.
  • Цена номенклатурной позиции. Тип значения: «Число». Цена номенклатурной позиции.
  • Количество номенклатуры. Составной тип значения: «Число», «Булево». Распознанное количество номенклатуры.
  • Представление срока поставки. Тип значения: «Строка». Представление срока поставки номенклатуры.
  • Запрет продажи. Тип значения: «Булево». Признак того, что данную номенклатуру запрещено продавать.
  • Запрет закупки. Тип значения: «Булево». Признак того, что данную номенклатуру запрещено закупать.
  • Кратность поставок. Тип значения: «Число». Отражает кратность поставок позиции из прайс-листа на данную номенклатуру.
  • Ключ строки. Тип значения: «Строка». Хеш от строки, собранной из полей, помеченных в структуре файла прайс-листа как "Ключевые".
  • Тег позиции (группа скидки и т.д.). Тип значения: «Строка». Используется в алгоритмах ценообразования, в частности для реализации с ее помощью механизма "Группы скидки", применяемыми оригинальными поставщиками.
  • Валюта. Ссылается на справочник «Валюты». Валюта цены в прайс-листе. Если не указана в файле загрузки, то используется валюта, указанная в самом прайс-листе контрагента.
  • Вес. Тип значения: «Число». Вес позиции прайс-листа.
  • Единица измерения. Ссылается на справочник «Единицы измерения». Единица измерения, к которой сопоставлена данная строка прайс-листа.
  • Минимальный срок поставки номенклатуры. Тип значения: «Число». Минимальный срок поставки до склада поставщика (в секундах). В нем хранятся разобранные данные из поля "Срок поставки в прайс-листе".
  • Максимальный срок поставки номенклатуры. Тип значения: «Число». Максимальный срок поставки до склада поставщика (в секундах). В нем хранятся разобранные данные из поля "Срок поставки в прайс-листе".
  • Номенклатура прайс-листа. Ссылается на справочник «Номенклатура». Сопоставленная строке прайс-листа позиция номенклатуры.

Реквизиты

  • № по каталогу. Тип значения: «Строка». Артикул номенклатуры.
  • Производитель в прайс-листе. Тип значения: «Строка». Производитель, как он указан в прайс-листе.
  • Артикул в прайс-листе. Тип значения: «Строка». Артикул как он указан в прайс-листе.
  • Количество в прайс-листе. Тип значения: «Строка». Количество номенклатуры как оно указано в прайс-листе.
  • Срок поставки в прайс-листе. Тип значения: «Строка». Представление срока поставки номенклатуры как он есть в прайс-листе.
  • Остальные поля в прайс листе. Тип значения: «ХранилищеЗначения». Остальные поля, хранящиеся в строке прайс-листа.
  • Запрет загрузки. Тип значения: «Булево». Признак строки, которую не нужно загружать в основной прайс-лист.

Способ вызова

Не предназначен для интерактивного вызова и редактирования.
