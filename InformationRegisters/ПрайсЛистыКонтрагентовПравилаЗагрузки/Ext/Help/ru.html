Регистр сведений "Прайс-листы контрагентов. Правила загрузки"

Назначение

Регистр предназначен для назначения правил загрузки прайс-листов контрагентов.

Относится к подсистеме: «Рарус».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Прайс-лист контрагента. Реквизит является обязательным для заполнения. Ссылается на справочник: «Прайс-листы контрагентов». Прайс-лист контрагента.
  • Назначение правила. Реквизит является обязательным для заполнения. Ссылается на перечисление: «Назначение правил загрузки». Определяет назначение правила загрузки прайс-листа контрагента.
  • Объект правила. Составной тип значения: «Строка», ссылается на справочник «Производители». Строка производителя, производитель или ключ строки.
  • Идентификатор правила. Тип значения: «Уникальный идентификатор». Предназначен для возможности создания различных правил для одной и той же строки прайс-листа.

Ресурсы

  • Вид правила. Ссылается на перечисление: «Виды правил загрузки». Вид правила загрузки прайс-листа контрагента.
  • Имя реквизита прайс-листа. Тип значения: «Строка». Имя реквизита прайс-листа, подлежащего обработке по правилу.
  • Значение. Составной тип значения: «Число», «Булево», «Строка», Ссылается на справочники: «Производители», «Номенклатура», «Единицы измерения», «Валюты». Присваиваемое реквизиту значение.

Реквизиты

  • Порядок применения. Тип значения: «Число». Последовательность применения правил загрузки (В пределах “Объекта правила”).

Способ вызова

Не предназначен для интерактивного вызова и редактирования.
