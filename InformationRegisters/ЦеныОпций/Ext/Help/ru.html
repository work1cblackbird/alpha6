Регистр сведений "Цены опций"

Назначение

Предназначен для отражения ценовой политики компании на опции автомобилей. В нем хранится история цен опций в разрезе типов цен.

Параметры

Периодичность:Регистратор

Режим записи: Подчинен регистратору

Измерения

  • Тип цен. Ссылается на справочник «Типы цен». Тип цен для которого хранятся цены.
  • Модель. Ссылается на справочник «Модели автомобилей». Модель автомобиля, для которой сохраняются цены на опции.
  • Вариант комплектации. Ссылается на справочник «Варианты комплектации». Вариант комплектации модели автомобиля, для которой сохраняются цены на опции. 
  • Опция. Ссылается на справочники «Опции автомобилей». Опция автомобиля, для которой сохраняются цены.
