Регистр сведений "Ограничение ассортимента подразделения"

Назначение

Предназначен для хранения ассортимента товаров разрешенных к приемке на данное подразделение.

Измерения

  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение, для которого формируем ассортимент.
  • Номенклатура. Ссылается на справочник «Номенклатура». Товар, входящий, в ассортимент подразделения.

Ресурсы

  • Состояние. Ссылается на перечисление «Состояния товара в ассортименте подразделения». Состояние товара в ассортименте подразделения.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Процент наценки. Тип значения: «Число». Процент наценки для автоматического формирования цены продажи.
  • Цена розничная. Тип значения: «Число». Розничная цена для переоценки остатков перед списанием.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • Базовый ассортимент. Тип значения: «Булево». Товар вводится в базовый ассортимент.

Способ вызова

Интерактивного вызова не предусмотрено.
