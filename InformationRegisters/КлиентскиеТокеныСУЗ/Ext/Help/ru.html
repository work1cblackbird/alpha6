Регистр сведений "Токены устройств станции управления заказами"

Назначение

Регистр сведений "Токены устройств станции управления заказами" содержит список полученных ключей сессий для формирования заказа кодов маркировки и его получения из ГИС МТ. Используется при наличии настройки обмена через API СУЗ.

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник Организации. Организация, по которой настроен обмен с ГИС МТ. 
  • Идентификатор соединения. Реквизит является обязательным для заполнения. Тип значения: «Строка». Идентификатор соединения устройства, по которому получен клиентский токен.

Ресурсы

  • Токен. Реквизит является обязательным для заполнения. Тип значения: «Строка». Полученный клиентский токен для формирования запроса.
  • Срок действия. Тип значения: «Дата». Значение рассчитанного срока действия токена для проверки необходимости получения нового.

Способ вызова

Данный регистр является служебным и не выводится в подсистеме.
