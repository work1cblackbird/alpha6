Регистр сведений "Цены авторабот"

Назначение

Предназначен для отражения ценовой политики компании по выполнению авторабот. В нем хранится история цен авторабот в разрезе классов автомобилей, договоров взаиморасчетов, цехов и видов ремонта. Цена автоработы может указываться или в стоимости одного часа выполнения автоработы или в валюте.

Параметры

Периодичность:Позиция регистратора

Режим записи: Подчинен регистратору

Измерения

  • Тип цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, для которого хранятся цены.
  • Авторабота. Реквизит является обязательным для заполнения. Ссылается на справочник «Автоработы». Авторабота или группа авторабот, для которых сохраняются цены.
  • Модель. Ссылается на справочник «Модели». Модель или группа моделей, для которой сохраняются цены авторабот.
  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Контрагент, историю цен которого необходимо хранить.
  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с контрагентом, для которого сохраняются цены авторабот.
  • Цех. Ссылается на справочник «Цеха». Цех, для которого сохраняются цены авторабот. 
  • Вид ремонта. Ссылается на справочник «Виды ремонта». Вид ремонта, для которого сохраняются цены авторабот.

Ресурсы

  • Нормочас. Ссылается на справочник «Нормочасы». Стоимость одного часа выполнения автоработы.
  • Валюта. Ссылается на справочник «Валюты». Валюта, в которой задана цена автоработы.
  • Цена. Тип значения: «Число». Цена автоработы в указанной валюте.

Способ вызова

Доступен для просмотра из формы списка справочника «Автоработы».
