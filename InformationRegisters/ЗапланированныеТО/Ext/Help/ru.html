Регистр сведений "Запланированные ТО"

Назначение

Регистр предназначен для хранения информации о планировании проведения технического осмотра автомобиля.

Параметры

Периодичность: В пределах секунды 

Режим записи: Независимый

Измерения

  • Автомобиль. Ссылается на справочник  «Автомобили». Автомобиль, для которого определяется опция.

Ресурсы

  • Пробег следующего ТО. Тип значения: «Число». Запланированный пробег следующего ТО.
  • Дата следующего ТО. Тип значения: «Дата». Запланированная дата следующего ТО.

Реквизиты

  • Автор. Ссылается на справочник «Пользователи». Реквизит предназначен для хранения ссылки на автора записи - пользователя, который создал запись.

  • Источник. Тип значения: «ДокументСсылка.ЗаказНаряд».

  • Представление источника. Тип значения: «Строка». 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения и редактирования хранимой в регистре информации. 
  • Форма записи. Форма записи регистра.
