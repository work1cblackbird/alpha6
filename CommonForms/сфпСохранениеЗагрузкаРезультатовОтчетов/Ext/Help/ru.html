Форма"Сохранение результатов отчета"

Обработка предназначена для сохранения и последующего использования результата сформированного отчета в других отчетах или для заполнения документов.  Сохраненные результаты отчетов храниться в информационной базе и доступны только в текущем сеансе работы.

 

Форма сохранения вызывается по кнопке «Сохранить результат» в форме отчетов.

 

Сохраненные данные можно использовать в качестве значения отбора в отчетах построенных на базе универсального отчета.  Загрузка результатов доступна для отборов,  у которых установлен тип сравнения «В списке»  или «Не в списке».

 

Сохраненные результаты отчетов можно использовать в качестве источник данных для заполнения табличных частей документов: «Формирование анкет», «Событие», «Телемаркетинг». Мастер заполнения вызывается кнопкой «Заполнить по результатам отчетов».

 

Обработка имеет два режима работы:

"Сохранение результата отчета"

В поле "Наименование" необходимо указать имя сохраняемой настройки (имя можно сгенерировать автоматически).   Также в данном поле можно выбрать ранее сохраненный результат для перезаписи.

 

Список сохраняемых значений будет заполнен автоматически при выборе сохраняемых полей.  В списке доступных полей отчета доступно групповое выделение, т.е. если необходимо сохранить значения нескольких полей, то можно сначала их выбрать, а потом  перенести их в сохраняемые поля.

 

"Заполнение из результата отчета"

В поле "Наименование" необходимо выбрать ранее сохраненный результат какого-либо отчета.

 

После выбора сохраненной настройки система автоматически сопоставит колонки сохраненных значений и  колонки приемника по типу хранимых данных. При необходимости можно указать соответствие вручную в форме соответствия полей открываемой кнопкой «Настройка».
