Общая форма "Параметры документа"

Назначение

Форма используется, когда необходимо изменить для выбранного документа следующие реквизиты: организация, подразделение, отношение документа к регламентированному учету, автор, менеджер, проект, документ-основание, штрихкод, валюту, тип цены или скидку. Форма и вызов един для всех документов. Если для отдельного вида документа какой-либо из перечисленных реквизитов не используется или запрещен для изменения, то при открытии он автоматически скрывается или блокируется.

Реквизиты

  • Создание.  Момент времени первой записи документа.
  • Проведение.  Момент времени последнего проведения документа.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете. 
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Менеджер. Ссылается на справочник «Сотрудники». Сотрудник, заключивший сделку с контрагентом.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.
  • Документ-основание. Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу-основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен закупки.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Валюта упр. учета. Ссылается на справочник «Валюты». Валюта документа для ведения управленческого учета.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").  
  • Валюта договора. Ссылается на справочник «Валюты». Валюта по которой будут вестись взаиморасчеты.
  • Курс валюты договора. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты договора.
  • Карточка. Ссылается на справочник «Карточки». Карточка покупателя, по которой продавали товар.
  • Скидка/наценка. Ссылается на справочник «Типы скидок и наценок». Скидка/наценка документа.
  • Скидка/наценка на работы. Ссылается на справочник «Типы скидок и наценок». Скидка/наценка на работы документа.
  • Штрихкод. Позволяет назначить или изменить штрихкод документа, если включено штрихкодирование печатных форм.
