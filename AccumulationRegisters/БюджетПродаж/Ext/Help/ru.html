Регистр накопления "Бюджет продаж"

Назначение

 Оборотный регистр "Бюджет продаж" предназначен для накопления информации по спланированным данным продаж в разрезе номенклатуры, либо категорий номенклатуры.

Измерения

  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, для которого спрогнозированы данные.
  • Сценарий планирования. Реквизит является обязательным для заполнения. Ссылается на справочник «Сценарии планирования». Сценарий планирования, в разрезе которого было произведено планирование.
  • Номенклатура. Реквизит является обязательным для заполнения. Имеет составной тип. Ссылается на справочник «Номенклатура», либо на справочник «Типы номенклатуры». Спрогнозированный товар или категория номенклатуры. 
  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция, определяющая вид бюджета: бюджет продаж по номенклатуре, либо бюджет продаж по категориям. 

Ресурсы

  • Количество. Тип значения: «Число». Количество спрогнозированного товара. Имеет смысл только для бюджета продаж по номенклатуре. 
  • Сумма рег.. Тип значения: «Число». Спрогнозированная сумма продаж в валюте регламентированного учета.
  • Сумма упр.. Тип значения: «Число». Спрогнозированная сумма продаж в валюте управленческого учета.
  • Себестоимость упр.. Тип значения: «Число». Спрогнозированная себестоимость продаж в валюте управленческого учета.
  • Сумма НДС. Тип значения: «Число». Спрогнозированная сумма НДС от продаж в валюте регламентированного учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.
