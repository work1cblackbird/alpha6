Регистр накопления "Выработка сотрудников"

Назначение

Регистр «Выработка сотрудников» служит для быстрого получения информации о количестве выполненных работ и суммы выполнения работ тем или иным исполнителем. При этом учитывается вид производимого ремонта (платный или бесплатный) и сумма скидки, предоставленная на выполнение ремонта.

Измерения

  • Сотрудник. Ссылается на справочник «Сотрудники». Сотрудник, выполнявший работы по заказ-наряду.
  • Работа. Ссылается на справочник «Автоработы». Авторабота, выполненная сотрудником.
  • Вид ремонта. Ссылается на справочник «Виды ремонта». Вид ремонта заказ-наряда.
  • Цех. Ссылается на справочник «Цеха». Цех выполнения работы.
  • Автомобиль. Ссылается на справочник «Автомобили». Ремонтируемый автомобиль.
  • Вид нормочаса. Ссылается на справочник «Нормочасы». Нормочас выполнения автоработы.

Ресурсы

  • Количество. Тип значения: «Число». Количество выполненных работ.
  • Количество нормочасов. Тип значения: «Число». Количество нормочасов затраченных при выполнении работ.
  • Сумма (регл.). Тип значения: «Число». Сумма  в валюте регламентного учета с учетом налогов.
  • Сумма (упр.). Тип значения: «Число». Сумма управленческого учета с учетом налогов.
  • Сумма НДС (регл.). Тип значения: «Число». Сумма НДС в валюте регламентного учета.
  • Сумма НДС (упр.). Тип значения: «Число». Сумма НДС в валюте управленческого учета.
  • Выручка без НДС (регл.). Тип значения: «Число». Сумма выполненных работ  в валюте регламентного учета без учета суммы НДС.
  • Выручка без НДС (упр.). Тип значения: «Число». Сумма выполненных работ  в валюте управленческого учета без учета суммы НДС.
  • Выручка до применения скидки (регл.). Тип значения: «Число». Сумма выполненных работ  в валюте регламентного учета с учетом налогов и без учета суммы скидки.
  • Выручка до до применения скидки (упр.). Тип значения: «Число». Сумма выполненных работ  в валюте управленческого учета без учета с учетом налогов и без учета суммы скидки.
  • Выручка до применения скидки без НДС (регл.). Тип значения: «Число». Сумма выполненных работ  в валюте регламентного учета без учета суммы НДС и без учета суммы скидки.
  • Выручка до применения скидки без НДС (упр.). Тип значения: «Число». Сумма выполненных работ  в валюте управленческого учета без учета суммы НДС и без учета суммы скидки.

Реквизиты

  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.

Способ вызова

Из меню "Перейти" формы документа «Заказ-наряд» возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Заказ-наряд».
