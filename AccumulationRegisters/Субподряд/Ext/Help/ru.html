Регистр накопления "Субподряд"

Назначение

Для учета выполнения субподрядных работ служит оборотный регистр «Субподряд». В регистре накапливаются суммы выполненных работ и их перепродажи в разрезе дополнительной информации.

Измерения

  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был зафиксирован факт получения/реализации работы.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Субподрядчик, выполняющий работы.
  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с субподрядчиком.
  • Заказ-наряд. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ-наряд». Заказ-наряд, в рамках которого выполнены субподрядные работы.
  • Работа. Реквизит является обязательным для заполнения. Ссылается на справочники «Автоработы» и «Номенклатура». Авторабота или услуга, которая выполняется по субподряду.
  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.

Ресурсы 

  • Сумма. Тип значения: «Число». Сумма реализации субподрядных работ в регламентированной валюте с учетом налогов.  
  • Сумма НДС. Тип значения: «Число». Сумма НДС реализации субподрядных работ в валюте регламентного учета.
  • Сумма упр.. Тип значения: «Число». Сумма реализации субподрядных работ в валюте управленческого учета с учетом налогов.
  • Себестоимость регл.. Тип значения: «Число». Себестоимость субподрядной работы в валюте регламентированного учета с учетом налогов.
  • Сумма НДС входящий. Тип значения: «Число». Себестоимость НДС субподрядных работ в валюте регламентного учета.
  • Себестоимость упр.. Тип значения: «Число». Себестоимость субподрядной работы в валюте управленческого учета с учетом налогов.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Поступление товаров», «Заказ-наряд».
