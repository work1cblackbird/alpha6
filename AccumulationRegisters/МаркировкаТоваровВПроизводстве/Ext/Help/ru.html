Регистр накопления "Маркировка товаров в производстве"

Назначение

Для учета деталей с маркировкой, перемещенных в производство для выполнения работ по заказ-нарядам, в программе присутствует регистр «Маркировка товаров в производстве». Смысл остатков на регистре: это остатки деталей с маркировкой, находящихся в производстве для выполнения работ по заказ-нарядам, но работы по которым еще не завершены. Отрицательные остатки на регистре не имеют смысла.

Измерения

  • Заказ-наряд. Ссылается на документ «Заказ-наряд». Заказ-наряд, детали на выполнение которого были перемещены в производство. 
  • Цех. Ссылается на справочник «Цеха». Цех, в который были перемещены детали для выполнения работ по заказ-наряду. 
  • Номенклатура. Ссылается на справочник «Номенклатура». Деталь с маркировкой, перемещенная в производство. 
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика детали с маркировкой, перемещенной в производство. 
  • Код маркировки. Тип значения: «Строка». Полное представление маркировки детали.
  • ГТД. Ссылается на справочник «Грузовые таможенные декларации»

Ресурсы

  • Количество. Тип значения: «Число». Количественный остаток детали с маркировкой в производстве.

Дополнительная информация

Документы, порождающие движения: «Заказ-наряд», «Извлечение товаров из производства», «Перемещение товаров в производство».
