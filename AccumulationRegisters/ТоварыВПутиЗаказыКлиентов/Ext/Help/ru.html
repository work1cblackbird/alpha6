Регистр накопления "Товары в пути (заказы клиентов)"

Назначение

Для учета товаров, которые находятся в пути из филиала в филиал в разрезе заказов клиентов, в программе присутствует регистр «Товары в пути (заказы клиентов)». 

Измерения

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Организация. 
  • Отправитель. Реквизит является обязательным для заполнения. Ссылается либо на справочник «Склады компании», либо на справочник «Подразделения компании». Филиал организации, из которого отправили товар.
  • Получатель. Ссылается либо на справочник «Склады компании», либо на справочник «Подразделения компании». Филиал организации, в который отправили товар. 
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура. 
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры.
  • Заказ. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ внутренний», Ссылается на документ «Заказ покупателя». Документ оформления заказа.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочники: «Контрагенты и контакты», «Подразделения компании». Контрагент, разместивший заказ.

Ресурсы

  • Заказано. Тип значения: «Число». Количество номенклатуры по заказу. 
  • Резерв. Тип значения: «Число». Количество номенклатуры, зарезервированной на складе для этого заказа.  
  • Резерв из свободного запаса. Тип значения: «Число». Количество номенклатуры, зарезервированной на складе для этого заказа из свободного запаса. 
  • Сумма в валюте договора. Тип значения: «Число». Сумма заказанной номенклатуры в валюте договора.
  • Сумма в валюте управленческого учёта. Тип значения: «Число». Сумма заказанной номенклатуры в валюте управленческого учета.

Способ вызова

Из меню "Перейти" формы документа «Перемещение товаров» возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Перемещение товаров».
