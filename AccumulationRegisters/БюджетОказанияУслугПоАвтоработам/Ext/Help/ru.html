Регистр накопления "Бюджет оказания услуг по автоработам"

Назначение

Оборотный регистр "Бюджет оказания услуг по автоработам " предназначен для накопления информации по спланированным данным работ.

Измерения

  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, для которого спрогнозированы данные закупок. 
  • Сценарий планирования. Реквизит является обязательным для заполнения. Ссылается на справочник «Сценарии планирования». Сценарий планирования, в разрезе которого было произведено планирование.
  • Авторабота. Реквизит является обязательным для заполнения. Ссылается на справочник «Автоработы». Спрогнозированная работа.

Ресурсы

  • Количество. Тип значения: «Число». Количество спрогнозированных работ.
  • Сумма рег.. Тип значения: «Число». Спрогнозированная сумма оплаты за работу в валюте регламентированного учета.
  • Сумма упр.. Тип значения: «Число». Спрогнозированная сумма оплаты за работу в валюте управленческого учета.
  • Сумма НДС. Тип значения: «Число». Спрогнозированная сумма НДС по оплате за работу в валюте регламентированного учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Бюджет оказания услуг по автоработам».
