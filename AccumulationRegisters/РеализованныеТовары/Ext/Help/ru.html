Регистр накопления "Реализованные товары"

Назначение

Для учета проданных товаров, ранее принятых на реализацию, за которые необходимо отчитаться перед поставщиком в программе присутствует регистр «Реализованные товары» (не балансовый). Смысл остатков на регистре: это остатки товаров, принятых ранее на реализацию от поставщика и проданных, но по которым еще не сформирован отчет. Отрицательные остатки на регистре не имеют смысл.

Измерения

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Комитент, чей товар был продан.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура.
  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор комиссии с комитентом.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры.
  • Документ передачи. Ссылается на документ «Поступление товаров». Документ, по которому к нам поступил проданный товар.

Ресурсы

  • Количество. Тип значения: «Число». Количество проданного товара.
  • Сумма (регл.). Тип значения: «Число». Стоимость товаров по закупке в валюте регламентированного учета.
  • Сумма НДС (регл.). Тип значения: «Число». Сумма НДС в валюте регламентированного учета.
  • Сумма без НДС (регл.). Тип значения: «Число». Сумма в валюте регламентированного учета без налогов.
  • Сумма (упр.). Тип значения: «Число». Стоимость товаров по закупке в валюте управленческого учета.
  • Сумма НДС (упр.). Тип значения: «Число». Сумма НДС в валюте управленческого учета.
  • Сумма без НДС (упр.). Тип значения: «Число». Сумма в валюте управленческого учета без налогов.  
  • Сумма продажи (упр.). Тип значения: «Число». Сумма, по которой товар был продан в валюте управленческого учета.
  • Сумма продажи (регл.). Тип значения: «Число». Сумма, по которой товар был продан в валюте регламентированного учета.

Реквизиты

  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.
  • Вознаграждение (упр.). Тип значения: «Число». Сумма комиссионного вознаграждения в валюте управленческого учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Возврат от покупателя», «Инвентаризация», «Отчет комитенту», «Реализация товаров», «Списание товаров».
