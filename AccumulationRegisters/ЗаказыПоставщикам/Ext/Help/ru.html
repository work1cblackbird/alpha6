Регистр накопления "Заказы поставщикам"

Назначение

Для учета заказов поставщикам служит одноименный регистр (не балансовый). Смысл остатков на регистре следующий: это остатки невыполненных заказов поставщикам. Отрицательные остатки на регистре не имеют смысла.

Измерения

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочники: «Контрагенты и контакты», «Подразделения компании». Поставщик заказа.
  • Заказ поставщику. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ поставщику». Документ оформления заказа.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура заказа.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры заказа.

Ресурсы

  • Заказано. Тип значения: «Число». Количество номенклатуры по заказу.
  • Сумма в валюте управленческого учёта. Тип значения: «Число». Сумма заказанной номенклатуры в валюте управленческого учета.

Реквизиты

  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.
  • Сумма в валюте договора. Тип значения: «Число». Сумма заказанной номенклатуры в валюте договора.
  • Сумма в валюте управленческого учёта. Тип значения: «Число». Сумма заказанной номенклатуры в валюте управленческого учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Заказ поставщику», «Корректировка заказа поставщику», «Поступление товаров».
