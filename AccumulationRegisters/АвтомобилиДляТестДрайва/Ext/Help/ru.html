Регистр накопления "Автомобили для тест-драйва"

Назначение

Данный регистр предназначен для хранения набора автомобилей доступный для тест-драйва в разрезе подразделений.

Измерения

  • Подразделение компании. Ссылается на справочник «Подразделения компании». Подразделение компании к которому приписан автомобиль.
  • Модель. Ссылается на справочник «Модели автомобилей». Модель автомобиля.
  • Автомобиль. Ссылается на справочник «Автомобили». Автомобиль.
  • Актив. Ссылается на справочник «Прочие активы». Актив к которому привязан автомобиль.

Ресурсы

  • Количество. Тип значения: «Число». Количественный остаток автомобилей на складе.

Реквизиты

  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.
  • Максимальный пробег. Тип значения: «Число».  Максимальный пробег автомобиля на тест-драйве.
  • Максимальная продолжительность эксплуатации.  Тип значения: «Дата». Дата до которой возможно использование автомобиля для тест-драйва.
