Регистр накопления "Продажи автомобилей"

Назначение

Для учета торговой деятельности  по продаже автомобилей компании служит оборотный регистр «Продажи автомобилей». В регистре накапливается количество и суммы продаж автомобилей в разрезе дополнительной информации.

Измерения

  • Подразделение компании. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был зафиксирован факт продажи автомобиля.
  • Автомобиль. Ссылается на справочник «Автомобили». Проданный автомобиль.
  • Поставщик. Ссылается на справочник «Контрагенты и контакты». Поставщик, у которого автомобиль был куплен.
  • Покупатель. Ссылается на справочник «Контрагенты и контакты». Покупатель, который купил автомобиль.
  • Статус партии. Ссылается на перечисление «Статусы партий». Статус партии, определяющий, чей автомобиль мы продали (собственный компанией или принятый на комиссию).
  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.
  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с покупателем.
  • Склад компании. Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого был продан автомобиль.

Ресурсы

  • Количество. Тип значения: «Число». Количество проданных автомобилей.
  • Сумма. Тип значения: «Число». Сумма с учетом налогов.
  • Сумма НДС. Тип значения: «Число». Сумма НДС в валюте регламентного учета.
  • Выручка без НДС. Тип значения: «Число». Сумма в валюте регламентированного учета без налогов.
  • Сумма скидки. Тип значения: «Число». Сумма скидки, предоставленная при продаже автомобиля.
  • Выручка до применения скидки с НДС. Тип значения: «Число». Сумма в валюте регламентированного учета без учета суммы скидки и с учетом налогов.
  • Выручка до применения скидки без НДС. Тип значения: «Число». Сумма в валюте регламентированного учета без учета суммы скидки и без налогов.
  • Сумма (упр.). Тип значения: «Число». Сумма управленческого учета с учетом налогов.
  • Сумма НДС (упр.). Тип значения: «Число». Сумма НДС управленческого учета с учетом налогов.
  • Выручка без НДС (упр.). Тип значения: «Число». Сумма управленческого учета без налогов.
  • Сумма скидки (упр.). Тип значения: «Число». Сумма скидки управленческого учета , предоставленная при продаже автомобиля.
  • Выручка до применения скидки с НДС (упр.). Тип значения: «Число». Сумма управленческого учета без учета суммы скидки и с учетом налогов.
  • Выручка до применения скидки без НДС (упр.). Тип значения: «Число». Сумма управленческого учета без учета суммы скидки и без налогов.
  • Себестоимость(регл.). Тип значения: «Число». Себестоимость регл. учета с учетом налогов.
  • Сумма НДС входящий (регл.). Тип значения: «Число». Сумма входящего НДС в валюте регл. учета.
  • Себестоимость без НДС (регл.). Тип значения: «Число». Себестоимость регл. учета без налогов. 
  • Себестоимость (упр.). Тип значения: «Число». Себестоимость упр. учета с учетом налогов.  
  • Сумма НДС входящий (упр.). Тип значения: «Число». Сумма НДС себестоимости товаров в валюте управленческого учета.
  • Себестоимость без НДС (упр.). Тип значения: «Число». Сумма входящего НДС в валюте упр. учета.

Дополнительная информация

Из меню "Перейти" форм списков документов и форм документов возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Возврат автомобилей от покупателя», «Отчет комиссионера за автомобили», «Реализация автомобилей».
