Регистр накопления "Партии товаров отданные"

Назначение

В программе ведется партионный учет товаров, переданных на реализацию в разрезе «Контрагент -- Номенклатура -- Партия», реализованный на балансовом регистре «Партии товаров отданные». Остатки на регистре имеют простой смысл - это остатки товаров, переданных на реализацию, за которые реализатор не отчитался (которые реализатор еще не продал). Отрицательные остатки на регистре не имеют смысла. Хозяйственные операции по передачи товаров на реализацию приводят к увеличению остатков на регистре, хозяйственные операции по отчету за реализованный товар приводят к уменьшению остатков на регистре.

Измерения

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Комиссионер, которому мы передали товар на комиссию.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура, переданная на комиссию.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры.
  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор комиссии.
  • Документ передачи. Ссылается на документы: «Инвентаризация», «Реализация товаров». Документ передачи товаров на комиссию.
  • Партия. Ссылается на документы: «Инвентаризация», «Комплектация», «Поступление товаров», «Разукомплектация». Документ, по которому наша компания купила данный товар.

Ресурсы

  • Количество. Тип значения: «Число». Количество переданного на комиссию товара.
  • Сумма (регл.). Тип значения: «Число». Сумма в валюте регламентированного учета с налогами.
  • Сумма НДС (регл.). Тип значения: «Число». Сумма НДС в валюте регламентированного учета.
  • Сумма себестоимости (регл.). Тип значения: «Число». Себестоимость товаров, переданных на реализацию, в валюте регламентированного учета.
  • Сумма себестоимости НДС (регл.). Тип значения: «Число». Сумма НДС себестоимости товаров в валюте регламентированного учета.
  • Сумма себестоимости без НДС (регл.). Тип значения: «Число». Сумма себестоимости товаров в валюте регламентированного учета без налогов.
  • Сумма (упр.). Тип значения: «Число». Сумма в валюте управленческого учета с налогами.
  • Сумма НДС (упр.). Тип значения: «Число». Сумма НДС в валюте управленческого учета.
  • Сумма себестоимости (упр.). Тип значения: «Число». Себестоимость товаров, переданных на реализацию, в валюте управленческого учета.
  • Сумма себестоимости НДС (упр.). Тип значения: «Число». Сумма НДС себестоимости товаров в валюте управленческого учета.
  • Сумма себестоимости без НДС (упр.). Тип значения: «Число». Сумма себестоимости товаров в валюте управленческого учета без налогов.

Реквизиты

  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.
  • Сумма продажи (упр.). Тип значения: «Число». Сумма в валюте управленческого учета с налогами. 

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Возврат от покупателя», «Инвентаризация», «Отчет комиссионера», «Переоценка», «Реализация товаров», «Списание товаров».
