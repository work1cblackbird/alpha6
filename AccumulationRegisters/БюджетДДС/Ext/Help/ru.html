Регистр накопления "Бюджет движения денежных средств"

Назначение

Оборотный регистр "Бюджет движения денежных средств" предназначен для накопления информации по спланированным данным движения денежных средств в разрезе статей ДДС.

Измерения

  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, для которого спрогнозированы данные.
  • Сценарий планирования. Реквизит является обязательным для заполнения. Ссылается на справочник «Сценарии планирования». Сценарий планирования, в разрезе которого было произведено планирование.
  • Статья ДДС. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи ДДС». Статья ДДС, для которой спрогнозировано движение. 

Ресурсы

  • Сумма рег.. Тип значения: «Число». Спрогнозированная сумма движения денежных средств в валюте регламентированного учета.
  • Сумма упр.. Тип значения: «Число». Спрогнозированная сумма движения денежных средств в валюте управленческого учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Бюджет ДДС».
