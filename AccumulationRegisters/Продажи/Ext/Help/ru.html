Регистр накопления "Продажи"

Назначение

Для учета торговой деятельности компании служит оборотный регистр «Продажи». В регистре накапливается количество и суммы продаж в разрезе дополнительной информации, по расходным документам и заказам покупателей, где упомянут источник рекламы.

Измерения

  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был зафиксирован факт продажи товара.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Проданный товар или оказанная услуга.
  • Поставщик. Ссылается на справочник «Контрагенты и контакты». Поставщик, у которого товар был куплен.
  • Покупатель. Ссылается на справочник «Контрагенты и контакты». Покупатель, который купил товар или которому была оказана услуга.
  • Статус партии. Ссылается на перечисление «Статусы партий». Статус партии, определяющий, чей товар мы продали (собственный компанией или принятый на комиссию).
  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.
  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с покупателем.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры.
  • Склад компании. Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого товар был продан.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС.
  • Партия. Ссылается на документы: «Инвентаризация», «Комплектация», «Поступление товаров», «Разукомплектация». Документ, по которому компания продала данный товар.

Ресурсы

  • Количество. Тип значения: «Число». Количество проданного товара.
  • Количество нормочасов. Тип значения: «Число». Количество нормочасов затраченных при выполнении работ.
  • Сумма (регл.). Тип значения: «Число». Сумма в валюте регламентированного учета с учетом налогов.
  • Сумма НДС (регл.). Тип значения: «Число». Сумма НДС в валюте регламентированного учета.
  • Выручка без НДС (регл.). Тип значения: «Число». Выручка без НДС в валюте регламентированного учета.
  • Сумма скидки (регл.). Тип значения: «Число». Сумма скидки в валюте регламентированного учета.
  • Выручка до применения скидки с НДС (регл.). Тип значения: «Число». Выручка до применения скидки с НДС в валюте регламентированного учета.
  • Выручка до применения скидки без НДС (регл.). Тип значения: «Число». Выручка до применения скидки с НДС в валюте регламентированного учета без налога.
  • Себестоимость (регл.). Тип значения: «Число». Себестоимость в валюте регламентированного учета с учетом налога.
  • Сумма НДС входящий (регл.). Тип значения: «Число». Сумма входящего НДС в валюте регламентированного учета.
  • Себестоимость без НДС (регл.). Тип значения: «Число». Себестоимость в валюте регламентированного учета без учета налогов.
  • Сумма (упр.). Тип значения: «Число». Сумма в валюте управленческого учета с учетом налога.
  • Сумма НДС (упр.). Тип значения: «Число». Сумма НДС в валюте управленческого учета.
  • Выручка без НДС (упр.). Тип значения: «Число». Выручка без НДС в валюте управленческого учета.
  • Сумма скидки (упр.). Тип значения: «Число». Сумма скидки в валюте управленческого учета.
  • Себестоимость (упр.). Тип значения: «Число». Себестоимость в валюте управленческого учета с учетом налога.
  • Выручка до применения скидки с НДС (упр.). Тип значения: «Число». Выручка до применения скидки с НДС в валюте управленческого учета.
  • Выручка до применения скидки без НДС (упр.). Тип значения: «Число». Выручка до применения скидки в валюте управленческого учета без налога.
  • Сумма НДС входящий (упр.). Тип значения: «Число». Сумма входящего НДС в валюте управленческого учета.
  • Себестоимость без НДС (упр.). Тип значения: «Число». Себестоимость в валюте управленческого учета без учета налогов.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Возврат от покупателя», «Отчет комиссионера», «Реализация товаров».
