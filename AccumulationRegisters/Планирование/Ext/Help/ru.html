Регистр накопления "Планирование"

Назначение

Предназначен для накопления данных по всем планам и фактам. Регистраторы - документы «План» и «Факт».

Измерения

  • Вид плана. Ссылается на справочник «Виды планов компании». Вид хранимого плана / факта.
  • Вид аналитики планирования 1. Ссылается на характеристики «Аналитические разрезы планирования». Значение первого аналитического реквизита таблицы.
  • Вид аналитики планирования 2. Ссылается на характеристики «Аналитические разрезы планирования». Значение второго аналитического реквизита таблицы.
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение, по которому хранится план фактные данные.

Ресурсы

  • Показатель плана 1. Тип значения: «Число». Значение первого числового показателя плана.
  • Показатель плана 2. Тип значения: «Число». Значение второго числового показателя плана.
  • Показатель плана 3. Тип значения: «Число». Значение третьего числового показателя плана.
  • Показатель факта 1. Тип значения: «Число». Значение первого числового показателя факта.
  • Показатель факта 2. Тип значения: «Число». Значение второго числового показателя факта.
  • Показатель факта 3. Тип значения: «Число». Значение третьего числового показателя факта.

Реквизиты

  • Дата начала. Тип значения: «Дата». Дата начала планирования.
  • Дата конца. Тип значения: «Дата». Дата окончания планирования.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «План / факт».
