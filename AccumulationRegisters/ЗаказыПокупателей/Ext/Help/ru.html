Регистр накопления "Заказы покупателей"

Назначение

Для учета заказов покупателей служит одноименный регистр (не балансовый). Смысл остатков на регистре следующий: это остатки невыполненных заказов покупателей. Отрицательные остатки на регистре не имеют смысла.

Измерения

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочники: «Контрагенты и контакты», «Подразделения компании». Контрагент, разместивший заказ.
  • Заказ. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ внутренний», Ссылается на документ «Заказ покупателя». Документ оформления заказа.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура заказа.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры заказа.
  • Склад компании. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором был зарезервирован товар для этого заказа.

Ресурсы

  • Заказано. Тип значения: «Число». Количество номенклатуры по заказу.
  • Резерв. Тип значения: «Число». Количество номенклатуры, зарезервированной на складе для этого заказа. 
  • Резерв из свободного запаса. Тип значения: «Число». Количество номенклатуры, зарезервированной на складе для этого заказа из свободного запаса. 
  • Сумма в валюте договора. Тип значения: «Число». Сумма заказанной номенклатуры в валюте договора.
  • Сумма в валюте управленческого учёта. Тип значения: «Число». Сумма заказанной номенклатуры в валюте управленческого учета.

Реквизиты

  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.
  • Вид операции. Ссылается на перечисление «Виды операций заказов». Классифицирует операцию с заказом (Заказ, Резервирование, Закрытие заказа...).
  • Идентификатор замены. Тип значения: «Уникальный идентификатор». Предназначен для указания связи между замененными и заменившими товарами в заказе.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Заказ внутренний», «Заказ покупателя», «Корректировка заказа покупателя», «Перемещение товаров», «Поступление товаров», «Реализация товаров», «Резервирование заказов покупателя», «Снятие резервов заказов покупателя».
