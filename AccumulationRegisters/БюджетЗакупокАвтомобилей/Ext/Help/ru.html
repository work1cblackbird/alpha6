Регистр накопления "Бюджет закупок автомобилей"

Назначение

 Оборотный регистр "Бюджет закупок автомобилей" предназначен для накопления информации по спланированным данным закупок автомобилей в разрезах моделей и вариантов конфигурации.

Измерения

  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, для которого спрогнозированы данные.
  • Сценарий планирования. Реквизит является обязательным для заполнения. Ссылается на справочник «Сценарии планирования». Сценарий планирования, в разрезе которого было произведено планирование.
  • Модель. Реквизит является обязательным для заполнения. Ссылается на справочник «Модели». Спрогнозированная модель автомобиля.
  • Вариант комплектации. Реквизит не является обязательным для заполнения. Ссылается на справочник «Варианты комплектации». Спрогнозированный вариант комплектации автомобиля. 
  • Хозяйственная операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция, определяющая вид бюджета: бюджет закупок по моделям, либо бюджет закупок по вариантам комплектации. 

Ресурсы

  • Количество. Тип значения: «Число». Количество спрогнозированных автомобилей.
  • Сумма рег.. Тип значения: «Число». Спрогнозированная сумма в валюте регламентированного учета.
  • Сумма упр.. Тип значения: «Число». Спрогнозированная сумма в валюте управленческого учета.
  • Сумма НДС. Тип значения: «Число». Спрогнозированная сумма НДС в валюте регламентированного учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Бюджет закупок автомобилей».
