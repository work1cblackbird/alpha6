Регистр накопления "Начисления сотрудникам по заказ-нарядам"

Назначение

Регистр «Начисления сотрудникам по заказ-нарядам» служит для быстрого получения информации о начислениях сотрудникам по выполненным автоработам заказ-наряда.

Измерения

  • Организация. Ссылается на справочник «Организации». Организация, в которой выполнена работа сотрудником по заказ-наряду. 
  • Подразделение компании. Ссылается на справочник «Подразделения компании». Подразделение компании, в которой выполнена работа сотрудником по заказ-наряду. 
  • Сотрудник. Ссылается на справочник «Сотрудники». Сотрудник, выполнявший работы по заказ-наряду. 
  • Заказ-наряд. Ссылается на документ «Заказ-наряд». Документ, на основании которого выполнена авторабота сотрудником. 
  • Вид начисления. Ссылается на план видов расчета «Начисления и удержания». Вид начисления сотруднику за выполненную автоработу. 
  • Авторабота. Ссылается на справочник «Автоработы». Авторабота, выполненная сотрудником. 

Ресурсы

  • Начислено. Тип значения: «Число». Сумма начислений сотруднику в валюте регламентированного учета. 
  • Начислено (упр.). Тип значения: «Число». Сумма начислений сотруднику в валюте управленческого учета.
