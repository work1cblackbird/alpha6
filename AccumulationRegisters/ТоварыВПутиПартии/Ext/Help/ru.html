Регистр накопления "Товары в пути (партии)"

Назначение

Для учета товаров, которые находятся в пути из филиала в филиал в разрезе партий, в программе присутствует регистр «Товары в пути (партии)». 

Измерения

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Организация. 
  • Отправитель. Реквизит является обязательным для заполнения. Ссылается либо на справочник «Склады компании», либо на справочник «Подразделения компании». Филиал организации, из которого отправили товар.
  • Получатель. Ссылается либо на справочник «Склады компании», либо на справочник «Подразделения компании». Филиал организации, в который отправили товар. 
  • Номенклатура.  Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура. 
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика номенклатуры.
  • Статус партии. Ссылается на перечисление «Статусы партий». Статус партии, определяющий, чей товар (купленный компанией или принятый на комиссию). 
  • Партия. Ссылается на документы: «Перемещение товаров», «Инвентаризация», «Комплектация», «Возврат от покупателя», «Пересортица товаров», «Поступление товаров», «Разукомплектация». Партия, по которой поступил или был взят на комиссию товар.
  • ГТД. Ссылается на справочник «ГТД». Грузовые таможенные декларации.

Ресурсы

  • Количество. Тип значения: «Число». Количество товаров, отправленных из одного филиала организации в другой, в разрезе партий. 
  • Сумма (регл.). Тип значения: «Число». Сумма в валюте регламентированного учета с налогами. 
  • Сумма НДС (регл.). Тип значения: «Число». Сумма НДС в валюте регламентированного учета. 
  • Сумма без НДС (регл.). Тип значения: «Число». Сумма в валюте регламентированного учета без налогов. 
  • Сумма (упр.). Тип значения: «Число». Сумма в валюте управленческого учета с налогами. 
  • Сумма НДС (упр.). Тип значения: «Число». Сумма НДС в валюте управленческого учета. 
  • Сумма без НДС (упр.). Тип значения: «Число». Сумма в валюте управленческого учета без налогов. 

Способ вызова

Из меню "Перейти" формы документа «Перемещение товаров» возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Перемещение товаров».
